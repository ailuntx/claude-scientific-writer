# 推荐强度指南

## GRADE临床推荐框架

### 推荐的组成部分

每个临床推荐应包括：

1. **人群**：谁应该接受干预？
2. **干预措施**：具体治疗/行动是什么？
3. **对照**：与什么替代方案相比？
4. **结局**：预期结果是什么？
5. **强度**：推荐有多强？
6. **证据质量**：我们对证据有多大把握？

### 推荐强度（1级 vs 2级）

#### 强推荐（1级）

**使用时机**：
- 期望效果明显超过不期望效果（或反之）
- 高质量或中等质量证据
- 价值观和偏好：预期变异性小
- 资源影响：具有成本效益或成本考虑很小

**措辞**："我们推荐..."或"临床医生应该..."

**影响**：
- 大多数患者应接受推荐的干预措施
- 遵守推荐可作为质量指标
- 政策制定者可将其作为绩效指标进行调整

**示例**：

```
强推荐支持（1级）：

"我们推荐奥希替尼80 mg每日一次作为携带EGFR外显子19缺失或L858R突变的
晚期NSCLC成人患者的一线治疗（强推荐，高质量证据 - GRADE 1A）。"

理由：
- 大型PFS获益：18.9 vs 10.2个月（HR 0.46，p<0.001）
- OS获益：38.6 vs 31.8个月（HR 0.80，p=0.046）  
- 更好的耐受性：3-4级不良事件更少
- 证据：高质量（大型RCT，偏倚风险低）
- 获益明显超过危害
```

```
强推荐反对（1级）：

"我们反对在新诊断的胶质母细胞瘤患者中一线使用贝伐单抗来改善总生存期
（强反对，高质量证据 - GRADE 1A）。"

理由：
- 无OS获益：HR 0.88（0.76-1.02），p=0.10（AVAglio试验）
- 毒性：3级及以上不良事件增加（66% vs 52%）
- 证据：高质量（两个大型3期RCT）
- 危害超过缺乏生存获益
```

#### 有条件/弱推荐（2级）

**使用时机**：
- 期望效果和不期望效果接近平衡
- 低质量或极低质量证据
- 价值观和偏好：实质性变异性
- 资源影响：高成本或有限可及性

**措辞**："我们建议..."或"临床医生可能..."

**影响**：
- 不同的患者适合不同的选择
- 共同决策至关重要
- 政策制定需要实质性辩论和利益相关者参与

**示例**：

```
有条件推荐支持（2级）：

"我们建议考虑在铂类-培美曲塞一线化疗后对未出现疾病进展的晚期非鳞状
NSCLC患者进行培美曲塞维持治疗（有条件推荐，中等质量证据 - GRADE 2B）。"

理由：
- 中等PFS获益：4.0 vs 2.0个月（HR 0.62）
- 无OS获益：13.9 vs 11.0个月（HR 0.79，p=0.23）
- 毒性：持续化疗负担
- 生活质量：症状控制与治疗副作用之间的权衡
- 患者价值观：有些患者优先考虑治疗休息时间，有些优先考虑疾病控制
- 共同决策至关重要
```

```
有条件推荐 - 两种方案均可接受（2级）：

"我们建议帕博利珠单药治疗或帕博利珠单抗联合铂类双药化疗作为PD-L1≥50%
NSCLC的一线治疗，基于患者偏好和临床因素（有条件推荐，高质量证据 - 
GRADE 2A）。"

理由：
- 两种方案均为NCCN 1类首选
- 单药治疗：毒性更低，口服vs静脉，生活质量更好
- 联合方案：ORR更高（48% vs 39%），PFS数值上更长
- OS：两种策略相似
- 患者价值观：差异很大（耐受性vs反应率优先）
```

### 证据质量（⊕⊕⊕⊕ 到 ⊕○○○）

#### 高质量（⊕⊕⊕⊕）

- 进一步研究极不可能改变效果估计的信心
- 精心设计的RCT的一致结果
- 无严重局限性
- 直接证据（目标人群、干预、结局）
- 精确估计（窄置信区间）

**示例**：FLAURA试验用于EGFR+ NSCLC中的奥希替尼 - 大型RCT，一致结果，偏倚风险低，直接结局

#### 中等质量（⊕⊕⊕○）

- 进一步研究可能影响信心并可能改变估计
- 存在一些局限性的RCT或来自观察性研究的非常强证据
- 一些不一致性、间接性、不精确性或发表偏倚

**示例**：有局限性的大型RCT，或具有中等异质性的多个RCT

#### 低质量（⊕⊕○○）

- 进一步研究很可能对估计的信心产生重要影响
- 观察性研究或有严重局限性的RCT
- 在一致性、直接性、精确性或偏倚方面存在严重问题

**示例**：精心设计的队列研究，或高失访率和不清楚分配隐藏的RCT

#### 极低质量（⊕○○○）

- 效果估计非常不确定
- 病例系列、专家意见、机制推理
- 非常严重的局限性

**示例**：回顾性病例系列，无系统综述的专家共识

## 结合强度与质量

### 九种可能的组合

| 证据质量 | 强支持（↑↑） | 弱支持（↑） | 强反对（↓↓） | 弱反对（↓） |
|---------|------------|------------|-------------|------------|
| **高质量（⊕⊕⊕⊕）** | 1A级 | 2A级 | 1A级（反对） | 2A级（反对） |
| **中等质量（⊕⊕⊕○）** | 1B级 | 2B级 | 1B级（反对） | 2B级（反对） |
| **低质量（⊕⊕○○）** | 1C级* | 2C级 | 1C级（反对）* | 2C级（反对） |
| **极低质量（⊕○○○）** | 1D级* | 2D级 | 1D级（反对）* | 2D级（反对） |

*罕见：强推荐通常需要至少中等质量的证据

### 不寻常的组合（发生时）

**低质量证据的强推荐（1C级）**

罕见，但可能发生于：
- 来自观察性数据的大效应量（RR>5或<0.2）
- 低质量证据，但获益-危害平衡明确
- 示例：房颤抗凝治疗（在RCT之前，有强有力的观察性数据）

**高质量证据的弱推荐（2A级）**

发生于：
- 获益和危害接近平衡
- 患者价值观差异很大
- 示例：低风险个体的阿司匹林一级预防（获益小，出血风险存在，患者价值观各异）

## 措辞模板

### 强推荐

**支持（↑↑）**：
- "我们推荐[干预措施]用于[人群]。"
- "临床医生应该[行动]。"
- "推荐[干预措施]。"

**反对（↓↓）**：
- "我们反对对[人群]使用[干预措施]。"
- "临床医生不应该[行动]。"
- "不推荐[干预措施]。"

### 有条件/弱推荐

**支持（↑）**：
- "我们建议[干预措施]用于[人群]。"
- "临床医生可能考虑[行动]。"
- "[干预措施]可考虑用于精选患者。"

**反对（↓）**：
- "我们建议不对[人群]使用[干预措施]。"
- "临床医生可能避免[行动]。"
- "一般不推荐[干预措施]。"

**两者均可接受**：
- "我们建议根据患者偏好选择[方案A]或[方案B]。"
- "两种方法都合理。"

## 可视化文档的颜色编码

**强推荐（绿色背景）**：
- RGB(0, 153, 76) 或 #009954
- 清晰的视觉优先
- 用于1A级、1B级

**有条件推荐（黄色背景）**：
- RGB(255, 193, 7) 或 #FFC107
- 表示需要讨论
- 用于2A级、2B级、2C级

**研究/探索性（蓝色背景）**：
- RGB(33, 150, 243) 或 #2196F3
- 临床试验考虑
- 证据不足以支持标准治疗

**不推荐（红色边框/背景）**：
- RGB(220, 20, 60) 或 #DC143C
- 强反对推荐
- 证据显示有害或无获益

## 常见场景

### 场景1：强证据，明确的获益-危害平衡

**示例**：帕博利珠单抗用于PD-L1≥50% NSCLC

- 证据：大型3期RCT（KEYNOTE-024），n=305，设计良好
- 结果：PFS HR 0.50（0.37-0.68），OS HR 0.60（0.41-0.89）
- 毒性：3-5级不良事件低于化疗（27% vs 53%）
- 患者价值观：大多数优先考虑疗效和耐受性

**推荐**：强支持（1A级）

### 场景2：中等证据，平衡的权衡

**示例**：切除后黑色素瘤的辅助免疫治疗

- 证据：显示无复发生存获益的RCT，OS数据不成熟
- 结果：复发风险降低但持续毒性
- 毒性：需要类固醇的免疫相关不良事件（一些严重）
- 成本：12个月治疗的高年度成本
- 患者价值观：可变（一些优先考虑预防复发，一些避免毒性）

**推荐**：有条件支持（2B级）

### 场景3：低证据，但严重后果

**示例**：人工心脏瓣膜抗凝治疗

- 证据：无RCT（会违反伦理），观察性数据和机制推理
- 后果：不抗凝血栓栓塞风险非常高
- 获益-危害：尽管证据质量低，仍明确

**推荐**：强支持（1C级）

### 场景4：高证据，但患者偏好不同

**示例**：乳房切除术后乳房重建

- 结局和生活质量的高质量数据
- 权衡：美容获益vs额外手术、恢复时间
- 价值观：高度个人化的决定，偏好差异大

**推荐**：有条件（2A级）- 讨论选项，由患者决定

- 主要研究：[引用]

## 质量检查清单

在最终确定推荐之前，请验证：

- [ ] 推荐声明清晰且可操作
- [ ] 明确说明强度（强 vs 有条件）
- [ ] 证据质量已分级（高/中等/低/极低）
- [ ] 提供GRADE标注（1A、1B、2A、2B、2C）
- [ ] 引用具体研究结果的证据
- [ ] 量化获益（带CI的效应量）
- [ ] 量化危害（不良事件率）
- [ ] 解释获益/危害平衡
- [ ] 解决患者价值观考虑（如有条件）
- [ ] 提及替代方案
- [ ] 记录指南一致性
- [ ] 解决特殊人群（老年人、肾/肝功能不全）
- [ ] 指定监测要求
