# 生物标志物分类与解读指南

## 概述

生物标志物是生物状态或条件的可测量指标。在临床决策支持中，生物标志物指导诊断、预后、治疗选择和监测。本指南涵盖基因组学、蛋白质组学和分子生物标志物，重点关注临床可操作性。

## 生物标志物类别

### 预后生物标志物

**定义**：预测临床结局（生存、复发），与治疗方式无关

**按疾病分类的示例**

**癌症**
- **Ki-67 指数**：高增殖（>20%）预示乳腺癌预后不良
- **TP53 突变**：多种癌症类型预后不良
- **肿瘤分期/分级**：TNM分期，组织学分级
- **LDH 升高**：黑色素瘤、淋巴瘤预后不良
- **AFP 升高**：肝细胞癌预后不良

**心血管**
- **NT-proBNP/BNP**：升高水平可预测心力衰竭死亡率
- **Troponin**：可预测ACS不良事件
- **CRP**：炎症标志物，可预测心血管事件

**感染性疾病**
- **HIV病毒载量**：如不治疗可预测疾病进展
- **HCV基因型**：可预测所需治疗时间

**应用**：风险分层，治疗强度选择，临床试验入组

### 预测生物标志物

**定义**：识别可能受益（或不能受益）于特定治疗的患者

**阳性预测生物标志物（治疗获益）**

**肿瘤学 - 靶向治疗**
- **EGFR 19号外显子缺失/L858R → EGFR TKI**：有效率60-70%，PFS 10-14个月
- **ALK重排 → ALK抑制剂**：ORR 70-90%，PFS 25-34个月
- **HER2扩增 → Trastuzumab**：仅在HER2+（IHC 3+或FISH+）中获益
- **BRAF V600E → BRAF抑制剂**：ORR 50%，PFS 6-7个月（黑色素瘤）
- **PD-L1 ≥50% → Pembrolizumab**：ORR 45%，PFS 10个月 vs 6个月（化疗）

**肿瘤学 - 免疫治疗**
- **MSI-H/dMMR → 抗PD-1**：ORR 40-60%，涵盖多种肿瘤类型
- **TMB高 → 免疫治疗**：研究性，一些获益信号
- **PD-L1表达 → 抗PD-1/PD-L1**：更高表达与更好疗效相关

**血液学**
- **BCR-ABL → Imatinib (CML)**：完全细胞遗传学缓解80%
- **CD20+ → Rituximab (淋巴瘤)**：仅在CD20表达细胞中获益
- **CD33+ → Gemtuzumab ozogamicin (AML)**：在CD33+亚组中获益

**阴性预测生物标志物（耐药/无获益）**
- **KRAS突变 → 抗EGFR单抗（CRC）**：无获益，禁忌
- **EGFR T790M → 1代/2代TKI**：耐药机制，使用奥希替尼
- **RAS/RAF野生型required → BRAF抑制剂（黑色素瘤）**： paradoxical MAPK激活

### 诊断生物标志物

**定义**：检测或确认疾病存在

**感染性疾病**
- **PCR检测病原体DNA/RNA**：SARS-CoV-2、HIV、HCV病毒载量
- **抗体滴度**：IgM（急性），IgG（既往暴露/免疫）
- **抗原检测**：快速检测（链球菌、流感、COVID）

**自身免疫**
- **ANA**：筛查系统性狼疮、结缔组织病
- **抗CCP**：类风湿关节炎的特异性指标
- **抗dsDNA**：系统性狼疮，与疾病活动度相关
- **ANCA**：血管炎（c-ANCA用于GPA，p-ANCA用于MPA）

**癌症**
- **PSA**：前列腺癌筛查/监测
- **CA 19-9**：胰腺癌，胆道梗阻
- **CEA**：结直肠癌监测
- **AFP**：肝细胞癌，生殖细胞肿瘤

### 药效学生物标志物

**定义**：评估治疗反应或作用机制

**示例**
- **HbA1c**：糖尿病血糖控制（通常目标<7%）
- **LDL胆固醇**：他汀疗效（高危患者目标<70 mg/dL）
- **血压**：抗高血压疗效（目标<130/80 mmHg）
- **病毒载量抑制**：抗逆转录病毒疗效（目标<20 copies/mL）
- **INR**：华法林抗凝监测（大多数适应症目标2-3）

## 基因组生物标志物

### 突变分析

**驱动突变（致癌）**
- **激活突变**：通路组成性激活（BRAF V600E，EGFR L858R）
- **失活突变**：肿瘤抑制因子丢失（TP53，PTEN）
- **热点突变**：反复出现的位置（KRAS G12/G13，PIK3CA H1047R）
- **变异等位基因频率（VAF）**：克隆性（VAF≈50%为克隆性，<10%为亚克隆）

**耐药突变**
- **EGFR T790M**：对1代/2代TKI耐药（40-60%病例）
- **ALK G1202R，I1171N**：对早期ALK抑制剂耐药
- **ESR1突变**：对芳香化酶抑制剂耐药（乳腺癌）
- **RAS突变**：对抗EGFR治疗获得性耐药（CRC）

**突变检测方法**
- **组织NGS**：全面基因组分析，300-500个基因
- **液体活检**：ctDNA分析，无创，序贯监测
- **PCR检测**：靶向热点检测，FDA批准的伴随诊断
- **等位基因特异性PCR**：对已知突变高灵敏度（cobas EGFR检测）

### 拷贝数变异（CNV）

**扩增**
- **HER2 (ERBB2)**：乳腺癌，胃癌 → trastuzumab，pertuzumab
  - 检测：IHC（0，1+，2+，3+）→ 如2+则FISH（HER2/CEP17比值≥2.0）
- **MET扩增**：NSCLC耐药机制 → crizotinib，capmatinib
  - 切点：基因拷贝数≥5，GCN/CEP7比值≥2.0
- **EGFR扩增**：胶质母细胞瘤，部分NSCLC
- **FGFR2扩增**：胃癌 → 研究性FGFR抑制剂

**缺失**
- **PTEN缺失**：常见于多种癌症，预测PI3K通路激活
- **RB1缺失**：小细胞转化，预后不良
- **CDKN2A/B缺失**：细胞周期失调
- **纯合缺失**：两个等位基因完全丢失（意义更大）

**检测方法**
- **FISH（荧光原位杂交）**：HER2，ALK重排
- **NGS拷贝数 calling**：覆盖深度分析
- **SNP阵列**：全基因组CNV检测
- **ddPCR**：定量拷贝数测量

### 基因融合与重排

**致癌融合**
- **ALK融合**（NSCLC）：EML4-ALK最常见（60%），20+个伙伴
  - 检测：IHC（D5F3抗体），FISH（break-apart探针），NGS/RNA-seq
- **ROS1融合**（NSCLC，胶质母细胞瘤）：CD74-ROS1，SLC34A2-ROS1等
- **RET融合**（NSCLC，甲状腺）：KIF5B-RET，CCDC6-RET
- **NTRK融合**（多种肿瘤类型，罕见）：ETV6-NTRK3等
  - 泛肿瘤：Larotrectinib，entrectinib获批用于多种肿瘤
- **BCR-ABL**（CML，ALL）：t(9;22)，费城染色体

**融合伙伴的考虑因素**
- 伙伴影响药物敏感性（EML4-ALK 3型变异更敏感）
- 5' vs 3'融合影响检测方法
- 内含子断裂点不同（RNA-seq比DNA面板更全面）

**检测方法**
- **FISH break-apart探针**：ALK，ROS1，RET
- **IHC**：ALK蛋白过表达（筛查），ROS1
- **RT-PCR**：靶向融合检测
- **RNA-seq**：全面融合检测，识别新伙伴

### 肿瘤突变负荷（TMB）

**定义**：每兆碱基DNA的体细胞突变数量

**分类**
- **TMB高**：≥10 mut/Mb（部分定义≥20 mut/Mb）
- **TMB中等**：6-9 mut/Mb
- **TMB低**：<6 mut/Mb

**临床应用**
- **免疫治疗预测**：TMB更高 → 更多新抗原 → 免疫反应更好
- **FDA批准**：Pembrolizumab用于TMB-H（≥10 mut/Mb）实体瘤（2020）
- **局限性**：未在所有肿瘤类型中验证，检测方法变异

**通常TMB高的肿瘤类型**
- 黑色素瘤（中位数10-15 mut/Mb）
- NSCLCL（尤其是吸烟相关，8-12 mut/Mb）
- 尿路上皮癌（8-10 mut/Mb）
- 微卫星不稳定肿瘤（30-50 mut/Mb）

### 微卫星不稳定（MSI）与错配修复（MMR）

**分类**
- **MSI高（MSI-H）**：5个位点中≥2个不稳定或≥30%标记物
- **MSI低（MSI-L）**：5个位点中<2个不稳定
- **微卫星稳定（MSS）**：无不稳定

**错配修复状态**
- **dMMR（缺陷）**：MLH1、MSH2、MSH6或PMS2通过IHC丢失
- **pMMR（完整）**：四个MMR蛋白均表达

**临床意义**
- **MSI-H/dMMR肿瘤**：大多数实体瘤的3-5%，结直肠癌的15%
- **免疫治疗敏感性**：ORR 30-60%对抗PD-1治疗
  - Pembrolizumab FDA批准用于MSI-H/dMMR实体瘤（2017）
  - Nivolumab ± ipilimumab获批
- **化疗耐药**：MSI-H CRC不能从5-FU辅助化疗获益
- **Lynch综合征**：如果MSI-H + 年轻年龄 + 家族史，考虑胚系MMR突变

**检测流程**
```
结直肠癌（所有新诊断）：
1. MMR蛋白IHC（MLH1，MSH2，MSH6，PMS2）
   ├─ 全部完整 → pMMR (MSS) → 如有指征用标准化疗
   │
   └─ 丢失一个或多个 → dMMR (可能MSI-H)
      └─ 反射性MLH1启动子超甲基化检测
         ├─ 甲基化 → 散发性MSI-H，免疫治疗可选
         └─ 非甲基化 → Lynch综合征胚系检测
```

## 表达生物标志物

### 免疫组化（IHC）

**PD-L1表达（免疫检查点）**
- **检测**：22C3（FDA），28-8，SP263，SP142（评分有些差异）
- **评分**：肿瘤比例评分（TPS）= %膜着色的肿瘤细胞
  - TPS <1%：低/阴性
  - TPS 1-49%：中等
  - TPS ≥50%：高
- **综合阳性评分（CPS）**：（PD-L1+肿瘤细胞+免疫细胞）/总肿瘤细胞×100
  - 用于部分适应症（如CPS≥10用于Pembrolizumab治疗HNSCC）

**激素受体（乳腺癌）**
- **ER/PR阳性**：IHC核着色≥1%（ASCO/CAP指南）
  - Allred评分0-8（比例+强度）- 历史性
  - H评分0-300（各强度百分比）- 定量
- **临床切点**：
  - ER ≥1%：内分泌治疗指征
  - ER 1-10%："低阳性"，可能获益较低
  - ER+伴PR丢失：可能内分泌耐药

**HER2检测（乳腺癌/胃癌）**
```
IHC初检：
├─ 0或1+：HER2阴性（无需进一步检测）
│
├─ 2+：不确定 → 反射性FISH检测
│  ├─ FISH+（HER2/CEP17比值≥2.0或HER2拷贝≥6/细胞）→ HER2阳性
│  └─ FISH- → HER2阴性
│
└─ 3+：HER2阳性（无需FISH）
   └─ >10%肿瘤细胞均匀强烈完整膜着色

HER2阳性：Trastuzumab治疗指征
HER2低（IHC 1+或2+/FISH-）：Trastuzumab deruxtecan资格（2022）
```

### RNA表达分析

**基因表达签名（乳腺癌）**

**Oncotype DX（21基因检测）**
- **复发评分（RS）**：0-100
  - RS <26：低风险 → 单独内分泌治疗（大多数患者）
  - RS 26-100：高风险 → 化疗+内分泌治疗
- **人群**：ER+/HER2-，淋巴结阴性或1-3个阳性淋巴结
- **证据**：TAILORx试验（N=10,273）验证RS<26可省略化疗

**MammaPrint（70基因检测）**
- **结果**：高风险vs低风险（二元）
- **人群**：早期乳腺癌，ER+/HER2-
- **证据**：MINDACT试验验证低风险可省略化疗

**Prosigna（PAM50）**
- **结果**：复发风险（ROR）评分+内在亚型
- **亚型**：腔面A，腔面B，HER2富集型，基底样
- **应用**：绝经后，ER+，淋巴结阴性或1-3个淋巴结

**RNA-Seq融合检测**
- **优势**：检测新融合伙伴，定量表达
- **应用**：NTRK融合（罕见，多个伙伴），RET融合
- **局限性**：需要新鲜/冷冻组织或高质量FFPE RNA

## 分子亚型

### 胶质母细胞瘤（GBM）分子分类

**Verhaak 2010分类（4个亚型）**

**原始神经亚型**
- **特征**：PDGFRA扩增，IDH1突变（继发性GBM），TP53突变
- **年龄**：通常为较年轻患者
- **预后**：较好预后（中位OS 15-18个月）
- **治疗**：可能从贝伐单抗获益少于其他亚型

**神经亚型**
- **特征**：神经元标记物（NEFL，GABRA1，SYT1，SLC12A5）
- **争议**：可能代表正常脑组织污染
- **预后**：中等
- **治疗**：标准替莫唑胺为基础的治疗

**经典亚型**
- **特征**：EGFR扩增（97%），7号染色体增益，10号染色体缺失
- **关联**：缺乏TP53、PDGFRA、NF1突变
- **预后**：中等
- **治疗**：可能从EGFR抑制剂获益（研究性）

**间充质亚型**
- **特征**：NF1突变/缺失，间充质标记物高表达（CHI3L1/YKL-40）
- **免疫特征**：更高 macrophage/microglia浸润
- **亚组**：间充质-免疫活化型（高免疫签名）
- **预后**：预后不良（中位OS 12-13个月）
- **治疗**：可能更好响应抗血管生成治疗，免疫治疗研究性

**临床应用**
```
GBM分子分型报告：

患者队列：间充质-免疫活化型亚型（n=15）

分子特征：
- NF1改变：73%（11/15）
- YKL-40高表达：100%（15/15）
- 免疫基因签名：升高（中位z-score +2.3）
- CD163+巨噬细胞：高密度（中位180/mm²）

治疗意义：
- 标准治疗：替莫唑胺为基础（Stupp方案）
- 考虑：复发时使用贝伐单抗（可能获益增强）
- 临床试验：免疫检查点抑制剂±抗血管生成治疗
- 预后：中位OS 12-14个月（比原始神经型差）

建议：
如符合资格可入组联合免疫治疗试验，否则标准治疗并在进展早期考虑贝伐单抗。
```

### 乳腺癌内在亚型

**PAM50分类**

**腔面A**
- **特征**：ER+，HER2-，低增殖（Ki-67 <20%）
- **基因签名**：高ER相关基因，低增殖基因
- **预后**：最佳预后，低复发风险
- **治疗**：通常单独内分泌治疗足够
- **化疗**：除非高风险特征否则很少需要

**腔面B**
- **特征**：ER+，HER2-或HER2+，高增殖（Ki-67 ≥20%）
- **亚型**：腔面B（HER2-）和腔面B（HER2+）
- **预后**：中等预后
- **治疗**：化疗+内分泌治疗；如HER2+加用trastuzumab

**HER2富集型**
- **特征**：HER2+，ER-，PR-
- **基因签名**：高HER2和增殖基因，低ER基因
- **预后**：不治疗预后差，HER2靶向治疗预后好
- **治疗**：化疗+trastuzumab+pertuzumab

**基底样**
- **特征**：ER-，PR-，HER2-（三阴性），高增殖
- **基因签名**：基底角蛋白（CK5/6，CK17），EGFR
- **重叠**：与TNBC 80%一致性，但不完全相同
- **预后**：侵袭性强，高早期复发风险
- **治疗**：化疗（铂类，蒽环类），如BRCA突变可用PARP抑制剂
- **免疫治疗**：PD-L1+可从pembrolizumab+化疗获益

### 结直肠癌共识分子亚型（CMS）

**CMS1（14%）：MSI免疫**
- **特征**：MSI高，BRAF突变，强免疫激活
- **预后**：复发后生存率低，尽管有免疫浸润
- **治疗**：免疫治疗高度有效，5-FU化疗无效

**CMS2（37%）：经典**
- **特征**：上皮性，显著WNT和MYC激活
- **预后**：生存率较好
- **治疗**：辅助化疗获益

**CMS3（13%）：代谢**
- **特征**：代谢失调，KRAS突变
- **预后**：中等生存
- **治疗**：可能从靶向代谢治疗获益（研究性）

**CMS4（23%）：间充质**
- **特征**：基质浸润，TGF-β激活，血管生成
- **预后**：生存率最差，常在晚期诊断
- **治疗**：可能从抗血管生成治疗获益（贝伐单抗）

## 伴随诊断

### FDA批准的生物标志物-药物配对

**必需检测（标签适应症）**
```
生物标志物              药物                    适应症                  检测
EGFR 19号外显子缺失/L858R  奥希替尼                NSCLC                   cobas EGFR v2, NGS
ALK重排                Alectinib, brigatinib    NSCLC                   Vysis ALK FISH, IHC (D5F3)
BRAF V600E              vemurafenib, dabrafenib  黑色素瘤, NSCLC      THxID BRAF, cobas BRAF
HER2扩增               trastuzumab, pertuzumab  乳腺癌,胃癌           HercepTest IHC, FISH
ROS1重排               crizotinib, entrectinib  NSCLC                   FISH, NGS
PD-L1 ≥50% TPS         Pembrolizumab（单药）    NSCLC一线              22C3 pharmDx
MSI-H/dMMR             Pembrolizumab            任何实体瘤             IHC (MMR), PCR (MSI)
NTRK融合               Larotrectinib, entrectinib 泛肿瘤             FoundationOne CDx
BRCA1/2突变            Olaparib, talazoparib   乳腺癌,卵巢,前列腺     BRACAnalysis CDx
```

**补充诊断（信息性，非必需）**

- **PD-L1 1-49%**：决定联合vs单药治疗选择
- **TMB高**：可能预测免疫治疗获益（FDA未批准适应症）
- **STK11/KEAP1突变**：与免疫治疗耐药相关
- **同源重组缺陷（HRD）**：预测PARP抑制剂获益

## 临床可操作性框架

### OnCoKB证据级别（纪念斯隆凯特琳癌症中心）

**1级：FDA批准**
- 生物标志物-药物配对在特定肿瘤类型中获得FDA批准
- 示例：EGFR L858R → 奥希替尼用于NSCLC

**2级：标准治疗超适应症**
- 生物标志物-药物在专业指南中用于特定肿瘤类型（生物标志物未获FDA批准）
- 示例：BRAF V600E → dabrafenib + trametinib用于CRC（NCCN推荐）

**3级：临床证据**
- 临床试验证据支持生物标志物-药物关联
  - 3A：令人信服的临床证据
  - 3B：不同肿瘤类型的标准治疗或研究性

**4级：生物学证据**
- 仅临床前证据（细胞系，小鼠模型）
- 4：支持关联的生物学证据

**R1-R2级：耐药**
- R1：与耐药相关的标准治疗
- R2：研究性或临床前耐药证据

### CIViC（癌症变异临床解读）

**证据级别**
- **A**：在临床实践中验证或通过监管关联验证
- **B**：临床试验或其他主要患者数据支持关联
- **C**：带分子分析的病例研究
- **D**：临床前证据（细胞培养，动物模型）
- **E**：推断性关联（文献综述，专家意见）

**临床意义分层**
- **I层**：具有强临床意义的变异（预测性、诊断性、预后在专业指南中）
- **II层**：具有潜在临床意义的变异（临床试验或病例研究证据）
- **III层**：意义不明确的变异
- **IV层**：良性或可能良性变异

## 多生物标志物面板

### 全面基因组分析（CGP）

**FoundationOne CDx**
- **基因**：324个基因（SNV，indel，CNV，重排）
- **附加**：TMB，MSI状态
- **FDA批准**：18+种靶向治疗的伴随诊断
- **周转时间**：10-14天
- **组织**：FFPE，40张未染色切片或组织块

**Guardant360 CDx（液体活检）**
- **基因**：74个基因在无细胞DNA（cfDNA）中
- **样本**：2管血（共20 mL）
- **FDA批准**：奥希替尼（EGFR）伴随诊断，NSCLC
- **应用**：无创，序贯监测，当组织不可及时
- **局限性**：灵敏度低于组织（尤其是低肿瘤负荷）

**Tempus xT**
- **基因**：648个基因（DNA）+全转录组（RNA）
- **优势**：RNA检测融合，表达签名
- **应用**：研究和临床使用
- **未获FDA批准**：目前不是伴随诊断

### 按肿瘤类型的检测建议

**NSCLC（NCCN指南）**
```
所有晚期NSCLC诊断时进行广泛分子分析：

必需（FDA批准的治疗可用）：
✓ EGFR突变（18、19、20、21号外显子）
✓ ALK重排
✓ ROS1重排
✓ BRAF V600E
✓ MET 14号外显子跳跃
✓ RET重排
✓ NTRK融合
✓ KRAS G12C
✓ PD-L1 IHC

推荐（为告知治疗策略）：
✓ 全面NGS面板（涵盖上述所有+新兴靶点）
✓ 如组织不足考虑液体活检

靶向治疗进展时：
✓ 重复组织活检或液体活检检测耐药机制
✓ 示例：EGFR T790M，ALK耐药突变，MET扩增
```

**转移性结直肠癌**
```
抗EGFR治疗（cetuximab，panitumumab）前必需：
✓ RAS检测（KRAS 2、3、4号外显子；NRAS 2、3、4号外显子）
  └─ RAS突变 → 不使用抗EGFR治疗（耐药）
✓ BRAF V600E
  └─ 如BRAF V600E+ → 考虑encorafenib + cetuximab + binimetinib

所有转移性CRC推荐：
✓ MSI/MMR检测（免疫治疗适应症）
✓ HER2扩增（如RAS/BRAF WT考虑研究性trastuzumab治疗）
✓ NTRK融合（罕见，<1%，但可操作）

左侧vs右侧：
- 左侧（降结肠，乙状结肠，直肠）：预后更好，抗EGFR更有效
- 右侧（盲肠，升结肠）：预后更差，抗EGFR效果较差，考虑贝伐单抗
```

**黑色素瘤**
```
所有晚期黑色素瘤：
✓ BRAF V600突变（30-50%皮肤黑色素瘤）
  └─ 如BRAF V600E/K → Dabrafenib + trametinib或vemurafenib + cobimetinib
✓ NRAS突变（20-30%）
  └─ 无批准靶向治疗，考虑MEK抑制剂试验
✓ KIT突变（黏膜、肢端、慢性日光损伤黑色素瘤）
  └─ 如KIT 11或13号外显子突变 → Imatinib（超适应症）
✓ PD-L1（可选，免疫治疗不需要）

注意：葡萄膜黑色素瘤有不同生物学（GNAQ，GNA11突变）
```

## 生物标志物切点和阈值

### 建立临床切点

**切点确定方法**

**数据驱动方法**
- **中位分割**：简单但任意，可能不是最优
- **三分位/四分位**：分为3-4组
- **ROC曲线分析**：最大化灵敏度和特异性
- **最大选择秩统计**：找到最优预后切点
- **需要验证**：独立队列确认必不可少

**生物学信息**
- **检测限**：检测定量下限
- **机制基础**：通路激活阈值
- **药效学**：靶点结合阈值
- **正常范围**：与健康个体比较

**临床定义**
- **指南推荐**：专业学会确定
- **监管批准**：FDA为伴随诊断指定的阈值
- **试验定义**：关键临床试验使用的切点

**PD-L1示例**
- **切点**：1%、5%、10%、50% TPS用于不同试验
- **依赖上下文**：因药物、疾病、治疗线而异
- **≥50%**：Pembrolizumab单药（KEYNOTE-024）
- **≥1%**：Atezolizumab联合，更广泛人群

### 连续vs分类

**连续分析优势**
- 保留信息（无二分法损失）
- 维持统计功效
- 可评估剂量反应关系
- 每单位增加或每标准差HR

**分类分析优势**
- 临床可解释（高vs低）
- 便于治疗决策（二元：使用靶向治疗是/否）
- 与监管批准一致（生物标志物阳性=符合资格）

**最佳实践**：同时报告连续和分类分析
- 连续生物标志物的Cox模型
- 按临床相关切点分层分析
- 亚组分析确认一致性

## 胚系vs体细胞检测

### 胚系（遗传）突变

**胚系检测适应症**
- **癌症易感综合征**：BRCA1/2，Lynch综合征（MLH1，MSH2），Li-Fraumeni（TP53）
- **家族史**：多个受影响亲属，诊断年龄轻
- **肿瘤特征**：年轻患者MSI-H，<60岁三阴性乳腺癌
- **治疗意义**：PARP抑制剂用于BRCA突变（胚系或体细胞）

**常见遗传性癌症综合征**
- **BRCA1/2**：乳腺癌，卵巢癌，胰腺癌，前列腺癌
  - 检测：所有卵巢癌，<60岁TNBC，男性乳腺癌
  - 治疗：PARP抑制剂（olaparib，talazoparib）
  - 预防：预防性乳房切除术，卵巢切除术（降低风险）
- **Lynch综合征（MLH1，MSH2，MSH6，PMS2）**：结直肠癌，子宫内膜癌，卵巢癌，胃癌
  - 检测：MSI-H/dMMR肿瘤，阿姆斯特丹II标准家族
  - 监测：20-25岁开始每1-2年结肠镜检查
- **Li-Fraumeni（TP53）**：年轻时候多种癌症
- **PTEN（Cowden综合征）**：乳腺癌，甲状腺癌，子宫内膜癌

**遗传咨询**
- 检测前咨询：对患者和家属的影响
- 检测后咨询：管理，监测，家属检测
- 知情同意：遗传歧视担忧（GINA保护）

### 体细胞（仅肿瘤）检测

**肿瘤组织检测**
- 仅检测癌细胞中存在的突变（非遗传）
- 大多数癌症驱动突变是体细胞的（肺癌中KRAS，EGFR）
- 对家属无影响
- 指导治疗选择

**区分胚系和体细胞**
- **变异等位基因频率**：胚系~50%（杂合）或~100%（纯合）；体细胞多变
- **匹配正常**：配对肿瘤-正常测序是确定性的
- **数据库**：胚系变异数据库（gnomAD，ClinVar）
- **反射性胚系检测**：如怀疑致病胚系变异则触发检测

## 质量保证

### 分析验证

- **灵敏度**：最低检测5-10%变异等位基因频率
- **特异性**：假阳性率<1%
- **重复性**：复件间一致性>95%
- **准确率**：与验证的 orthogonal方法一致性>99%
- **周转时间**：样本接收到报告的中位时间

### 临床验证

- **阳性预测值**：生物标志物阳性患者中对治疗有反应的比例
- **阴性预测值**：生物标志物阴性患者中对治疗无反应的比例
- **临床实用性**：检测是否改善患者结局
- **成本效益**：检测和治疗成本vs获得的QALY

### 能力验证

- CAP/CLIA临床实验室能力验证
- 参与外部质量保证计划
- 与参考实验室盲样交换
- 记录失败纠正措施
