# 临床决策算法指南

## 概述

临床决策算法为诊断、治疗选择和患者管理提供系统性的逐步指导。本指南涵盖算法开发、验证以及使用决策树和流程图的视觉呈现。

## 算法设计原则

### 关键组成部分

**决策节点**
- **问题/标准**：清晰、可测量的临床参数
- **二元 vs 多选项**：是/否（简单）vs 多个选项（复杂）
- **客观**：实验室值、影像学发现 vs 主观：临床判断

**行动节点**
- **治疗**：具体干预措施及剂量
- **检查**：额外的诊断程序
- **转诊**：专科会诊、更高级别的护理
- **观察**：观察等待并制定随访计划

**终止节点**
- **结局**：最终决策点
- **随访**：重新评估的时间安排
- **退出标准**：何时退出算法

### 设计标准

**清晰性**
- 明确的决策点
- 互斥的路径
- 无循环（除非有意设置重新评估周期）
- 清晰的入口和出口

**临床有效性**
- 基于证据的决策标准
- 生物标志物的经验证临界值
- 符合指南的建议
- 证据有限时的专家共识

**可用性**
- 每条路径最多7个决策点（认知负荷）
- 视觉层次结构（突出最常见路径）
- 首选可打印的单页格式
- 颜色编码表示紧急/安全程度

**完整性**
- 覆盖所有可能的场景
- 边缘病例的默认路径
- 不寻常表现的安全网规定
- 明确说明升级标准

## 临床决策树

### 诊断算法

**胸痛评估算法**

```
入口：胸痛患者

├─ STEMI标准？（≥2个连续导联ST段抬高≥1mm）
│  ├─ 是 → 激活导管室，阿司匹林325mg，肝素，氯吡格雷600mg
│  │        转运行直接PCI（目标门-球囊时间<90分钟）
│  └─ 否 → 继续评估

├─ 高危特征？（血流动力学不稳定、心律失常、肌钙蛋白升高）
│  ├─ 是 → 入住CCU，连续肌钙蛋白，心内科会诊
│  │        如为NSTEMI考虑早期血管造影
│  └─ 否 → 计算TIMI或HEART评分

├─ TIMI评分0-1或HEART评分0-3？（低危）
│  ├─ 是 → 观察6-12小时，连续肌钙蛋白，如阴性则进行负荷试验
│  │        如全部阴性则出院，72小时内心内科随访
│  └─ 否 → TIMI 2-4或HEART 4-6（中危）

├─ TIMI评分2-4或HEART评分4-6？（中危）
│  ├─ 是 → 入住遥测监护，连续肌钙蛋白，负荷影像学 vs CT血管造影
│  │        药物治疗：阿司匹林、他汀、β受体阻滞剂
│  └─ 否 → TIMI≥5或HEART≥7（高危）→ 按NSTEMI处理

决策终点：风险分层路径，记录30天事件率
```

**肺栓塞诊断算法（Wells标准）**

```
入口：疑似PE

步骤1：计算Wells评分
  临床特征得分：
  - DVT临床体征：3分
  - PE比替代诊断更可能：3分
  - 心率>100：1.5分
  - 过去4周内制动/手术：1.5分
  - 既往PE/DVT：1.5分
  - 咯血：1分
  - 恶性肿瘤：1分

步骤2：风险分层
  ├─ Wells评分≤4（PE不太可能）
  │  └─ D-二聚体检测
  │     ├─ D-二聚体阴性（<500 ng/mL）→ 排除PE，考虑替代诊断
  │     └─ D-二聚体阳性（≥500 ng/mL）→ CTPA
  │
  └─ Wells评分>4（PE可能）
     └─ CTPA（跳过D-二聚体）

步骤3：CTPA结果
  ├─ PE阳性 → 分层严重程度
  │  ├─ 大面积PE（低血压、休克）→ 溶栓 vs 栓塞切除术
  │  ├─ 次大面积PE（RV应变、肌钙蛋白+）→ 入住ICU，考虑溶栓
  │  └─ 低危PE → 抗凝，考虑门诊管理
  │
  └─ PE阴性 → 排除PE，调查替代诊断

步骤4：治疗决策（如PE确认）
  ├─ 抗凝绝对禁忌症？
  │  ├─ 是 → 放置IVC过滤器，治疗基础疾病
  │  └─ 否 → 抗凝治疗
  │
  ├─ 癌症相关血栓？
  │  ├─ 是 → 首选LMWH（依度沙班作为替代）
  │  └─ 否 → 首选DOAC（阿哌沙班、利伐沙班、依度沙班）
  │
  └─ 疗程：至少3个月，如为非诱因或复发性则延长
```

### 治疗选择算法

**NSCLC一线治疗算法**

```
入口：晚期/转移性NSCLC，足够的PS（ECOG 0-2）

步骤1：生物标志物检测完成？
  ├─ 否 → 反射性检测：EGFR、ALK、ROS1、BRAF、PD-L1，考虑NGS
  │       暂缓系统性治疗等待结果（除非快速进展）
  └─ 是 → 进入步骤2

步骤2：可操作的基因组改变？
  ├─ EGFR 19号外显子缺失或L858R → 奥希替尼80mg每日
  │  └─ 替代：厄洛替尼、吉非替尼、阿法替尼（不太首选）
  │
  ├─ ALK重排 → 阿来替尼600mg每日两次
  │  └─ 替代：布格替尼、劳拉替尼、克唑替尼（不太首选）
  │
  ├─ ROS1重排 → 克唑替尼250mg每日两次或恩曲替尼
  │
  ├─ BRAF V600E → 达拉非尼+曲美替尼
  │
  ├─ MET 14号外显子跳跃 → 卡马替尼或特泊替尼
  │
  ├─ RET重排 → 塞尔帕替尼或普拉西替尼
  │
  ├─ NTRK融合 → 拉罗替尼或恩曲替尼
  │
  ├─ KRAS G12C → 索托雷西或阿达格拉西布（若无其他选项）
  │
  └─ 无可操作改变 → 进入步骤3

步骤3：PD-L1检测结果？
  ├─ PD-L1≥50%（TPS）
  │  ├─ 选项1：帕博利珠单抗200mg每3周（单药，NCCN 1类推荐）
  │  ├─ 选项2：帕博利珠单抗+铂类双药化疗
  │  └─ 选项3：阿替利珠单抗+贝伐珠单抗+卡铂+紫杉醇
  │
  ├─ PD-L1 1-49%（TPS）
  │  ├─ 首选：帕博利珠单抗+铂类双药化疗
  │  └─ 替代：铂类双药化疗单药
  │
  └─ PD-L1 <1%（TPS）
     ├─ 首选：帕博利珠单抗+铂类双药化疗
     └─ 替代：铂类双药化疗±贝伐珠单抗

步骤4：铂类双药选择（如适用）
  ├─ 鳞状细胞癌组织学
  │  └─ 卡铂AUC 6 + 紫杉醇200 mg/m² 每3周（4个周期）
  │      或 卡铂AUC 5 + 白蛋白结合型紫杉醇100 mg/m² D1,8,15 每4周
  │
  └─ 非鳞状细胞癌组织学
     └─ 卡铂AUC 6 + 培美曲塞500 mg/m² 每3周（4个周期）
         如有反应则继续培美曲塞维持治疗
         如符合条件则加用贝伐珠单抗15 mg/kg（无咯血、脑转移）

步骤5：监测和疗效评估
  - 前12周每6周 imaging，之后每9周
  - 持续至进展或不可接受的毒性
  - 进展时进入二线算法
```

**心力衰竭管理算法（AHA/ACC指南）**

```
入口：心力衰竭诊断已确认

步骤1：确定HF类型
  ├─ HFrEF（EF≤40%）
  │  └─ 进入指南指导的药物治疗（GDMT）
  │
  ├─ HFpEF（EF≥50%）
  │  └─ 治疗合并症，利尿剂缓解充血，考虑SGLT2i
  │
  └─ HFmrEF（EF 41-49%）
     └─ 考虑HFrEF GDMT，证据不太充分

步骤2：HFrEF的GDMT（除非有禁忌，所有患者）

四联疗法（1类推荐）：

1. ACE抑制剂/ARB/ARNI
   ├─ 首选：沙库比曲缬沙坦49/51mg每日两次 → 滴定至97/103mg每日两次
   │  └─ 如为ACE-I初治或服用<10mg依那普利等价剂量
   ├─ 替代：ACE-I（依那普利、赖诺普利、雷米普利）至目标剂量
   └─ 替代：ARB（氯沙坦、缬沙坦）如不耐受ACE-I

2. β受体阻滞剂（低剂量开始，缓慢滴定）
   ├─ 比索洛尔1.25mg每日 → 目标10mg每日
   ├─ 琥珀酸美托洛尔12.5mg每日 → 目标200mg每日
   └─ 卡维地洛3.125mg每日两次 → 目标25mg每日两次（>85kg则50mg每日两次）

3. 盐皮质激素受体拮抗剂（MRA）
   ├─ 螺内酯12.5-25mg每日 → 目标50mg每日
   └─ 依普利酮25mg每日 → 目标50mg每日
   └─ 禁忌症：K>5.0，CrCl<30 mL/min

4. SGLT2抑制剂（无论糖尿病状态）
   ├─ 达格列净10mg每日
   └─ 恩格列净10mg每日

步骤3：根据表型的额外治疗

├─ 窦性心律+尽管使用β受体阻滞剂HR≥70？
│  └─ 是：添加伊伐布雷定5mg每日两次 → 目标7.5mg每日两次
│
├─ 非裔美国人+NYHA III-IV？
│  └─ 是：添加肼苯达嗪37.5mg每日三次 + 硝酸异山梨酯20mg每日三次
│           （目标：肼苯达嗪75mg每日三次 + ISDN 40mg每日三次）
│
├─ 心房颤动？
│  ├─ 心率控制（静息目标<80 bpm，活动时<110 bpm）
│  └─ 抗凝（首选DOAC，如为瓣膜性则用华法林）
│
└─ 铁缺乏（铁蛋白<100或<300伴TSAT<20%）？
   └─ 是：静脉铁补充（羧基麦芽糖铁）

步骤4：设备治疗评估

├─ EF≤35%，NYHA II-III，LBBB伴QRS≥150 ms，窦性心律？
│  └─ 是：心脏再同步治疗（CRT-D）
│
├─ EF≤35%，NYHA II-III，使用GDMT≥3个月？
│  └─ 是：一级预防ICD
│           （如预期寿命>1年且功能状态良好）
│
└─ EF≤35%，NYHA IV尽管GDMT，或晚期HF？
   └─ 转诊至晚期HF专科
      ├─ LVAD评估
      ├─ 心脏移植评估
      └─ 姑息治疗会诊

步骤5：监测和滴定

滴定期间每周至每两周就诊：
- 血压（目标SBP≥90 mmHg）
- 心率（目标50-60 bpm）
- 钾（目标4.0-5.0 mEq/L，如>5.5则暂停MRA）
- 肌酐（预期增加10-30%，如<30%且稳定可接受）
- 症状和充血状态（每日体重、NYHA分级）

GDMT稳定后：
- 每3-6个月就诊
- GDMT优化后3-6个月超声心动图，之后每年
- NT-proBNP或BNP趋势（生物标志物引导的治疗正在研究中）
```

## 风险分层工具

### 心血管风险评分

**TIMI风险评分（NSTEMI/不稳定型心绞痛）**

```
评分计算（0-7分）：
☐ 年龄≥65岁（1分）
☐ ≥3个心脏危险因素（高血压、高脂血症、糖尿病、吸烟、家族史）（1）
☐ 已知CAD（狭窄≥50%）（1）
☐ 过去7天使用ASA（1）
☐ 严重心绞痛（24小时内≥2次发作）（1）
☐ ST段改变≥0.5 mm（1）
☐ 心脏生物标志物升高（1）

风险分层：
├─ 评分0-1：14天内死亡/MI/紧急血运重建5%风险（低）
│  └─ 处理：观察、负荷试验、门诊随访
│
├─ 评分2：8%风险（低-中危）
│  └─ 处理：住院、药物治疗、负荷影像学
│
├─ 评分3-4：13-20%风险（中-高危）
│  └─ 处理：住院、积极药物治疗、早期侵入策略
│
└─ 评分5-7：26-41%风险（高）
   └─ 处理：积极治疗，紧急血管造影（<24小时）
```

**CHA2DS2-VASc评分（房颤卒中风险）**

```
评分计算：
☐ 充血性心力衰竭（1分）
☐ 高血压（1）
☐ 年龄≥75岁（2）
☐ 糖尿病（1）
☐ 既往卒中/TIA/血栓栓塞（2）
☐ 血管疾病（MI、PAD、主动脉斑块）（1）
☐ 年龄65-74岁（1）
☐ 性别类别（女性）（1）

最高评分：9分

治疗算法：
├─ 评分0（男性）或1（女性）：0-1.3%年卒中风险
│  └─ 不抗凝或阿司匹林（IIb类）
│
├─ 评分1（男性）：1.3%年卒中风险
│  └─ 考虑抗凝（IIa类）
│      因素：患者偏好、出血风险、合并症
│
└─ 评分≥2（男性）或≥3（女性）：≥2.2%年卒中风险
   └─ 推荐抗凝（I类）
      ├─ 首选：DOAC（阿哌沙班、利伐沙班、依度沙班、达比加群）
      └─ 替代：华法林（INR 2-3）如DOAC禁忌

出血风险评估（HAS-BLED）：
H - 高血压（SBP>160）
A - 异常肾/肝功能（各1分）
S - 卒中史
B - 出血史或易感性
L - 不稳定INR（如使用华法林）
E - 老年人（年龄>65）
D - 药物（抗血小板、NSAIDs）或酒精（各1分）

HAS-BLED≥3：高出血风险 → 可修改因素，考虑DOAC而非华法林
```

### 肿瘤学风险计算器

**MELD评分（肝细胞癌Eligibility）**

```
MELD = 3.78×ln(胆红素 mg/dL) + 11.2×ln(INR) + 9.57×ln(肌酐 mg/dL) + 6.43

解读：
├─ MELD <10：3个月死亡率1.9%（低）
│  └─ 考虑切除或消融治疗HCC
│
├─ MELD 10-19：3个月死亡率6-20%（中度）
│  └─ 如符合米兰标准则进行移植评估
│      米兰：单个≤5cm或≤3个病灶每个≤3cm，无血管侵犯
│
├─ MELD 20-29：3个月死亡率20-45%（高）
│  └─ 紧急移植评估，桥接治疗（TACE、消融）
│
└─ MELD ≥30：3个月死亡率50-70%（非常高）
   └─ 移植 vs 姑息治疗讨论
      如MELD>35-40通常太虚弱不适合移植
```

**Adjuvant! Online（乳腺癌复发风险）**

```
输入变量：
- 诊断时年龄
- 肿瘤大小
- 肿瘤分级（1-3）
- ER状态
- 淋巴结状态（0、1-3、4-9、≥10）
- HER2状态
- 合并症指数

输出：10年风险：
- 复发
- 乳腺癌死亡
- 总死亡

治疗获益估算：
- 化疗：复发绝对减少
- 内分泌治疗：复发绝对减少
- 曲妥珠单抗：绝对减少（如果HER2+）

临床应用：
├─ 低危（复发<10%）：如ER+考虑仅内分泌治疗
├─ 中危（10-20%）：化疗讨论，基因组检测
│  └─ Oncotype DX评分<26：仅内分泌治疗
│  └─ Oncotype DX评分≥26：化疗+内分泌治疗
└─ 高危（复发>20%）：如ER+化疗+内分泌治疗
```

## TikZ流程图最佳实践

### 视觉设计原则

**节点样式**
```latex
% 决策节点（菱形）
\tikzstyle{decision} = [diamond, draw, fill=yellow!20, text width=4.5em, text centered, inner sep=0pt]

% 处理节点（矩形）
\tikzstyle{process} = [rectangle, draw, fill=blue!20, text width=5em, text centered, rounded corners, minimum height=3em]

% 终止节点（圆角矩形）
\tikzstyle{terminal} = [rectangle, draw, fill=green!20, text width=5em, text centered, rounded corners=1em, minimum height=3em]

% 输入/输出（平行四边形）
\tikzstyle{io} = [trapezium, draw, fill=purple!20, text width=5em, text centered, minimum height=3em]
```

**按紧急程度配色**
- **红色**：危及生命，需要立即行动
- **橙色**：紧急，小时内行动
- **黄色**：半紧急，24-48小时内行动
- **绿色**：常规，临床情况稳定
- **蓝色**：信息性，仅监测

**路径强调**
- 最常见路径用粗箭头
- 罕见情况用虚线箭头
- 箭头粗细与路径频率成比例
- 关键决策点周围加高亮框

## 算法验证

### 开发过程

**步骤1：文献综述和证据综合**
- 系统综述指南（NCCN、ASCO、ESMO、AHA/ACC）
- 临床试验荟萃分析
- 专家共识声明
- 当地实践模式和资源可用性

**步骤2：算法草案开发**
- 多学科团队投入（医生、护士、药剂师）
- 定义决策节点和标准
- 指定行动和结局
- 识别不确定领域

**步骤3：试点测试**
- 回顾性应用于历史病例（n=20-50）
- 识别算法未覆盖的场景
- 完善决策标准
- 终端用户可用性测试

**步骤4：前瞻性验证**
- 在临床实践中实施并收集数据
- 跟踪依从率（目标>80%）
- 监测结局与历史对照
- 用户满意度调查

**步骤5：持续质量改进**
- 每季度审查算法性能
- 根据新证据更新
- 解决偏离和依从性差的原因
- 版本控制和变更记录

### 性能指标

**过程指标**
- 算法依从率（遵循算法的病例百分比）
- 决策时间（从就诊到治疗开始的中位时间）
- 完成率（到达终止节点的病例百分比）

**结局指标**
- 护理适当性（与指南的一致性）
- 临床结局（死亡率、发病率、再入院率）
- 资源利用（住院时间、不必要检查）
- 安全性（不良事件、错误）

**用户体验指标**
- 易用性（Likert量表调查）
- 使用时间（导航算法中位时间）
- 感知有用性（报告算法有用的用户百分比）
- 使用障碍（定性反馈）

## 实施策略

### 融入临床工作流程

**电子健康记录集成**
- 关键决策点的临床决策支持（CDS）警报
- 链接到算法路径的医嘱套餐
- 从EHR数据自动填充风险评分
- 遵循算法结构的文档模板

**床旁工具**
- 快速参考口袋卡
- 交互式算法移动应用
- 临床区域墙面海报
- 链接到完整算法的二维码

**教育和培训**
- 算法原理讲座
- 病例练习
- 模拟场景
- 依从性审计和反馈

### 克服障碍

**常见障碍**
- 算法复杂性（决策点过多）
- 缺乏认知（未有效传播）
- 不同意建议（被视为照本宣科）
- 竞争优先事项（时间压力、多名患者）
- 资源限制（推荐检查/治疗不可用）

**缓解策略**
- 简化算法（每条路径首选≤7个决策点）
- champion网络（当地意见领袖推广算法）
- 根据当地情况定制（允许临床判断的灵活性）
- 衡量和报告结局（展示价值）
- 提供资源（确保算法推荐选项可用）

### 更新触发条件

**强制更新（3个月内）**
- FDA批准算法适应症的新药
- 指南变更（NCCN、ASCO、ESMO 1类推荐）
- 推荐药物增加安全警报或黑框警告
- 改变护理标准的主要临床试验结果

**常规更新（每年）**
- 小证据更新
- 基于当地性能数据优化
- 格式或可用性改进
- 添加遇到的新临床场景

**紧急更新（1周内）**
- 药物短缺需要替代路径
- 药物召回或安全撤市
- 爆发或大流行需要修改方案
