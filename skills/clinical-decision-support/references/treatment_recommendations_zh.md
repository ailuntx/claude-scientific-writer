# 治疗建议指南

## 概述

循证治疗建议为临床医生提供治疗决策的系统性指导。本指南涵盖制药和医疗环境中临床建议的制定、分级和呈现。

## 证据分级系统

### GRADE（推荐分级、评估、开发和评价）

**证据质量等级**

**高质量（⊕⊕⊕⊕）**
- 进一步研究极不可能改变对估计值的信心
- 标准：设计良好的RCT，结果一致，无严重局限性
- 示例：多项大型RCT显示相似的治疗效果

**中等质量（⊕⊕⊕○）**
- 进一步研究很可能对信心产生重要影响
- 标准：存在局限性的RCT或来自观察性研究的非常强证据
- 示例：单项RCT或多项存在一些不一致的RCT

**低质量（⊕⊕○○）**
- 进一步研究很可能对信心产生重要影响
- 标准：观察性研究或存在严重局限性的RCT
- 示例：病例对照研究、有混杂因素的队列研究

**极低质量（⊕○○○）**
- 效果估计非常不确定
- 标准：病例系列、专家意见或非常严重的局限性
- 示例：机制推理、非系统性临床观察

**推荐强度**

**强推荐（1级）**
- 收益明显超过风险和负担（或反之）
- 用词："我们推荐..."
- 含义：大多数患者应接受推荐方案
- 符号：↑↑（强支持）或 ↓↓（强反对）

**有条件/弱推荐（2级）**
- 存在权衡；收益和风险密切相关
- 用词："我们建议..."
- 含义：不同患者有不同选择；共同决策
- 符号：↑（弱支持）或 ↓（弱反对）

**GRADE标注示例**
- **1A**：强推荐，高质量证据
- **1B**：强推荐，中等质量证据
- **2A**：弱推荐，高质量证据
- **2B**：弱推荐，中等质量证据
- **2C**：弱推荐，低质量或极低质量证据

### 牛津循证医学中心（CEBM）等级

**1级：系统评价/荟萃分析**
- 1a：RCT的系统评价
- 1b：置信区间窄的单个RCT
- 1c："全或无"研究（治疗前所有患者死亡，治疗后部分存活）

**2级：队列研究**
- 2a：队列研究的系统评价
- 2b：单个队列研究（包括低质量RCT）
- 2c：结局研究、生态学研究

**3级：病例对照研究**
- 3a：病例对照研究的系统评价
- 3b：单个病例对照研究

**4级：病例系列**
- 病例系列、低质量队列或病例对照研究

**5级：专家意见**
- 基于机制的推理、无批判性评估的专家意见

**推荐等级**
- **A级**：一致的水平1研究
- **B级**：一致的水平2或3研究，或从水平1外推
- **C级**：水平4研究或从水平2或3外推
- **D级**：水平5证据或不一致/不确定的研究

## 治疗顺序和一线治疗

### 一线治疗

**选择标准**
- **标准治疗**：基于3期试验的指南推荐
- **患者因素**：体能状态、合并症、器官功能
- **疾病因素**：分期、分子特征、侵袭性
- **目标**：治愈（辅助/新辅助）、延长缓解、症状控制

**一线方案文档**
```
一线治疗方案：

方案1：方案A（NCCN 1类，ESMO I-A）
- 证据：3期RCT（n=1000），中位PFS 12个月 vs 8个月（HR 0.6，p<0.001）
- 人群：PD-L1 ≥50%，EGFR/ALK阴性
- 毒性特征：免疫相关不良事件（15% 3-4级）
- 推荐强度：1A（强推荐，高质量证据）

方案2：方案B（NCCN 1类，ESMO I-A）
- 证据：3期RCT（n=800），中位PFS 10个月 vs 8个月（HR 0.7，p=0.003）
- 人群：所有患者，无生物标志物选择
- 毒性特征：血液学毒性（25% 3-4级）
- 推荐强度：1A（强推荐，高质量证据）
```

### 二线及后续治疗

**二线选择**
- **既往疗效**：一线治疗的反应持续时间
- **进展模式**：寡进展 vs 广泛进展
- **残留毒性**：从一线毒性中恢复
- **生物标志物演变**：获得性耐药机制
- **临床试验可用性**：开发中的新药

**治疗史文档**
```
既往治疗：
1. 一线：帕博利珠单抗（12个周期）
   - 最佳疗效：部分缓解（肿瘤负荷减少45%）
   - PFS：14个月
   - 停药原因：疾病进展
   - 残留毒性：1级甲状腺功能减退（左甲状腺素治疗中）

2. 二线：多西他赛+雷莫芦单抗（6个周期）
   - 最佳疗效：疾病稳定
   - PFS：5个月
   - 停药原因：疾病进展
   - 残留毒性：2级周围神经病变

当前考虑：三线方案
- 临床试验 vs 铂类化疗
```

### 维持治疗

**指征**
- 诱导治疗缓解后的巩固
- 无持续细胞毒治疗下的进展预防
- 桥接确定性治疗（如移植）

**证据要求**
- 随机试验显示PFS获益
- 可耐受的长期毒性特征
- 生活质量维持或改善

## 生物标志物指导的治疗选择

### 伴随诊断

**FDA批准的生物标志物-药物配对**

**必需检测（治疗特异性）**
- **ALK重排 → 阿来替尼、布格替尼、洛拉替尼**（NSCLC）
- **EGFR 19外显子缺失/L858R → 奥希替尼**（NSCLC）
- **BRAF V600E → 达拉非尼+曲美替尼**（黑色素瘤、NSCLC、CRC）
- **HER2扩增/3+ → 曲妥珠单抗、帕妥珠单抗**（乳腺癌、胃癌）
- **PD-L1 ≥50% → 帕博利珠单抗单药**（NSCLC一线）

**补充诊断（信息性但非必需）**
- **PD-L1 1-49%**：优先考虑联合免疫治疗
- **TMB高**：可能预测免疫治疗获益（研究性）
- **MSI-H/dMMR**：帕博利珠单抗获批用于多种肿瘤类型

### 生物标志物检测算法

**NSCLC生物标志物组合**
```
诊断时反射性检测：
✓ EGFR突变（18、19、20、21外显子）
✓ ALK重排（IHC或FISH）
✓ ROS1重排（FISH或NGS）
✓ BRAF V600E突变
✓ PD-L1 IHC（22C3或SP263）
✓ 考虑：综合NGS组合

如果奥希替尼进展后EGFR+：
✓ 液体活检检测T790M（如果使用一代/二代TKI）
✓ 组织活检检测耐药机制
✓ MET扩增、HER2扩增、小细胞肺癌转化
```

**乳腺癌生物标志物算法**
```
初始诊断：
✓ ER/PR IHC
✓ HER2 IHC和FISH（如果2+）
✓ Ki-67增殖指数

如果转移性ER+/HER2-：
✓ ESR1突变（AI进展后液体活检）
✓ PIK3CA突变（阿培利司 eligibility）
✓ BRCA1/2胚系检测（PARP抑制剂 eligibility）
✓ PD-L1检测（如果考虑免疫治疗联合）
```

### 可操作突变

**I级：FDA批准的靶向治疗**
- 前瞻性试验的强证据
- 指南推荐
- 示例：EGFR 19外显子缺失、HER2扩增、ALK融合

**II级：临床试验或超适应症使用**
- 新兴证据，优先临床试验
- 示例：NTRK融合（拉罗替尼）、RET融合（塞尔帕替尼）

**III级：生物学合理性**
- 仅临床前证据
- 强烈建议入组临床试验
- 示例：新型激酶融合、罕见耐药突变

## 联合治疗方案

### 联合的依据

**机制**
- **非重叠毒性**：最大化每种药物的剂量强度
- **协同活性**：超出加法效果的增强疗效
- **互补机制**：同时靶向多个通路
- **预防耐药**：降低耐药克隆的选择压力

**联合设计原则**
- **序贯**：诱导后巩固（不同方案）
- **同步**：同时给药以产生协同
- **交替**：轮换方案以最小化耐药
- **间歇**：脉冲给药 vs 持续暴露

### 药物相互作用评估

**药代动力学相互作用**
- **CYP450诱导/抑制**：检查药物-药物相互作用
- **转运体相互作用**：P-gp、BCRP、OATP底物/抑制剂
- **蛋白结合**：高蛋白结合药物（华法林需注意）
- **肾/肝清除**：避免多种经肾清除的药物

**药效学相互作用**
- **叠加毒性**：避免重叠不良事件（如QTc延长）
- **拮抗**：确保机制互补而非对立
- **剂量调整**：联合治疗的预定剂量减量方案

### 联合方案文档

```
联合方案：药物A + 药物B

依据：
- 3期RCT显示PFS获益（16 vs 11个月，HR 0.62，p<0.001）
- 互补机制：药物A（VEGF抑制剂）+ 药物B（免疫检查点抑制剂）
- 非重叠毒性特征

剂量：
- 药物A：10 mg/kg IV每3周一次
- 药物B：1200 mg IV每3周一次
- 持续至进展或不可接受的毒性

主要毒性：
- 高血压（药物A）：30% 3-4级，用抗高血压药物管理
- 免疫相关不良事件（药物B）：15% 3-4级，糖皮质激素管理
- 未观察到显著的药代动力学相互作用

监测：
- 血压：第一个月每天一次，之后每周一次
- 甲状腺功能：每6周一次
- 肝功能：每个周期前
- 影像：每6周一次（RECIST v1.1）
```

### 剂量调整指南

**血液学毒性**
```
中性粒细胞和血小板计数        操作
中性粒细胞≥1.5且血小板≥100k  全量治疗
中性粒细胞1.0-1.5或血小板75-100k  延迟1周，复查
中性粒细胞0.5-1.0或血小板50-75k   延迟治疗，G-CSF支持，减量20%
中性粒细胞<0.5或血小板<50k       暂停治疗，G-CSF，必要时输血，减量40%

发热性中性粒细胞减少          暂停治疗，住院，抗生素，G-CSF
                                恢复后减量20-40%，考虑预防性G-CSF
```

**非血液学毒性**
```
不良事件       1级          2级              3级              4级
腹泻          继续         继续+洛哌丁胺    暂停至≤G1，减量20%  暂停，住院考虑停药
皮疹          继续         继续+局部用药    暂停至≤G1，减量20%  停药
肝毒性        继续         1周后复查，如加重暂停  暂停至≤G1，减量20-40% 永久停药
肺炎          继续         暂停，考虑糖皮质激素  暂停，糖皮质激素，如无改善停药  停药，高剂量糖皮质激素
```

## 特殊人群

### 老年患者（≥65-70岁）

**考虑因素**
- **器官功能减退**：肾/肝功能不全时调整
- **多重用药**：药物相互作用风险
- **衰弱**：老年评估（G8、VES-13、CARG评分）
- **治疗目标**：生活质量 vs 生存、功能独立性

**调整**
- 剂量减量：衰弱患者减少20-25%
- 延长间隔：每4周而非每3周
- 较温和方案：单药 vs 联合治疗
- 支持治疗：加强监测、G-CSF预防

### 肾功能损害

**按eGFR调整剂量**
```
eGFR (mL/min/1.73m²)  类别    操作
≥90                   正常    标准剂量
60-89                 轻度    标准剂量（大多数药物）
30-59                 中度    经肾清除药物减量25-50%
15-29                 重度    减量50-75%，避免肾毒性药物
<15（透析）           终末期肾病  避免大多数药物，个案决定
```

**需要调整的经肾清除药物**
- 卡铂（Calvert公式：AUC × [GFR + 25]）
- 甲氨蝶呤（如果CrCl <60减量50-75%）
- 卡培他滨（如果CrCl 30-50减量25-50%）

### 肝功能损害

**按胆红素和AST/ALT调整剂量**
```
类别            胆红素         AST/ALT        操作
正常            ≤ULN          ≤ULN           标准剂量
轻度（Child A）  1-1.5× ULN    任何           经肝代谢药物减量25%
中度（Child B） 1.5-3× ULN    任何           减量50%，考虑替代方案
重度（Child C） >3× ULN       任何           避免大多数药物，个案决定
```

**需要调整的经肝代谢药物**
- 多西他赛（如果胆红素升高减量25-50%）
- 伊立替康（如果胆红素1.5-3× ULN减量50%）
- 酪氨酸激酶抑制剂（大多数经CYP3A4代谢，减量50%）

### 妊娠和生育

**避孕要求**
- 治疗期间及治疗后6-12个月需有效避孕
- 高度致畸药物建议两种方法
- 男性患者：如果配偶有生育潜力需避孕

**生育力保护**
- 卵子/胚胎冷冻保存（女性，促性腺毒性治疗前）
- 精子库（男性，烷化剂、铂类治疗前）
- GnRH激动剂（卵巢抑制，疗效存在争议）
- 治疗前转诊至生殖内分泌科

**妊娠管理**
- 避免孕早期化疗（器官形成期）
- 选择性药物在孕中晚期安全（个案决定）
- 多学科团队：肿瘤科、母胎医学、新生儿科

## 临床试验考虑

### 何时推荐临床试验

**理想情况**
- 无标准治疗可用（罕见病、难治性设置）
- 多个等效标准方案（患者偏好新药）
- 标准治疗失败（二线及后续）
- 高风险疾病（辅助试验改善结局）

**试验选择标准**
- **阶段**：1期（剂量探索、安全性）、2期（疗效信号）、3期（比较有效性）
- **资格**：匹配患者的入排标准
- **机制**：新机制 vs 已确立机制，生物学原理
- **赞助者**：学术 vs 行业，试验设计质量
- **物流**：到试验地点的距离、访视频率、自费费用

### 共同决策

**告知患者**
- 不治疗的自然病程
- 标准治疗选项及证据、收益、风险
- 临床试验选项（如有）
- 治疗目标一致
- 患者价值观和偏好

**决策辅助**
- 收益的可视化表示（图标阵列）
- 需要治疗人数计算
- 生活质量权衡
- 决策冲突量表

## 文档标准

### 治疗计划文档

```
治疗计划

诊断：[疾病、分期、分子特征]

治疗目标：
☐ 治愈目的
☐ 延长疾病控制
☑ 姑息和生活质量
```
