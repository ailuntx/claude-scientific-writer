## 色盲安全配色方案

这些配色方案专为各种形式的色觉缺陷人群设计，确保可区分性。

### Wong 配色方案（7种颜色）

最广泛推荐的色盲安全配色方案，由 Bang Wong 为科学可视化开发。

| 名称 | 十六进制 | RGB | 用途 |
|------|---------|-----|------|
| 黑色 | `#000000` | 0, 0, 0 | 文字、轮廓 |
| 橙色 | `#E69F00` | 230, 159, 0 | 主强调色 |
| 天蓝色 | `#56B4E9` | 86, 180, 233 | 主数据色 |
| 蓝绿色 | `#009E73` | 0, 158, 115 | 次要数据 |
| 黄色 | `#F0E442` | 240, 228, 66 | 高亮 |
| 蓝色 | `#0072B2` | 0, 114, 178 | 主分类 |
| 朱红色 | `#D55E00` | 213, 94, 0 | 警报、强调 |
| 紫红色 | `#CC79A7` | 204, 121, 167 | 第三数据 |

**提示词用法：**
```
"Use Wong's colorblind-safe palette: orange (#E69F00), sky blue (#56B4E9), 
bluish green (#009E73), and blue (#0072B2) for data categories"
```

## 对比度检查

### WCAG 2.1 要求

| 对比度 | 要求 |
|--------|------|
| 4.5:1 | 正常文字（18pt以下） |
| 3:1 | 大号文字（18pt以上或14pt粗体） |
| 3:1 | 图形和UI组件 |

### 常见安全组合

**白色背景 (#FFFFFF) 上：**
| 文字颜色 | 十六进制 | 对比度 |
|----------|---------|--------|
| 黑色 | `#000000` | 21:1 ✓ |
| 深灰色 | `#333333` | 12.6:1 ✓ |
| 海军蓝 | `#1E3A5F` | 11.2:1 ✓ |
| 深绿色 | `#1B4332` | 10.9:1 ✓ |
| 深蓝色 | `#0072B2` | 5.7:1 ✓ |
| 中灰色 | `#666666` | 5.7:1 ✓ |
| 红色 | `#CC0000` | 5.5:1 ✓ |

**深色背景 (#1A1A2E) 上：**
| 文字颜色 | 十六进制 | 对比度 |
|----------|---------|--------|
| 白色 | `#FFFFFF` | 17.1:1 ✓ |
| 浅灰色 | `#E5E5E5` | 13.8:1 ✓ |
| 浅青色 | `#90E0EF` | 10.2:1 ✓ |
| 黄色 | `#F0E442` | 12.5:1 ✓ |
| 浅蓝色 | `#56B4E9` | 7.8:1 ✓ |

### 需要避免的组合

这些组合对比度差或对色盲用户有问题：

- 红色和绿色（最常见的色盲）
- 蓝色和紫色（难以区分）
- 浅绿色和黄色（对比度低）
- 红色和橙色（色相相似）
- 蓝色和灰色（可能混淆）
- 粉色和灰色（数值相似）

## 测试您的颜色

   - WebAIM Contrast Checker: https://webaim.org/resources/contrastchecker/
   - Coolors Contrast Checker: https://coolors.co/contrast-checker

2. **色盲模拟器：**
   - Coblis: https://www.color-blindness.com/coblis-color-blindness-simulator/

   - Color Oracle (桌面应用)

3. **配色生成器：**
   - Coolors: https://coolors.co/
   - Adobe Color: https://color.adobe.com/
   - Paletton: https://paletton.com/

### 快速灰度测试

将您的信息图表转换为灰度。如果所有元素仍然可区分，那么您的颜色选择是无障碍的。
