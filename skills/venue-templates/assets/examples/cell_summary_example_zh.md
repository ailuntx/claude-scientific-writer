## 完整示例1：衰老与衰老

### 摘要（最多150词）

```
细胞衰老是一种防止受损细胞增殖的应激反应，但可通过衰老相关分泌表型（SASP）驱动组织功能障碍。衰老细胞如何在表达促凋亡p53的情况下抵抗凋亡一直不清楚。在此，我们发现FOXO4是衰老细胞存活的关键介导因子。FOXO4在衰老细胞中高度表达，并直接与p53相互作用，将其保留在细胞核内，防止p53介导的凋亡。一种破坏FOXO4-p53相互作用的细胞可渗透肽可选择性诱导p53核排除和衰老细胞凋亡，而不影响增殖细胞。在体内，这种FOXO4肽可中和阿霉素诱导的衰老细胞，并恢复自然衰老小鼠的Fitness、毛皮密度和肾功能。这些发现确立了FOXO4介导的p53隔离作为衰老特异性存活途径，并证明了靶向消除衰老细胞的治疗潜力。
```

### 亮点（每条≤85个字符）

```
• FOXO4在衰老细胞中选择性上调并与p53结合

• FOXO4-p53相互作用将p53保留在细胞核内，防止凋亡

• 靶向FOXO4的肽特异性诱导衰老细胞凋亡

• FOXO4肽治疗恢复衰老小鼠的Fitness和器官功能
```

### eTOC简讯（30-50词）

```
Bar等通过p53隔离鉴定FOXO4为衰老细胞存活的关键介导因子。一种破坏FOXO4-p53相互作用的肽可选择性消除衰老细胞并恢复衰老小鼠的组织功能，确立了靶向衰老溶解治疗的可行性概念。
```

### 简述（1句）

```
靶向FOXO4的肽通过释放p53选择性消除衰老细胞，恢复衰老小鼠的组织功能。
```

## 完整示例2：基因组组织

### 摘要（最多150词）

```
染色体在细胞核内的三维组织影响基因表达、DNA复制和基因组稳定性。相分离已成为组织核内容物的潜在机制，但凝聚物是否能在体内塑造染色体结构仍不清楚。在此，我们表明转录共激活因子BRD4在超级增强子处形成液滴状凝聚物，将相关染色质组织成集线结构。BRD4凝聚物的光遗传学诱导可在数分钟内重塑染色体拓扑并激活转录。相反，用小分子JQ1破坏BRD4凝聚物可解离染色质集线并快速沉默超级增强子控制的基因。单分子追踪显示，凝聚物形成使转录 machinery的局部浓度增加100倍，解释了超级增强子的转录强力。这些结果确立了相分离作为染色质组织和转录控制的机制，对理解和靶向致癌性超级增强子具有意义。
```

### 亮点

```
• BRD4在活细胞超级增强子处形成液滴凝聚物

• BRD4凝聚物将染色质组织成转录活性集线

• 光遗传学凝聚物诱导快速重塑染色质拓扑

• 凝聚物使转录 machinery在局部浓度增加100倍
```

### eTOC简讯

```
Sabari等证明BRD4在超级增强子处形成相分离凝聚物，将染色质组织成集线结构并浓集转录 machinery。光遗传学操作显示，凝聚物形成直接驱动染色质重塑和转录激活。
```

## 完整示例3：代谢与免疫

### 摘要（最多150词）

```
免疫细胞在激活时经历剧烈的代谢重编程，从氧化磷酸化切换到有氧糖酵解。这种代谢转变被认为支持快速增殖的生物合成需求，但特定代谢物是否直接调节免疫细胞功能仍基本未知。在此，我们表明糖酵解代谢物磷酸烯醇式丙酮酸（PEP）通过抑制肌浆/内质网钙ATP酶（SERCA）活性来维持T细胞受体信号。PEP在活化的T细胞中积累并直接与SERCA结合，防止钙再摄取并延长储存操作钙进入。基因或药理学增强PEP水平可增强T细胞效应功能和体内抗癌免疫。相反，肿瘤来源的乳酸抑制PEP水平并损害T细胞钙信号，有助于肿瘤免疫逃逸。这些发现揭示了糖酵解中间体的意外信号作用，并建议代谢策略来增强癌症免疫治疗中的T细胞反应。
```

### 亮点

```
• 磷酸烯醇式丙酮酸（PEP）在T细胞激活期间积累

• PEP直接结合并抑制SERCA以维持钙信号

• 增强PEP水平增强抗肿瘤T细胞免疫

• 肿瘤乳酸抑制T细胞PEP水平和钙信号
```

### eTOC简讯

```
Ho等发现糖酵解代谢物磷酸烯醇式丙酮酸通过抑制SERCA直接调节T细胞钙信号。这种代谢-信号关联被肿瘤通过乳酸分泌所利用，为癌症免疫治疗提供了新靶点。
```

## Cell元素写作技巧

### 摘要技巧

1. **第一句**：建立生物学背景
2. **第二句**：说明未知内容（差距）
3. **"在此，我们展示/鉴定/证明"**：清晰过渡到您的工作
4. **中间句**：机制性关键发现
5. **最后一句**：意义和启示

### 亮点技巧

- **以名词或动词开头**："FOXO4形成..."或"...的激活"
- **每条一个发现**：不要合并多个要点
- **具体**：包含蛋白质/基因/通路名称
- **检查字符数**：严格≤85个字符（含空格）
- **覆盖不同发现**：不要重复相同要点

### eTOC简讯技巧

- **以作者姓名开头**："Smith等显示..."
- **仅一或两句话**：保持简洁
- **包含关键机制**：不仅是发现
- **以意义结尾**：为什么读者应该关注
