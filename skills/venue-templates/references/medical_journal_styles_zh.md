# 医学期刊写作风格指南

面向《新英格兰医学杂志》、《柳叶刀》、JAMA、BMJ、《内科学年鉴》及其他主要医学期刊的综合写作指南。

**最后更新**：2024年

---

## 概述

医学期刊优先考虑**临床相关性**、**患者结局**和**循证实践**。写作必须精确、以证据为基础，并可直接应用于临床决策。

### 核心理念

> "每一句话都应帮助临床医生为其患者做出更好的决策。"

**主要目标**：传达可改善患者护理和临床实践的研究发现。

---

## 受众与语气

### 目标读者

- 执业医师和临床医生
- 临床研究人员
- 医疗卫生政策制定者
- 医学教育者
- 部分公共卫生和患者权益倡导读者

### 语气特征

| 特征 | 描述 |
|---------------|-------------|
| **以证据为中心** | 根据研究设计进行适当的措辞调整 |
| **以患者为中心** | 关注患者结局，而非仅关注生物标志物 |
| **临床导向** | 强调实际适用性 |
| **精确** | 精确数字、可信区间、NNT |
| **审慎** | 主张与证据强度相匹配 |

### 文风

- **常见被动语态**："患者被随机分配至..."
- **可接受第一人称**："我们进行了一项试验..."
- **提及患者用第三人称**："患者"而非"受试者"

## 摘要：结构化格式

### 概述

大多数主要医学期刊要求**结构化摘要**，并带有标注章节。这是结构化摘要被期望的少数场合之一。

### 标准结构（基于IMRAD）

```
背景：[为什么需要这项研究——1-2句话]

方法：[研究设计、参与者、干预措施、主要结局——2-4句话]

结果：[主要和关键次要结局及统计数据——3-5句话]

结论：[临床意义，适当措辞调整——1-2句话]
```

### 各期刊字数限制

| 期刊 | 摘要限制 |
|---------|---------------|
| NEJM | 250词 |
| Lancet | 300词 |
| JAMA | 350词 |
| BMJ | 300词 |
| Annals | 325词 |

### 结构化摘要示例（NEJM风格）

```
背景
2型糖尿病与心血管风险增加相关，但强化血糖控制对心血管结局的影响仍不确定。

方法
我们随机分配了10,251例2型糖尿病和已确定心血管疾病的患者，接受强化降糖治疗（目标HbA1c <6.0%）或标准治疗（目标HbA1c 7.0-7.9%）。主要结局为非致死性心肌梗死、非致死性卒中或心血管原因死亡的复合终点。

结果
中位随访3.5年后，强化治疗组352例患者（6.9%）和标准治疗组371例患者（7.2%）发生了主要结局（风险比，0.90；95% CI，0.78-1.04；P=0.16）。强化治疗组重度低血糖更为常见（3.1% vs. 1.0%；P<0.001）。全因死亡率两组相似（5.0% vs. 4.8%；风险比，1.04；95% CI，0.87-1.24）。

结论
在2型糖尿病和已确定心血管疾病的患者中，与标准治疗相比，强化降糖治疗未显著减少主要心血管事件，且与重度低血糖增加相关。
```

## 证据语言

### 核心原则

**将语言与证据强度相匹配。**

### 按研究设计分类的语言

| 研究设计 | 适当的语言 |
|-------------|---------------------|
| **RCT的Meta分析** | "治疗X降低死亡率..." |
| **大型RCT** | "治疗X在本试验中降低了死亡率..." |
| **小型RCT** | "治疗X与死亡率降低相关..." |
| **队列研究** | "治疗X与死亡率降低相关..." |
| **病例对照** | "治疗X与死亡几率降低相关..." |
| **横断面研究** | "治疗X使用与死亡率降低相关..." |
| **病例系列** | "这些病例提示治疗X可能..." |
| **病例报告** | "本病例说明治疗X可以..." |

### 因果语言规则

❌ **绝不要说**（除非是RCT）："治疗X预防..." / "治疗X导致..."

✅ **用于观察性研究**："治疗X与..."相关 / "治疗X与..."相关

✅ **用于RCT**："治疗X导致..." / "治疗X降低..."

### 措辞调整短语

| 确定性程度 | 短语 |
|----------------|---------|
| **高** | "证实"、"显示"、"确认" |
| **中等** | "提示"、"表明"、"支持" |
| **低** | "可能"、"也许"、"可能" |
| **推测性** | "可能是"、"一种解释是" |

## 数据的报告

### 绝对风险与相对风险

**始终同时报告绝对和相对指标。**

❌ **不完整**："治疗使死亡率降低50%"

✅ **完整**："治疗使相对死亡率降低50%（绝对风险降低2.5个百分点；需要治疗人数，40）"

### 可信区间

**始终包含95%可信区间。**

❌ "风险比为0.75"

✅ "风险比为0.75（95% CI，0.62-0.91）"

### P值

- 可能时报告精确P值：P=0.003
- 对于非常小的值使用P<0.001
- 需结合临床意义和统计学显著性

### 需要治疗人数（NNT）

对于临床重要结局应包含NNT：

```
"该干预每治疗40例患者可预防1例额外死亡
（NNT=40；95% CI，28-67）。"
```

## 引言

### 长度与结构

- **3-4段**（500-700词）
- 聚焦于临床问题和理由

### 段落结构

**第一段：临床问题**
- 疾病负担（发病率、患病率、死亡率）
- 对患者和医疗系统的影响
- 为什么这在临床上重要

```
"2型糖尿病影响全球超过4.5亿成年人，是心血管疾病、肾衰竭和过早死亡的主要原因。尽管降糖治疗取得进展，糖尿病患者仍面临比普通人群高2-4倍的心血管事件风险。"
```

**第二段：现有知识和局限性**
- 存在哪些治疗/方法
- 哪些证据缺口仍然存在
- 为什么需要更多研究

**第三段：理由与目标**
- 为什么进行这项研究
- 明确陈述目标/假设
- 陈述主要结局

```
"因此，我们开展了一项随机对照试验，以评估与标准治疗相比，强化降糖治疗是否能够减少2型糖尿病和已确定心血管疾病患者的主要心血管事件。"
```

## 方法

### 结构（符合CONSORT/STROBE）

医学方法章节遵循报告指南：

```
方法
├── 研究设计
├── 设置和参与者
│   ├── 纳入标准
│   └── 招募
├── 随机化和盲法（针对RCT）
├── 干预措施
├── 结局
│   ├── 主要结局
│   └── 次要结局
├── 样本量计算
├── 统计分析
├── 伦理审批
└── 注册
```

### 关键要素

**纳入标准**
- 明确列出纳入和排除标准
- 具体说明（年龄范围、疾病定义、实验室数值）

**主要结局**
- 精确界定，包括评估时间
- 说明其测量方式

**统计分析**
- 预先设定的分析计划
- 缺失数据的处理
- 亚组分析（预先设定 vs. 探索性）

### 方法段落示例

```
我们纳入年龄40岁或以上的2型糖尿病成人患者（定义为HbA1c ≥6.5%或使用降糖药物）和已确定的心血管疾病（既往心肌梗死、卒中或血运重建手术）。如果患者HbA1c水平低于7.5%或高于11.0%、估算肾小球滤过率低于30 ml/分钟/1.73 m²体表面积，或在过去30天内有心血管事件，则予以排除。"
```

## 结果

### 结构

**开头：参与者流程**
- 筛选、纳入、随机化、随访、分析
- 参考CONSORT流程图

**基线特征**
- 表1：基线人口学和临床特征
- 注明任何不平衡

**主要结局**
- 首先和显著地报告
- 包含点估计、可信区间、P值
- 说明临床意义

**次要结局**
- 报告所有预先设定的次要结局
- 对多重比较持谨慎态度

**不良事件**
- 系统报告严重不良事件
- 按类别包含死亡、住院、SAE

### 结果段落示例

```
在12,537例接受资格评估的患者中，10,251例接受了随机化：5,128例分配至强化治疗组，5,123例分配至标准治疗组（图1）。两组基线特征相似（表1）。中位随访3.5年（四分位距，2.8-4.2），99.2%的患者有生存状态数据。

强化治疗组352例患者（6.9%）和标准治疗组371例患者（7.2%）发生了主要结局（风险比，0.90；95%可信区间[CI]，0.78-1.04；P=0.16）。绝对差异为0.3个百分点（95% CI，-0.7至1.4）。结果在预先设定的亚组中保持一致（图3）。"
```

## 讨论

### 结构

**第一段：主要发现总结**
- 重述主要结局结果
- 说明假设是否得到支持

**第二至三段：解释与背景**
- 发现与先前证据相比如何？
- 什么机制可能解释这些发现？
- 临床解释

**第四段：优势**
- 研究设计特征
- 可推广性
- 随访的完整性

**第五段：局限性**
- 具体且深思熟虑
- 讨论局限性如何影响解释
- 避免笼统陈述

**最后一段：结论与意义**
- 临床意义
- 政策意义
- 未来研究需求

### 局限性段落示例

```
我们的研究有几个局限性。首先，尽管进行了随机化，我们无法排除未测量因素导致的残余混杂。其次，开放标签设计可能引入了主观终点结局评估的偏倚，尽管主要终点死亡是客观的。第三，我们的发现可能无法推广至无已确定心血管疾病的患者，或医疗资源不同的医疗环境。第四，3.5年的随访可能不足以发现更长时间才显现的心血管获益。"
```

## 期刊具体要求

### NEJM（新英格兰医学杂志）

- **字数限制**：2,700词（不包括摘要、参考文献）
- **摘要**：250词，结构化
- **参考文献**：约40-50篇
- **图/表**：合计4-5个
- **风格**：权威性、确定性
- **重点**：主要临床试验、变革性研究

### Lancet

- **字数限制**：研究文章3,500词
- **摘要**：300词，结构化
- **摘要框（Panel）**：突出关键信息
- **研究背景**：必需章节，说明贡献
- **风格**：重视全球健康视角

### JAMA（美国医学会杂志）

- **字数限制**：原始研究3,000词
- **摘要**：350词，结构化
- **要点框**：必需的总结
- **可视化摘要**：鼓励使用
- **风格**：政策相关、公共卫生重点

### BMJ（英国医学杂志）

- **字数限制**：研究3,000词
- **摘要**：300词，结构化
- **本文新增内容**：必需的方框
- **优势与局限性框**：明确章节
- **风格**：实用、循证

### Annals of Internal Medicine

- **字数限制**：3,000词
- **摘要**：325词，结构化
- **风格**：聚焦内科实践
- **临床试验和Meta分析**：专业特色

## 报告指南合规性

### CONSORT（随机对照试验）

**25项清单**包括：
- 试验设计、随机化、盲法
- 参与者流程（需要流程图）
- 所有结局及效应量和CI
- 危害和不良事件

### STROBE（观察性研究）

**22项清单**用于：
- 队列、病例对照、横断面研究
- 设置、参与者、变量、数据来源
- 偏倚评估、敏感性分析

### PRISMA（系统综述）

**27项清单**包括：
- 检索策略
- 研究选择过程（流程图）
- 偏倚风险评估
- 合成方法

### STARD（诊断研究）

**30项**用于诊断准确性研究

## 医学写作中的常见错误

1. **过度声称因果关系**：对观察性数据使用"导致"
2. **仅报告相对风险**：未报告绝对指标
3. **缺少CI**：仅报告点估计而不报告不确定性
4. **局限性描述模糊**："我们的研究有局限性"而无具体说明
5. **忽视阴性结果**：选择性报告结局
6. **临床意义混淆**：统计学显著≠临床有意义
7. **亚组fishing**：将事后亚组分析作为验证性报告
8. **缺少CONSORT/STROBE项目**：报告不完整

## 投稿前检查清单

### 必需要素
- [ ] 结构化摘要（期刊特定格式）
- [ ] 试验注册号（针对RCT）
- [ ] 伦理委员会批准声明
- [ ] 利益冲突披露
- [ ] 完成CONSORT/STROBE清单

### 统计报告
- [ ] 主要结局报告含CI和P值
- [ ] 包含绝对和相对指标
- [ ] 报告所有预先设定的结局
- [ ] 计算显著临床结局的NNT

### 证据语言
- [ ] 主张与研究设计匹配
- [ ] 使用适当的措辞调整
- [ ] 仅对RCT使用因果语言

### 临床相关性
- [ ] 陈述临床意义
- [ ] 强调以患者为中心的结局
- [ ] 讨论可推广性
