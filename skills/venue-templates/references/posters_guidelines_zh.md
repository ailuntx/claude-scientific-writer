# 研究海报指南

创建有效学术海报的综合指南，包括尺寸、布局、排版和设计最佳实践。

**最后更新**：2024

---

## 标准海报尺寸

### 国际尺寸（ISO 216）

| 尺寸 | 尺寸（毫米） | 尺寸（英寸） | 常见用途 |
|------|-------------|-------------|----------|
| **A0** | 841 × 1189 | 33.1 × 46.8 | 最常见的国际标准 |
| **A1** | 594 × 841 | 23.4 × 33.1 | 较小的会议，便于携带 |
| **A2** | 420 × 594 | 16.5 × 23.4 | 小型海报，小场地 |

### 美国尺寸

| 尺寸 | 尺寸（英寸） | 尺寸（毫米） | 常见用途 |
|------|------------|-------------|----------|
| **36" × 48"** | 36 × 48 | 914 × 1219 | 常见的美国会议尺寸（纵向） |
| **42" × 56"** | 42 × 56 | 1067 × 1422 | 大幅面美国海报 |
| **48" × 36"** | 48 × 36 | 1219 × 914 | 横向方向 |
| **48" × 96"** | 48 × 96 | 1219 × 2438 | 超宽幅面 |

### 其他常见尺寸

| 尺寸 | 尺寸 | 备注 |
|------|--------|-------|
| **90 cm × 120 cm** | 900 × 1200 mm / 35.4 × 47.2 in | 欧洲常见 |
| **40" × 30"** | 1016 × 762 mm | 横向格式 |
| **3 ft × 4 ft** | 914 × 1219 mm / 36 × 48 in | 与36×48相同 |

### 方向

- **纵向**：最常见（高度 > 宽度）
  - 更适合长视觉流程（从上到下）
  - 示例：A0, 36"×48"
- **横向**：不太常见但有时更优
  - 更适合宽内容、时间线
  - 示例：48"×36", 40"×30"

**始终验证**：设计前请查看会议规格要求！

## 排版和字体大小

### 推荐的观看距离字体大小

海报从3-6英尺（1-2米）外观看：

| 元素 | 尺寸范围 | 推荐 |
|---------|-----------|-------------|
| **标题** | 60-85 pt | 72-85 pt |
| **作者姓名** | 48-60 pt | 54 pt |
| **单位 affiliation** | 32-40 pt | 36 pt |
| **章节标题** | 36-48 pt | 42 pt |
| **正文** | 24-32 pt | 28 pt |
| **图例** | 20-24 pt | 22 pt |
| **参考文献** | 18-22 pt | 20 pt |

### 字体系列

**无衬线（海报推荐）**：
- Arial
- Helvetica
- Calibri
- Futura
- Gill Sans
- **原因**：简洁，远距离可读性强

**衬线（谨慎使用）**：
- Times New Roman
- Georgia
- Palatino
- **何时使用**：如果喜欢，正文使用，但标题最好用无衬线

**等宽字体**：
- Courier New
- Consolas
- **何时使用**：仅用于代码片段

### 排版最佳实践

1. **限制字体**：最多使用1-2种字体系列
2. **层次**：建立清晰的尺寸层次
3. **粗细**：使用粗体强调，不要使用斜体
4. **对齐**：正文左对齐，标题居中
5. **间距**：宽松的行距（1.2-1.5）
6. **一致性**：相似元素使用相同字体

## 布局和设计原则

### 基于网格的布局

**列结构**：

| 布局 | 列数 | 适用于 |
|--------|---------|----------|
| **单列** | 1 | 简单，线性流程；时间线海报 |
| **双列** | 2 | 最常见；平衡布局 |
| **三列** | 3 | 内容密集；多部分研究 |
| **四列** | 4 | 非常密集；尽量避免 |

**推荐**：大多数研究海报使用**2或3列**

### 视觉流程

**阅读顺序**（西方惯例）：
1. 从上到下
2. 从左到右
3. Z形或F形模式

**章节排序**（典型）：
```
+----------------------------------+
|            标题                   |
|     作者，单位                   |
+----------------------------------+
| 引言       | 结果                 |
|            |                      |
| 方法       | 讨论                 |
|            |                      |
| [可选]     | 结论                 |
+----------------------------------+
|         参考文献 / 二维码         |
+----------------------------------+
```

### 间距和边距

- **外边距**：四周各1-2英寸（2.5-5厘米）
- **列间距**：0.5-1英寸（1.3-2.5厘米）
- **章节间间距**：0.5-1英寸
- **空白**：海报应有30-40%是空白

**避免**：密集、文本密集的布局，空白极少

## 配色方案

### 盲人友好的调色板

使用对色盲友好的颜色组合：

**Okabe-Ito 调色板**（推荐）：
- 橙色：#E69F00
- 天蓝色：#56B4E9
- 蓝绿色：#009E73
- 黄色：#F0E442
- 蓝色：#0072B2
- 朱红色：#D55E00
- 偏红紫色：#CC79A7
- 黑色：#000000
- 灰色：#999999

**Viridis 调色板**（渐变）：
- 适合热图和渐变
- 色盲友好且感知均匀

### 颜色使用指南

**背景**：
- **白色或浅灰色**：最常见，专业
- **浅色**：淡蓝色，米色（谨慎使用）
- **避免深色背景**：难以阅读，印刷成本高

**文本**：
- **深色浅底**：白色/浅色背景上的黑色或深灰色文本
- **对比度**：至少4.5:1（WCAG AA标准）

**强调色**：
- 最多使用2-3种强调色
- 章节标题，主要发现
- 整个海报保持一致

**图表**：
- 盲人友好的调色板
- 足够的对比度
- 灰度测试

### 颜色对比工具
- WebAIM对比度检查器：https://webaim.org/resources/contrastchecker/
- Coblis（色盲模拟器）：https://www.color-blindness.com/coblis-color-blindness-simulator/

## 内容结构

### 必要章节

#### 1. 标题部分
- **标题**：清晰、具体、有吸引力
- **作者**：姓名（强调作者加下划线）
- **单位**：机构，部门
- **标志**：机构标志（角落）
- **联系方式**：邮箱，论文/网站二维码

#### 2. 引言/背景
- **目的**：背景和动机
- **长度**：100-200字
- **包含**： 
  - 问题陈述
  - 研究空白
  - 目标/假设

#### 3. 方法
- **目的**：你如何进行研究
- **格式**： 
  - 优先使用要点
  - 复杂则使用流程图
  - 关键参数
- **包含**：样本量，程序，分析

#### 4. 结果
- **目的**：你发现了什么
- **格式**：主要是视觉（图表，表格，图形）
- **包含**：
  - 主要发现（2-4个主要结果）
  - 统计显著性
  - 视觉证据

#### 5. 讨论/结论
- **目的**：这意味着什么
- **长度**：100-200字
- **包含**：
  - 解释
  - 意义
  - 局限性
  - 未来工作

#### 6. 参考文献
- **格式**：小字体，缩写引用
- **包含**：仅关键引用（5-10个）
- **风格**：任何一致的风格

### 可选章节
- **摘要**：有时包含，通常省略
- **致谢**：资助，合作者
- **未来工作**：下一步

## 视觉元素

### 图表和图形

**原则**：
1. **简化**：去除杂乱，强调要点
2. **放大**：比论文中的更大
3. **清晰标注**：大的坐标轴标签，图例
4. **独立**：每个图表讲述一个完整的故事
5. **高分辨率**：至少300 dpi

**图表类型**：
- **照片**：高质量，适当裁剪
- **图表**：柱状图，折线图，散点图
- **热图**：使用盲人友好的色图
- **示意图**：图表，流程图
- **表格**：简单表格（复杂表格→图表）

### 表格

**何时使用**：
- 需要精确数值
- 条件间比较
- 汇总统计

**最佳实践**：
- **保持简单**：最多3-5列，5-10行
- **大字体**：与正文相同大小
- **清晰表头**：粗体列/行标题
- **交替行**：浅色阴影以提高可读性
- **最少线条**：仅水平线（不要垂直线）

### 图标和图形

**使用**：
- 图标增加视觉兴趣（方法，概念）
- 简单图形打断文本
- 箭头引导流程

**来源**：
- Noun Project：https://thenounproject.com/
- BioRender：https://biorender.com/（科学插图）
- Font Awesome：https://fontawesome.com/（图标）

**注意**：不要过度使用；保持专业性

## LaTeX 海报包

### beamerposter

**描述**：Beamer的海报扩展  
**最佳用于**：学术会议，经典布局  
**优点**： 
- Beamer用户熟悉
- 简洁，专业的外观
- 多种主题可用

**缺点**：
- 不如tikzposter灵活
- 可能冗长

**模板**：`assets/posters/beamerposter_academic.tex`

**示例用法**：
```latex
\documentclass[final]{beamer}
\usepackage[size=a0,scale=1.24]{beamerposter}
\usetheme{Berlin}
```

---

### tikzposter

**描述**：使用TikZ的现代海报包  
**最佳用于**：彩色，现代设计  
**优点**：
- 高度可定制
- 现代，有吸引力的主题
- 基于块的布局

**缺点**：
- 学习曲线较陡
- 编译可能较慢

**模板**：`assets/posters/tikzposter_research.tex`

**示例用法**：
```latex
\documentclass[25pt, a0paper, portrait]{tikzposter}
\usetheme{Autumn}
\usecolorstyle{Denmark}
```

---

### baposter

**描述**：盒子和海报系统  
**最佳用于**：结构化，多列布局  
**优点**：
- 出色的列控制
- 标题框，结构化布局
- 适合密集内容

**缺点**：
- 语法复杂
- 较少使用

**模板**：`assets/posters/baposter_conference.tex`

**示例用法**：
```latex
\documentclass[a0paper,portrait]{baposter}
```

## 打印和文件准备

### 文件格式

**用于打印**：
- **PDF**：通用标准
- **高分辨率**：至少300 dpi，照片600 dpi
- **色彩空间**：大多数打印机使用RGB（与打印机确认）
- **嵌入字体**：确保所有字体嵌入
- **拼合**：无透明度问题

### 打印质量检查

打印前：
1. **校对**：检查拼写错误，错误
2. **颜色**：在打印预览中检查
3. **分辨率**：放大到100%，检查图表质量
4. **边距**：验证没有内容被切掉
5. **测试打印**：打印小版本（A4）检查布局

### 打印服务商

**选项**：
1. **大学打印店**：通常最便宜，方便
2. **FedEx Office**：广泛可用
3. **在线服务**： 
   - Vistaprint
   - Printful
   - Academic Poster Printing（专业）

**费用**：A0光面海报通常50-150美元

### 纸张类型

| 纸张类型 | 描述 | 最佳用于 |
|-----------|-------------|----------|
| **哑光** | 无反光 finish | 光线充足的场地，最小眩光 |
| **光面** | 有光泽，色彩鲜艳 | 照片，彩色图表 |
| **绸缎/半光面** | 介于哑光和光面之间 | 平衡选项（推荐） |
| **织物** | 抗皱，可卷起 | 旅行，重复使用 |

**推荐**：大多数学术海报使用**绸缎或哑光**

## 二维码

### 包含内容

生成链接到以下内容的二维码：
- **论文PDF**：已发表或预印本
- **补充材料**：数据，代码，视频
- **个人网站**：实验室或研究者页面
- **视频摘要**：1-2分钟视频总结
- **在线版本**：互动海报

### 放置位置

- **常见位置**： 
  - 右下角
  - 参考文献附近
  - 联系方式附近
- **尺寸**：3-4英寸（7-10厘米）见方
- **标签**："扫描获取论文"或"更多信息"

### 二维码生成器
- 二维码生成器：https://www.qr-code-generator.com/
- QRStuff：https://www.qrstuff.com/
- LaTeX包：用于在LaTeX中生成的`qrcode`

## 设计最佳实践

### 应该做

✓ **使用大字体**（正文28pt+）  
✓ **保持文本 minimal**（海报的30-40%）  
✓ **使用视觉元素**（60-70%图表，图形，图像）  
✓ **讲故事**（清晰的叙事流程）  
✓ **盲人友好的颜色**  
✓ **测试可读性**（从6英尺外查看）  
✓ **包含联系方式**（邮箱，二维码）  
✓ **校对**（多次！）

### 不应该做

✗ **不要使用小字体**（正文<24pt）  
✗ **不要过度拥挤**（留出空白）  
✗ **不要使用复杂表格**（简化或可视化）  
✗ **不要使用完整段落**（使用要点）  
✗ **不要使用多种字体**（最多1-2种）  
✗ **不要使用低分辨率图像**（<300 dpi）  
✗ **不要使用红绿对比**（色盲问题）  
✗ **不要把它变成论文**（海报≠论文）

## 海报展示技巧

### 海报会议期间

1. **站在海报旁边**：保持可用，投入
2. **准备好电梯演讲**：准备1-2分钟总结
3. **不同深度**：短版（1分钟），中等（3分钟），深入（10分钟）
4. **吸引访客**：提问，邀请讨论
5. **名片**：准备好
6. **笔记本**：记录反馈，问题
7. **传单**：可选的1页摘要（带二维码）

### 讲解海报

**30秒版本**：
- 问题是什么？
- 你做了什么？
- 你发现了什么？

**2分钟版本**：
- 背景 + 动机
- 方法（简要）
- 主要结果（展示主要图表）
- 结论 + 意义

**5+分钟版本**：
- 完整讲解
- 回答具体问题
- 讨论局限性，未来工作

## 可访问性注意事项

### 视觉可访问性

1. **颜色对比**：高对比度（至少4.5:1）
2. **盲人友好**：使用Okabe-Ito或类似调色板
3. **字体大小**：足够大，可以从远处阅读
4. **字体选择**：清晰的无衬线字体
5. **替代文本**：考虑提供文字描述

### 物理可访问性

1. **安装高度**：足够低，以便轮椅使用者可以阅读底部
2. **二维码**：提供替代方案（短网址，传单）

## 总结

**要点**：

1. **尺寸**：验证会议要求（通常A0或36"×48"）
2. **字体**：大（正文28pt+，标题72pt+）
3. **布局**：2-3列，充足的空白
4. **视觉元素**：60-70%视觉内容
5. **颜色**：盲人友好，高对比度
6. **内容**：讲故事，保持文本 minimal
7. **质量**：300+ dpi，测试打印
8. **可访问性**：从远处可读，层次清晰

**记住**：海报**不是论文**——它是旨在引发对话的视觉摘要！
