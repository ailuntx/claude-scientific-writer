# Cell Press 写作风格指南

Cell、Neuron、Immunity、Molecular Cell、Developmental Cell、Cell Reports 等 Cell Press 期刊的综合写作指南。

**最后更新**：2024

---

## 概述

Cell Press 期刊强调**机制深度**、**严谨的实验**和**生物学见解**。与 Nature/Science 注重广泛可读性不同，Cell 论文的读者是欣赏技术细节和全面数据的生物学家。

### 核心理念

> "Cell 论文讲述一个完整的机制故事，并提供详尽的实验支持。"

**主要目标**：通过广泛的实验验证提供深入的生物学见解，推动对基本机制的理解。

---

## Cell Press 的独特特征

Cell Press 具有其他期刊不具备的几个独特元素：

### 1. 摘要（不是摘要）

Cell 使用"摘要"代替"摘要"——功能相似但强调综合。

### 2. 图形摘要（必需）

出现在目录页上的可视化摘要。**这是所有 Cell Press 期刊的必需项。**

### 3. eTOC 简介

电子目录的 30-50 词"电梯演讲"。

### 4. 亮点

3-4 个要点（每个 ≤85 个字符），概括主要发现。

### 5. 简述

一句话总结论文。

---

## 读者与语气

### 目标读者

- 相关领域的专家生物学家
- 熟悉技术和术语
- 期望全面的数据和机制深度
- 重视严谨性和可重复性

### 语气特征

| 特征 | 描述 |
|---------------|-------------|
| **技术性** | 使用该领域的适当术语 |
| **机制性** | 关注如何和为什么，而不只是什么 |
| **全面性** | 对问题进行彻底探索 |
| **数据丰富** | 广泛的实验支持 |
| **精确** | 准确的术语和量化 |

### 文风

- **可以接受第一人称（"我们"）**："我们证明……"
- **鼓励使用主动语态**："我们发现……"
- **自信但审慎**：强有力的主张需要强有力的证据

## 摘要

### 风格要求

- Cell 最多为 **150 个词**；其他 Cell Press 期刊有所不同
- **流畅的段落**（而非结构化部分）
- **信息密集**：每句话都应传达关键点
- **机制重点**：发现了什么以及其工作原理

### 摘要结构

1. **背景**（1 句）：生物学问题/挑战
2. **方法**（1 句）：你做了什么
3. **主要发现**（2-4 句）：主要结果及其机制
4. **意义**（1 句）：这揭示了关于生物学的什么

### 示例摘要（Cell 风格）

```
细胞衰老是一种阻止增殖并促进组织重塑的应激反应，但控制衰老细胞
命运的具体机制仍不清楚。在此，我们发现转录因子 FOXO4 是衰老细
胞活性的关键调节因子。FOXO4 在衰老细胞中高度表达，并将 p53 隔
离在线粒体之外，防止细胞凋亡。使用一种破坏 FOXO4-p53 相互作用的
细胞穿透肽，我们可以在体外和体内选择性诱导衰老细胞凋亡。将该肽
给予老年小鼠可恢复其体能、毛皮密度和肾脏功能。这些发现揭示了
FOXO4-p53 是衰老的脆弱点，并为在衰老中靶向清除衰老细胞的干预
提供了概念验证。
```

## 图形摘要

### 目的

用于目录的单面板可视化摘要，捕捉整篇论文的信息。

### 要求

- **尺寸**：方形格式，通常 1200 × 1200 像素
- **布局**：简洁、不杂乱
- **内容**：展示工作流程、主要发现和机制
- **文字**：最少的标签，易读的大字体
- **颜色**：鲜艳但专业

### 设计元素

```
典型的图形摘要组件：
1. 起点（细胞、生物体、条件）
2. 干预/处理（箭头、符号）
3. 主要测量或观察
4. 结果/结论（可视化表现）
5. 连接各元素的最小文字标签
```

### 示例描述（用于示意图生成）

```
"图形摘要显示：左面板 - 正常细胞中 FOXO4（蓝色）和 p53（绿色）
分离。中间面板 - 衰老细胞中 FOXO4 结合 p53，防止细胞凋亡。右面板 -
FOXO4 肽破坏相互作用，使 p53 到达线粒体，触发细胞凋亡。底部箭头显示
老年小鼠 → 处理 → 恢复活力的小鼠。"
```

## 亮点

### 格式

3-4 个要点，每个 ≤85 个字符（包括空格）

### 内容指南

- 以动作动词或关键名词开头
- 包括具体发现
- 每个亮点独立成文
- 覆盖论文的不同方面

### 示例亮点

```
• FOXO4 在衰老细胞中选择性表达

• FOXO4 隔离 p53，防止衰老细胞凋亡

• 靶向 FOXO4 的肽可选择性诱导衰老细胞死亡

• 衰老细胞清除肽治疗可恢复老年小鼠的功能
```

## eTOC 简介

### 格式

电子目录 30-50 个词

### 写作指南

- 由作者撰写（编辑可能会修改）
- 以作者姓名或主要发现开头
- 写一个完整、吸引人的句子
- 突出最令人兴奋的方面

### 示例 eTOC 简介

```
Baar 等人发现了 FOXO4 作为衰老细胞的脆弱点，并开发了一种可诱导
衰老细胞靶向凋亡的肽。用这种衰老细胞清除肽治疗老年小鼠可恢复
其体能和器官功能。
```

## 引言

### 长度与结构

- **4-6 段**（800-1200 个词）
- 比 Nature/Science 更全面
- 可以包含更多技术细节和文献

### 逐段指南

**第 1 段：生物学背景**
- 建立生物学过程或系统
- 为什么理解这一点很重要？
- 介绍主要参与者和机制

**第 2-3 段：领域现状**
- 详细回顾相关先前工作
- 建立已知的机制
- 比 Nature/Science 更全面

**第 4 段：差距**
- 什么仍然是未知的或有争议的？
- 为什么这是一个关键问题？
- 什么阻碍了进展？

**第 5 段：你的方法**
- 你如何解决这个问题？
- 你使用了什么技术/系统？
- 为什么你的方法是合适的？

**最后一段：主要发现预览**
- 简要陈述你发现了什么
- 这如何推动该领域的发展？
- 建立结果的结构

### 示例引言段落

```
细胞衰老的特征是稳定的细胞周期停滞、深刻的染色质改变以及一种被称为
衰老相关分泌表型（SASP）的复杂分泌表型（Coppé 等，2008；Rodier 和 
Campisi，2011）。衰老细胞随年龄增长而积累，并出现在病变部位，在那里
它们可以通过 SASP 介导的炎症和组织结构破坏来驱动组织功能障碍
（van Deursen，2014）。靶向消除衰老细胞——衰老细胞清除——已经成为一种
有前景的治疗策略，遗传学和药理学方法已在啮齿类动物衰老和年龄相关
疾病模型中证明了益处（Baker 等，2011，2016；Chang 等，2016）。
```

## 结果

### 组织

Cell 论文通常有 **5-8 个结果部分**，每个都有描述性小标题：

```
结果
├── 第 1 部分：现象的发现
├── 第 2 部分：机制的特征
├── 第 3 部分：分子参与者的鉴定
├── 第 4 部分：功能验证
├── 第 5 部分：体内确认
├── 第 6 部分：治疗概念验证
└── 第 7 部分：更广泛的意义
```

### 小标题风格

Cell 使用**陈述性小标题**陈述发现：

❌ "FOXO4 表达的分析"（描述性 - 避免）
✅ "FOXO4 在衰老细胞中选择性上调"（陈述性）

### 结果写作风格

- **详细的背景**：Cell 期望在结果中比 Nature 提供更多的方法学背景
- **逐图叙述**：每个主要图通常对应一个结果部分
- **统计严谨性**：所有量化都带有统计
- **生物学解释**：比纯粹的结果部分有更多解释

### 示例结果段落

```
为了鉴定调节衰老细胞活性的转录因子，我们对增殖和衰老的人成纤维细胞
（IMR90 细胞通过复制性耗竭、电离辐射或致癌基因诱导的衰老）进行了
RNA 测序。差异表达分析显示，在所有衰老模式下，有 47 个转录因子显著
上调（FDR < 0.05，倍数变化 > 2；图 1A 和表 S1）。其中，FOXO4 显示
最高和最一致的上调（12.3 ± 2.1 倍；图 1B），这一发现我们通过定量 RT-PCR
（图 1C）和免疫印迹分析（图 1D）得到确认。免疫荧光显微镜显示 FOXO4
在衰老细胞中积累于细胞核，但在增殖细胞中则不然（图 1E、F）。
```

## 讨论

### 结构

Cell 的讨论**彻底且具有机制性**：

**第 1 段：总结**
- 重述主要发现
- 综合主要信息

**第 2-4 段：机制解释**
- 深入探讨你的发现如何与已知生物学相契合
- 提出模型
- 详细讨论分子机制

**第 5 段：与文献比较**
- 你的发现与先前工作如何关联？
- 解决明显的矛盾

**第 6 段：意义与应用**
- 治疗意义
- 更广泛的重要性

**第 7 段：局限性**
- 诚实的评估
- 剩余的开放问题

**最后一段：结论**
- 整体的核心信息
- 未来的方向

```
资源可用性
  主要联系人
  材料可用性
  数据和代码可用性

实验模型和受试者详情
```

## 图表

### 图表理念

Cell 论文**图表较多**，有大量的多面板图表：

- 通常 **6-8 个主要图表**
- **多面板格式**：每个图表 6-12 个面板很常见
- **数据密集**：全面的实验支持
- **扩展数据**：用于额外验证的补充图表

### 面板标签

面板用小写字母标记：**(A)**、**(B)**、**(C)** 等

### 图例格式

```
图 3. FOXO4 将 p53 隔离在衰老细胞的细胞核中

(A) 增殖（左）和衰老（右）IMR90 细胞中 p53（绿色）和 FOXO4（红色）
的免疫荧光显微镜检查。dapi（蓝色）标记细胞核。比例尺，10 μm。

(B) 增殖与衰老细胞中细胞核 p53 强度的量化。数据表示平均值 ± 标准误；
n = 3 个生物学重复，每个条件 >100 个细胞。***p < 0.001，双尾学生 t 检验。

(C 和 D) 增殖（C）和衰老（D）细胞裂解液中 FOXO4 和 p53 的共免疫沉淀。
IgG，免疫球蛋白 G 对照。

(E) FOXO4-p53 相互作用的近邻连接分析。红色点表示相互作用事件。
比例尺，10 μm。

(F) FOXO4 介导的衰老细胞中 p53 隔离模型。

另见图 S3 和表 S2。
```

## 参考文献

### 引用格式

- **作者-年份格式**：(Smith 等，2023) 或 Smith 等（2023）
- **多个引用**：(Smith 等，2020；Jones 等，2021)
- **两位作者**：(Smith 和 Jones，2023)
- **三位或更多**：(Smith 等，2023)

### 参考文献格式

```
Baker, D.J., Wijshake, T., Tchkonia, T., LeBrasseur, N.K., Childs, B.G., 
van de Sluis, B., Kirkland, J.L., and van Deursen, J.M. (2011). Clearance 
of p16Ink4a-positive senescent cells delays ageing-associated disorders. 
Nature 479, 232–236.
```

## 常见错误

1. **机制不足**：描述发生了什么而没有说明如何发生
2. **对照不足的实验**：缺少关键对照
3. **表型验证薄弱**：使用单一方法而非多种方法
4. **缺少体内工作**：Cell 论文通常期望有动物研究
5. **图表面板不完整**：没有显示所有相关条件
6. **忘记图形摘要**：必需元素
7. **超出亮点字符限制**：每个要点 ≤85 个字符

## 提交前检查清单

### 必需元素
- [ ] 图形摘要（方形格式）

- [ ] eTOC 简介（30-50 个词）
- [ ] 摘要（≤150 个词）
- [ ] 关键资源表

### 内容
- [ ] 贯穿全文的机制深度
- [ ] 多种互补方法
- [ ] 体内验证（如果适用）
- [ ] 陈述性小标题
- [ ] 全面的图表面板

### 风格
- [ ] 术语的技术精确性
- [ ] 作者-年份引用
- [ ] 图例完整且独立
- [ ] STAR 方法格式正确
