# ScholarEval 评估框架

## 概述

本文档为ScholarEval框架的每个维度提供了详细的评估标准、评分标准和质量指标。在对学术工作进行系统性评估时，请使用这些标准。

---

## 维度一：问题构建与研究问题

### 质量指标

**优秀（5）：**
- 研究问题具体、可测量且表达清晰
- 问题针对文献中的重要缺口，具有高影响潜力
- 范围在约束条件下适当且可行
- 新颖贡献与现有工作明确区分
- 理论或实践意义具有令人信服的论证

**良好（4）：**
- 研究问题清晰，存在轻微歧义
- 问题具有相关性，中等影响潜力
- 范围总体适当，存在轻微可行性问题
- 贡献可识别但非突破性
- 意义论证充分

**合格（3）：**
- 研究问题存在但缺乏具体性
- 问题相关性不明确或为增量性
- 范围可能过宽或过窄
- 贡献不明确或与现有工作高度重叠
- 意义论证薄弱

**需改进（2）：**
- 研究问题模糊或定义不佳
- 问题缺乏明确的相关性或意义
- 范围不适当或不可行
- 贡献未阐明
- 无明确的显著性论证

**较差（1）：**
- 无清晰的研究问题
- 问题微不足道或无关紧要
- 范围存在根本性缺陷
- 无可识别的贡献
- 无显著性论证

### 评估清单

- [ ] 研究问题是否明确阐述？
- [ ] 该问题能否通过所提出的方法回答？
- [ ] 问题对该领域是否重要？
- [ ] 在资源约束下范围是否可行？
- [ ] 新颖性/贡献是否表达清晰？
- [ ] 关键假设是否明确说明？
- [ ] 成功标准或预期结果是否定义？

## 维度二：文献综述

### 质量指标

**优秀（5）：**
- 全面覆盖关键领域的相关文献
- 批判性综合识别模式、矛盾和缺口
- 文献具有时效性（快速发展的领域主要来自近3-5年）
- 来源权威且经过同行评审
- 清晰定位当前工作在学术对话中的位置
- 识别本研究解决的真正研究缺口

**良好（4）：**
- 覆盖良好，关键领域有轻微缺口
- 以综合为主，辅以描述
- 文献基本具有时效性，部分较旧的基础性文献
- 来源总体权威
- 工作定位存在但可以更强
- 识别了研究缺口但可能不是关键缺口

**合格（3）：**
- 部分覆盖，存在明显缺口
- 描述性总结多于综合性分析
- 文献来源混合了时效性和过时来源
- 权威来源与非严谨来源混合
- 在现有文献中定位较弱
- 研究缺口模糊或存疑

**需改进（2）：**
- 覆盖极少，存在重大缺口
- 纯描述性，无综合分析
- 文献基本过时
- 来源缺乏权威性或严谨性
- 几乎无当前工作的定位
- 未识别明确的研究缺口

**较差（1）：**
- 文献综述不足或缺失
- 无综合分析
- 来源过时或不适当
- 未参与学术对话
- 未识别缺口

### 评估清单

- [ ] 综述是否覆盖所有主要相关领域？
- [ ] 文献是否经过综合而非仅总结？
- [ ] 来源是否时效且权威？
- [ ] 是否呈现了对比观点？
- [ ] 研究缺口是否清晰识别？
- [ ] 当前工作是否在现有文献中正确定位？
- [ ] 引文平衡是否适当（不过度依赖少数作者）？
- [ ] 是否包含经典/基础性著作？

### 常见问题

- **覆盖不足**：遗漏关键论文或研究流派
- **描述性列举**：逐篇总结论文而无综合分析
- **来源过时**：依赖超过5-10年的文献
- ** cherry-picking（选择性引用）**：仅引用支持假设的著作
- **组织不佳**：缺乏主题或概念结构
- **缺口识别薄弱**：缺口微不足道或并非真正缺口

## 维度三：方法论与研究设计

### 质量指标

**优秀（5）：**
- 研究设计与研究问题完美契合
- 方法严谨、有效且可靠
- 程序详细到足以复制
- 对照、随机化或三角验证适当
- 潜在偏见被承认并减轻
- 伦理考量得到全面处理
- 局限性明确讨论

**良好（4）：**
- 设计适当，存在轻微契合问题
- 方法合理，存在小的有效性问题
- 程序基本可复制
- 存在一些对照或验证
- 主要偏见已处理
- 提及伦理考量
- 讨论了一些局限性

**合格（3）：**
- 设计部分适合问题
- 方法存在明显有效性问题
- 程序缺乏细节，无法完全复制
- 对照或验证有限
- 偏见减轻最少
- 伦理处理表面化
- 很少讨论局限性

**需改进（2）：**
-设计与研究问题契合度差
- 方法存在严重的有效性问题
- 程序过于模糊，无法复制
- 无对照或验证
- 未处理偏见
- 未处理伦理问题
- 无局限性讨论

**较差（1）：**
- 方法论不当或缺失
- 方法根本有缺陷
- 不可复制
- 无有效性考量
- 无伦理考量
- 无局限性承认

### 评估清单

- [ ] 方法论是否适合研究问题？
- [ ] 程序是否描述得足够详细？
- [ ] 研究能否从描述中复制？
- [ ] 是否处理了有效性和可靠性？
- [ ] 是否识别并减轻了潜在偏见？
- [ ] 是否讨论了伦理考量？
- [ ] 是否承认了局限性？
- [ ] 样本量是否有论证（定量研究）？
- [ ] 定性方法是否严谨（如适用）？

### 设计特定考量

**定量研究：**
- 样本量与功效分析
- 对照组和随机化
- 测量有效性和可靠性
- 统计假设检验

**定性研究：**
- 抽样策略和饱和度
- 数据收集程序
- 编码和分析框架
- 可信度标准（可信性、迁移性等）

**混合方法：**
- 整合理由
- 排序论证
- 数据趋同策略

## 维度四：数据收集与来源

### 质量指标

**优秀（5）：**
- 数据来源高度可信且适当
- 样本量充足且论证充分
- 数据收集程序严谨且系统
- 数据质量控制到位
- 抽样策略确保代表性
- 缺失数据极少且处理适当

**良好（4）：**
- 数据来源可信，存在轻微问题
- 样本量足够
- 收集程序系统
- 存在一些质量控制
- 抽样合理
- 缺失数据已处理

**合格（3）：**
- 数据来源可接受但非最佳
- 样本量边缘
- 收集程序缺乏一些严谨性
- 质量控制有限
- 抽样可能存在偏差问题
- 缺失数据处理基础

**需改进（2）：**
- 数据来源存在可信度问题
- 样本量不足
- 收集程序临时拼凑
- 无质量控制
- 抽样明显有偏差
- 未处理缺失数据

**较差（1）：**
- 数据来源不当或不可靠
- 样本量不足
- 收集无系统性
- 无质量考量
- 抽样根本有缺陷
- 缺失数据过多

### 评估清单

- [ ] 数据来源是否可信且适当？
- [ ] 样本量是否足以得出结论？
- [ ] 抽样策略是否描述清晰？
- [ ] 样本是否代表目标人群？
- [ ] 数据收集程序是否系统？
- [ ] 是否描述了数据质量控制？
- [ ] 是否处理了缺失数据？
- [ ] 是否讨论了任何潜在数据偏差？

---

## 维度五：分析与解读

### 质量指标

**优秀（5）：**
- 分析方法完全适合数据和问题的关联性
- 分析严谨，使用适当技术
- 结果解读逻辑性强且充分支持
- 考虑了替代解释
- 主张与证据成比例
- 假设得到验证
- 分析透明且可复制

**良好（4）：**
- 方法适当，存在轻微问题
- 分析合理
- 解读基本逻辑性
- 考虑了一些替代方案
- 主张基本与证据匹配
- 检查了关键假设
- 分析基本透明

**合格（3）：**
- 方法可接受但非最佳
- 分析存在一些技术问题
- 解读存在逻辑缺口
- 未充分探索替代方案
- 一些主张超出证据
- 假设未完全验证
- 分析透明度有限

**需改进（2）：**
- 方法对数据/问题存疑
- 分析存在重大技术缺陷
- 解读缺乏充分支持
- 无替代解释
- 主张明显超出证据
- 未检查假设
- 分析不透明

**较差（1）：**
- 方法不当
- 分析根本有缺陷
- 解读不合逻辑
- 未考虑替代方案
- 主张无证据支持
- 无假设验证
- 分析不透明

### 评估清单

- [ ] 分析方法是否适当？
- [ ] 统计检验/定性方法是否正确应用？
- [ ] 是否检验了假设？
- [ ] 解读是否逻辑性强且有充分支持？
- [ ] 是否考虑了替代解释？
- [ ] 主张是否与证据强度一致？
- [ ] 能否从描述中复制分析？
- [ ] 是否承认了不确定性？

### 定量分析

- 适当的统计检验
- 假设检验（正态性、同质性等）
- 报告效应量
- 提供置信区间
- 多重检验校正（如适用）
- 模型诊断

### 定性分析

- 编码框架清晰
- 评分者间信度（如适用）
- 讨论饱和度
- 检查负面案例
- 成员检查或验证
- 清晰的审计追踪

## 维度六：结果与发现

### 质量指标

**优秀（5）：**
- 结果呈现清晰且全面
- 可视化有效且适当
- 统计或定性严谨性显而易见
- 有效突出关键发现
- 结果直接回答研究问题
- 模式和关系清晰展示
- 报告负面和零结果

**良好（4）：**
- 结果清晰，存在轻微呈现问题
- 可视化基本有效
- 存在严谨性
- 主要发现可识别
- 结果基本回答问题
- 展示了一些模式
- 包含一些负面结果

**合格（3）：**
- 结果呈现合格但可以更清晰
- 可视化基础或存在问题
- 严谨性存疑
- 存在发现但未突出
- 与问题部分对齐
- 模式不清晰
- 可能遗漏负面结果

**需改进（2）：**
- 结果呈现不清晰或令人困惑
- 可视化差或误导
- 缺乏严谨性
- 发现难以识别
- 与问题对齐弱
- 无清晰模式
- 仅显示阳性结果

**较差（1）：**
- 结果呈现差或缺失
- 可视化不当或缺失
- 无严谨性证据
- 发现不清晰
- 结果未回答问题
- 无可识别的模式
- 结果具有选择性

### 评估清单

- [ ] 结果是否清晰呈现？
- [ ] 结果是否直接回答研究问题？
- [ ] 可视化是否适当且有效？
- [ ] 是否突出关键发现？
- [ ] 是否报告了负面/零结果？
- [ ] 是否报告了适当的精确度（p值、置信区间、效应量）？
- [ ] 定性发现是否有数据摘录支持？
- [ ] 是否有选择性报告的证据？

### 呈现质量

**表格：**
- 清晰的标签和标题
- 适当的精确度
- 逻辑组织
- 不过于复杂

**图形：**
- 清晰的轴和图例
- 适当的图表类型
- 专业的外观
- 无障碍（适合色盲）

**文本：**
- 突出关键发现
- 避免与表格/图形重复
- 使用适当的统计语言

## 维度七：学术写作与呈现

### 质量指标

**优秀（5）：**
- 写作清晰、简洁且精确
- 组织逻辑性好，流畅性极佳
- 学术语调适当且一致
- 语法和机械错误为零
- 术语使用正确
- 对目标受众可访问
- 摘要/总结全面且准确

**良好（4）：**
- 写作清晰，存在轻微不自然
- 组织逻辑性好，流畅性好
- 语调基本适当
- 少数语法/机械错误
- 术语基本正确
- 基本可访问
- 摘要适当

**合格（3）：**
- 写作可理解，但有清晰度问题
- 组织存在一些逻辑缺口
- 语调不一致
- 明显的语法/机械错误
- 一些术语误用
- 对目标受众存在可访问性问题
- 摘要不完整或模糊

**需改进（2）：**
- 写作经常不清晰或冗长
- 组织和流畅性差
- 语调不当
- 频繁的语法/机械错误
- 术语问题
- 对目标受众不可访问
- 摘要差或缺失

**较差（1）：**
- 写作不清晰且难以理解
- 无清晰组织
- 语调不当
- 普遍的语法/机械错误
- 术语不正确
- 不可访问
- 无适当的摘要

### 评估清单

- [ ] 写作是否清晰简洁？
- [ ] 组织是否逻辑？
- [ ] 语调是否适合学术写作？
- [ ] 语法和机械是否正确？
- [ ] 术语使用是否适当？
- [ ] 是否在必要时解释了术语？
- [ ] 摘要是否准确总结工作？
- [ ] 各部分之间的过渡是否流畅？
- [ ] 目标受众是否明确？

### 常见写作问题

- **冗长**：不必要地复杂或冗长的散文
- **过度使用被动语态**：降低清晰度和直接性
- **段落结构**：缺乏主题句或连贯性
- **冗余**：不必要地重复信息
- **逻辑流**：想法之间过渡差
- **精确性**：模糊或歧义的语言
- **可访问性**：过于技术性或技术性不足

## 维度八：引用与参考文献

### 质量指标

**优秀（5）：**
- 所有主张都有适当引用
- 来源权威且时效
- 引用准确完整
- 代表了多样的观点
- 引用格式一致正确
- 自引与他引平衡适当
- 适当使用原始来源

**良好（4）：**
- 大部分主张有引用
- 来源基本权威
- 少数引用错误
- 来源多样性合理
- 格式基本一致
- 引用平衡良好
- 原始和二手来源混合

**合格（3）：**
- 一些主张缺少引用
- 来源质量混合
- 几个引用错误
- 来源多样性有限
- 格式不一致
- 引用平衡问题
- 过度依赖二手来源

**需改进（2）：**
- 许多主张未引用
- 来源存疑
- 大量引用错误
- 来源基础狭窄
- 格式不一致
- 过度自引或引用狭窄
- 来源不当（例如，仅二手来源）

**较差（1）：**
- 引用不足
- 来源不可靠
- 普遍的引用错误
- 来源多样性极少
- 无一致格式
- 严重的引用不平衡
- 来源类型不当

### 评估清单

- [ ] 所有事实主张是否都有引用？
- [ ] 是否在适当时候引用原始来源？
- [ ] 来源是否权威且经过同行评审？
- [ ] 所引用观点是否平衡？
- [ ] 引用是否准确（作者、日期、页码）？
- [ ] 引用格式是否一致？
- [ ] 自引是否适当（通常<20%）？
- [ ] 来源是否时效（对于时间敏感主题）？
- [ ] 是否在相关处包含经典/经典著作？

### 引用质量评估

**来源类型（大多数学术工作按偏好排序）：**
1. 同行评审的期刊文章
2. 知名出版社的学术书籍
3. 会议论文集（视领域而定）
4. 知名机构的技术报告
5. 学位论文/毕业论文
6. 预印本（谨慎使用，视领域而定）
7. 灰色文献（有限使用）
8. 网站（很少适当，除非用于事实数据）

**警示信号：**
- 维基百科作为主要来源
- 过度自引（>30%）
- 仅引用支持假设的论文
- 存在当前来源时使用过时来源
- 遗漏该领域的关键论文
- 仅引用摘要而非全文
- 引用格式不一致或不正确

## 跨维度考量

### 可复制性

跨维度评估：
- 方法是否详细到足以复制？
- 数据和代码是否可用（或解释了可用性）？
- 分析步骤是否透明？
- 是否指定了材料/仪器？

### 伦理

考虑：
- 机构审查委员会批准（针对人类受试者）
- 知情同意
- 隐私和保密
- 利益冲突
- 研究诚信
- 数据共享伦理

### 偏见与局限性

评估是否：
- 承认了潜在偏见
- 诚实地讨论了局限性
- 指定了边界条件
- 适当声称普遍性

### 影响与意义

## 评分指南

### 维度权重（建议，按上下文调整）

- 问题构建：15%
- 文献综述：15%
- 方法论：20%
- 数据收集：10%
- 分析：15%
- 结果：10%
- 写作：10%
- 引用：5%

### 整体评估阈值

- **卓越（4.5-5.0）**：准备好投递顶级期刊
- **强（4.0-4.4）**：准备好发表，需轻微修改
- **良好（3.5-3.9）**：需要重大修改，有前景的工作
- **可接受（3.0-3.4）**：需要显著修改
- **弱（2.0-2.9）**：存在根本问题，需要重大重做
- **较差（<2.0）**：不适合发表，除非完全重写

### 上下文调整

根据以下调整标准：
- **阶段**：提案 < 草稿 < 最终投稿
- **场所**：学生论文 < 会议 < 期刊 < 顶级期刊
- **类型**：理论 < 实证 < 元分析
- **领域**：标准因学科而异
- **目的**：教育 < 专业 < 发表
