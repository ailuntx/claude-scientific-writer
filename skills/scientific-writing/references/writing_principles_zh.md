# 科学写作原则

## 概述

有效的科学写作需要掌握确保清晰性、准确性和影响力的基本原则。与创意或叙事性写作不同，科学写作优先考虑准确性、简洁性和客观性。本指南涵盖了区分优秀科学写作与拙劣写作的核心原则，并提供了改进的实际策略。

## 科学写作的三大支柱

### 1. 清晰性

**定义：** 写作内容能够被目标受众立即理解，不存在歧义或困惑。

**为什么重要：** 科学本身已经足够复杂，不清晰的写作不应再增加混乱。读者应该专注于理解科学，而不是破译文章。

#### 提升清晰性的策略

**使用精确、无歧义的语言：**
```
较差："这种药物似乎对相当多的患者有帮助。"
更好："这种药物使68%（32/47）患者的症状减轻。"
```

**首次使用时定义技术术语：**
```
"我们测量了脑源性神经营养因子（BDNF），这是一种参与神经元存活和可塑性的蛋白质。"
```

**保持段落内和段落间的逻辑流畅：**
- 每个段落应有一个主要思想
- 主题句引出段落的重点
- 支持句展开该重点
- 过渡句连接各段落

**在能提高清晰度时使用主动语态：**
```
被动（较不清晰）："样本由研究人员进行分析。"
主动（更清晰）："研究人员分析了样本。"
```

然而，在方法学部分，当动作比执行者更重要时，被动语态是可接受的且通常是首选的：
```
"在基线和6周后采集了血样。"
```

**拆分冗长复杂的句子：**
```
较差："我们的研究涉及200名来自三家医院的参与者，并进行了12个月的随访，每4周使用经验证的问卷进行一次评估，结果显示干预组显著改善。"

更好："我们的研究涉及200名来自三家医院的参与者。参与者接受了12个月的随访，每4周使用经验证的问卷进行一次评估。干预组显示出显著改善。"
```

**使用具体的动词：**
```
较弱："这项研究关注了青少年抑郁。"
更强："这项研究探讨了导致青少年抑郁的因素。"
```

#### 常见的清晰性问题

**歧义代词：**
```
较差："A组接受药物，B组接受安慰剂。他们显示出改善。"
（"他们"指的是谁？）

更好："A组接受药物，B组接受安慰剂。药物组显示出改善。"
```

**修饰语位置不当：**
```
较差："我们使用自动血压计测量了患者的血压。"
（是患者使用血压计，还是我们使用？）

更好："我们使用自动血压计测量了患者的血压。"
```

**不明确的指代：**
```
较差："表达增加伴随增殖减少，这是出乎意料的。"
（出乎意料的是减少、陪伴，还是两者？）

更好："表达增加伴随增殖减少。这种反比关系是出乎意料的。"
```

### 2. 简洁性

**定义：** 以最少的必要词汇表达思想，同时不牺牲清晰性或完整性。

**为什么重要：** 简洁的写作尊重读者的时间。每个不必要的词汇都是错失的清晰性和影响力机会。正如该原则所述："我们重视简洁的写作，因为我们重视时间。"

#### 提升简洁性的策略

**消除冗余的词汇和短语：**

| 冗余 | 简洁 |
|-----|-----|
| "due to the fact that" | "because" |
| "in order to" | "to" |
| "it is important to note that" | [删除] |
| "a total of 50 participants" | "50 participants" |
| "completely eliminate" | "eliminate" |
| "has been shown to be" | "is" |
| "in the event that" | "if" |
| "at the present time" | "now" 或 "currently" |
| "conduct an investigation into" | "investigate" |
| "give consideration to" | "consider" |

**避免开场铺垫短语：**
```
冗余："值得注意的是，我们研究的结果表明......"
简洁："我们的结果表明......" 或 "结果显示......"
```

**使用强动词代替名词+动词组合：**

| 冗余 | 简洁 |
|-----|-----|
| "make a decision" | "decide" |
| "perform an analysis" | "analyze" |
| "conduct a study" | "study" 或 "studied" |
| "make an assessment" | "assess" |
| "provide information about" | "inform" |

**消除不必要的强调词：**
```
冗余："结果非常显著。"
简洁："结果是显著的。"（p值传达程度）
```

**避免不必要地重复信息：**
```
冗余："结果显示，参与干预组的参与者接受了治疗干预，有更好的结果。"
简洁："干预组有更好的结果。"
```

**使用较短的结构：**
```
冗余："尽管样本量很小......"
简洁："虽然样本量很小......"
```

#### 可接受的长度与不必要的长度

**并非所有长句都是不好的：**
```
这个详细的句子是可以的："我们使用液相色谱-串联质谱（LC-MS/MS）分析血样，使用Waters Acquity UPLC系统连接Xevo TQ-S质谱仪（Waters Corporation, Milford, MA）。"

为什么？因为每个元素都是必要的信息。
```

**关键问题：** 删除任何词汇是否会损失含义或精确性？如果是，就删除它。

### 3. 准确性

**定义：** 对数据、方法和解释的精确、正确的表述。

**为什么重要：** 科学可信度取决于准确性。不准确的报告会损害整个科学事业。

#### 提升准确性的策略

**报告具有适当精度的精确值：**
```
较差："平均值大约是25。"
更好："平均值为24.7 ± 3.2（SD）。"
```

**将精度与测量能力相匹配：**
```
不恰当："平均年龄为45.237岁"（暗示虚假的精度）
恰当："平均年龄为45.2岁"
```

**全文使用一致的术语：**
```
不一致：引言称为"认知功能"，方法学称为"心理表现"，结果称为"智力能力"。

一致：全程使用"认知功能"，或明确定义："认知功能（也称为心理表现）"
```

**区分观察与解释：**
```
观察："平均血压从145降至132 mmHg（p=0.003）。"
解释："这表明干预有效地降低了血压。"
```

**具体说明不确定性：**
```
模糊："这些测量可能存在一些误差。"
具体："根据仪器规格，测量的标准误差为±2.5 mmHg。"
```

**使用正确的统计语言：**
```
不正确："相关性高度显著（p=0.03）。"
正确："相关性具有统计显著性（p=0.03）。"
（p=0.03不是"高度"显著；那是p<0.001的专属）
```

**验证所有数字：**
- 检查正文中的数字与表格/图是否一致
- 验证n值求和正确
- 确认百分比计算正确
- 仔细检查所有统计数据

#### 常见的准确性问题

**过度概括：**
```
较差："运动预防抑郁。"
更好："在我们样本中，随机分配到运动干预的参与者比对照组表现出更少的抑郁症状（BDI-II平均差异3.2分，95% CI：1.5-4.9，p<0.001）。"
```

**不当的因果论断：**
```
较差（来自观察性研究）："维生素D补充剂降低癌症风险。"
更好："在这队列中，维生素D水平与癌症发病率呈负相关（HR=0.82，95% CI：0.71-0.95）。"
```

**不精确的数值描述：**
```
模糊："许多参与者退出了。"
精确："15/50（30%）参与者在研究完成前退出。"
```

## 其他关键原则

### 4. 客观性

**定义：** 不带偏见、夸大或未经支持的看法，公正地呈现信息。

**策略：**

**不带偏见地呈现结果：**
```
偏见："正如预期，我们优越的方法表现更好。"
客观："方法A的准确率高于方法B（87%对比76%，p=0.02）。"
```

**承认相互矛盾的证据：**
```
"我们的发现与Smith等人（2022年）的研究结果形成对比，他们报告没有显著效果。这种差异可能源于干预强度或人群特征的差异。"
```

**避免情感或评价性语言：**
```
主观："结果令人失望和担忧。"
客观："干预没有显著减轻症状（p=0.42）。"
```

**区分事实与推测：**
```
"观察到的细胞活力下降伴随caspase-3活性增加，表明凋亡可能是细胞死亡的主要机制。"
（使用"表明"和"可能是"来表示解释）
```

### 5. 一致性

**保持整个手稿的一致性：**

**术语：**
- 对同一概念使用相同的术语（不要为了多样化而使用同义词）
- 首次使用时定义缩写并始终如一地使用
- 对基因、蛋白质、化学物质使用标准命名法

**符号：**
- 统计符号（p值格式、CI呈现）
- 测量单位
- 数字格式（小数位数）

**时态：**
- 过去时态用于你自己的具体研究动作
- 现在时态用于既定事实
- 详见IMRAD结构参考中的时态指南

**风格：**
- 始终如一地遵循期刊指南
- 引用格式
- 标题大写
- 数字与文字的使用

### 6. 逻辑组织

**在手稿中创建清晰的"主线"：**

**段落结构：**
1. 主题句（主要思想）
2. 支持句（证据、解释）
3. 结论/过渡句（连接到下一个思想）

**章节流程：**
- 每个章节在前一个章节的基础上逻辑展开
- 引言中提出的问题在结果中得到回答
- 结果中呈现的发现在讨论中得到解释

**路标：**
```
"首先，我们检验......"
"接下来，我们调查......"
"最后，我们评估......"
```

**平行结构：**
```
非平行："目标是（1）测量血压，（2）评估认知功能，以及（3）我们想要评估情绪。"

平行："目标是（1）测量血压，（2）评估认知功能，以及（3）评估情绪。"
```

## 科学写作中的动词时态

### 一般准则

**现在时态用于：**
- 既定事实和普遍真理
  - "DNA由核苷酸组成。"
- 你正在得出的结论
  - "这些发现表明......"
- 提及图表和表格
  - "图1显示分布情况......"

**过去时态用于：**
- 已完成研究的具体发现（你自己的和他人的）
  - "Smith等人（2022年）发现......"
  - "我们观察到显著下降......"
- 你执行的方法
  - "参与者在基线时完成了问卷。"

**现在完成时用于：**
- 与当前相关的新进展
  - "最近的研究已证明......"
- 研究领域背景
  - "已提出几种方法......"

### 章节特定的时态

| 章节 | 主要时态 | 示例 |
|-----|---------|------|
| **摘要-背景** | 现在时或现在完成时 | "抑郁症影响数百万人" / "研究表明......" |
| **摘要-方法** | 过去时 | "我们招募了100名参与者" |
| **摘要-结果** | 过去时 | "干预减轻了症状" |
| **摘要-结论** | 现在时 | "这些发现表明......" |
| **引言-背景** | 现在时（事实），现在完成时（研究） | "运动有益处" / "研究表明......" |
| **引言-差距** | 现在时或现在完成时 | "然而，知之甚少......" |
| **引言-本研究** | 过去时或现在时 | "我们调查了......" / "本研究调查......" |
| **方法** | 过去时 | "我们收集了样本......" |
| **结果** | 过去时 | "平均年龄为45岁" |
| **讨论-你的发现** | 过去时 | "我们发现......" |
| **讨论-解释** | 现在时 | "这表明......" |
| **讨论-先前工作** | 过去时或现在时 | "Smith发现......" / "先前工作证明......" |

## 常见写作陷阱

### 1. 术语过度

**问题：** 过度使用技术术语而未定义

**示例：**
```
较差："我们使用qRT-PCR通过基于SYBR-Green的荧光检测来量化mRNA表达，该检测在从总RNA使用 oligo-dT引物合成cDNA后进行。"

更好："我们使用定量逆转录PCR（qRT-PCR）量化mRNA表达。总RNA使用oligo-dT引物逆转录为互补DNA（cDNA），然后用SYBR Green荧光检测进行扩增。"
```

### 2. 名词化

**问题：** 将动词变成名词，使写作冗重且间接

**示例：**

| 名词化 | 直接 |
|--------|------|
| "give consideration to" | "consider" |
| "make an assumption" | "assume" |
| "perform an investigation" | "investigate" |
| "conduct an examination" | "examine" |
| "achieve a reduction" | "reduce" |

### 3. 过度或不足的谨慎表达

**过度谨慎**（听起来不确定）：
```
"在某些条件下，这种干预可能也许有可能对症状产生一些影响。"
```

**谨慎不足**（夸大结论）：
```
"这种干预治愈了抑郁症。"
```

**适当的谨慎表达：**
```
"这种干预显著减轻了本样本中的抑郁症状，表明它可能对治疗轻中度抑郁有效。"
```

**适当使用谨慎词汇：**
- 建议、表明、暗示（不是证明、演示，用于相关性数据）
- 可能、也许、可以（可能性）
- 似乎、看起来（需要确认的观察）
- 可能、大概、或许（确定程度）

### 4. 拟人化

**问题：** 将人类特征归于非人类实体

**示例：**

| 拟人化 | 科学 |
|--------|------|
| "这项研究想要检验......" | "我们旨在检验......" 或 "这项研究检验......" |
| "数据表明它们想要......" | "数据表明......" |
| "本文将证明......" | "本文展示......" |
| "表1告诉我们......" | "表1显示......" |

### 5. 缩写滥用

**问题：**
- 过多的缩写会增加读者负担
- 缩写仅使用一两次的术语
- 首次使用时未定义缩写

**准则：**
- 仅对使用≥3-4次的术语进行缩写
- 如果在摘要中使用，则在摘要中首次使用时定义
- 在正文中首次使用时定义
- 标题中不要缩写
- 尽可能将每篇论文的新缩写限制在3-4个
- 使用标准缩写（DNA、RNA、HIV等）无需定义

**示例：**
```
较差："我们在基线时测量了脑源性神经营养因子（BDNF）。BDNF水平升高。"
（仅使用两次，缩写不必要）

更好："我们在基线时测量了脑源性神经营养因子。水平升高。"
```

## 具体的句子层面问题

### 悬垂修饰语

**问题：**
```
"孵化2小时后，我们测量了吸光度。"
（句子暗示"我们"被孵化）

更好："孵化样本2小时后，我们测量了吸光度。"
或者："2小时孵化后，我们测量了吸光度。"
```

### 逗号位置不当

**常见错误：**

**主语和动词之间：**
```
错误："干预组的参与者，显示了改善。"
正确："干预组的参与者显示了改善。"
```

**复合谓语中：**
```
错误："我们测量了血压，并记录了症状。"
正确："我们测量了血压并记录了症状。"
（当"and"不连接独立分句时，不要在"and"前加逗号）
```

### 代词一致

```
错误："每位参与者完成了他们的问卷。"
正确："每位参与者完成了他或她的问卷。"
或者更好："参与者完成了他们的问卷。"
```

### 主谓一致

```
错误："参与者群体是异质的。"
正确："参与者群体是异质的。"
（主语是"群体"[单数]，不是"参与者"）

但是："参与者是异质的。"（复数主语）
```

## 词汇选择

### 科学写作中常混淆的词汇

| 常误用 | 正确用法 |
|--------|----------|
| **affect / effect** | Affect（动词）：影响；Effect（名词）：结果；Effect（动词）：引起 |
| **among / between** | Among：三个或更多；Between：两个 |
| **continual / continuous** | Continual：重复的；Continuous：不间断的 |
| **data is / data are** | Data are（复数）；datum是（单数） |
| **fewer / less** | Fewer：可数项目；Less：连续量 |
| **i.e. / e.g.** | i.e.（即）：重述；e.g.（例如）：例子 |
| **imply / infer** | Imply：暗示；Infer：推断 |
| **parameter / variable** | Parameter：总体值；Variable：测量特征 |
| **principal / principle** | Principal：主要的；Principle：规则或概念 |
| **significant** | 保留用于统计显著性，而非重要性 |
| **that / which** | That：限制性从句；Which：非限制性从句 |

### 应避免或谨慎使用的词汇

**避免非正式语言：**
- "a lot of" → "many" 或 "substantial"
- "got" → "obtained" 或 "became"
- "showed up" → "appeared" 或 "was evident"

**避免模糊的量词：**
- "some" → 具体说明有多少
- "often" → 具体说明频率
- "recently" → 具体说明时间范围

**避免不必要的修饰词：**
- "very significant" → "significant"（p值显示程度）
- "quite large" → "large" 或具体说明大小
- "rather interesting" → 删除或解释为什么

## 数字和单位

### 何时使用数字 vs 文字

**使用数字：**
- 所有≥10的数字
- 带单位的数字（5 mg, 3 mL）
- 统计值（p=0.03, t=2.14）
- 年龄、日期、时间
- 分数和量表
- 百分比（15%）

**使用文字：**
- 未与单位连接的<10的数字（five participants）
- 句子开头的数字（拼出或重新组织）

**示例：**
```
"Five participants withdrew" 或 "There were 5 withdrawals"
（不要："5 participants withdrew"）

"We tested 15 samples at 3 time points"
"Mean age was 45 years"
```

### 单位和建议

**准则：**
- 数字和单位之间有空格（5 mg，不是5mg）
- 单位后不加句点（mg不是mg.）
- 除非领域惯例不同，否则使用SI单位
- 小数位数保持一致
- 文本中千位用逗号（12,500不是12500）

**范围：**
- 范围使用短横线（–）：15–20 mg
- 单位仅在第二个数字后出现：15–20 mg（不是15 mg–20 mg）

## 段落结构

### 理想的段落长度

**准则：**
- 通常3-7句话
- 每个段落一个主要思想
- 太短（<2句话）：可能表明观点需要发展或合并
- 太长（>10句话）：可能需要拆分

### 段落连贯性

**技巧：**

**1. 主题句：**
```
"运动训练通过多种机制改善心血管功能。[后续句子解释这些机制]"
```

**2. 过渡短语：**
- 首先，其次，第三，最后
- 此外，而且，另外
- 然而，尽管，相反
- 因此，于是，结果
- 例如，特别是，具体而言

**3. 关键术语重复：**
```
"...这个作用机制。这个机制可能解释......"
（不是："...这个机制。这个过程可能解释......"）
```

**4. 平行结构：**
```
"A组接受了药物。B组接受了安慰剂。C组没有接受治疗。"
（不是："A组接受了药物。安慰剂给了B组。第三组没有接受治疗。"）
```

## 不同发表场所的写作风格

### 四大写作风格类别

1. **面向广泛受众**（Nature, Science, PNAS）
2. **临床专业**（NEJM, Lancet, JAMA）
3. **技术专业**（领域特定期刊）
4. **ML会议**（NeurIPS, ICML, ICLR, CVPR）

### 写作风格比较

| 方面 | Nature/Science | 医学 | 专业期刊 | ML会议 |
|------|---------------|------|---------|--------|
| **句子长度** | 15-20词 | 12-18词 | 18-25词 | 12-20词 |
| **词汇** | 最少术语 | 临床术语 | 领域特定 | 技术+数学 |
| **语调** | 引人入胜，重要 | 保守 | 正式 | 直接，注重贡献 |
| **关键短语** | "Here we show" | "We conducted" | "To elucidate" | "We propose", "Our contributions" |

**ML会议风格：**

**特点：**
- 使用数学符号的直接技术语言
- 注重贡献（常见编号列表）
- 假设具有ML专业知识（CNN、transformer、SGD等）
- 强调新颖性和性能提升
- 预期有伪代码和方程

**示例开场（NeurIPS风格）：**
```
Vision transformers have achieved state-of-the-art performance on image classification,
but their quadratic complexity limits applicability to high-resolution images. We propose
Efficient-ViT, which reduces complexity to O(n log n) while maintaining accuracy. Our
contributions are: (1) a novel sparse attention mechanism, (2) theoretical analysis
showing preserved expressive power, and (3) empirical validation on ImageNet showing
15% speedup with comparable accuracy.
```
- 阐明技术背景下的研究问题
- 预览解决方案
- 编号的贡献
- 定量论断

### 关键写作差异

| 方面 | Nature/Science | 医学 | 专业期刊 | ML会议 |
|------|---------------|------|---------|--------|
| **段落长度** | 3-5句 | 5-7句 | 6-10句 | 4-6句 |
| **数学/方程** | 尽量少 | 罕见 | 中等 | 必不可少 |
| **主动语态** | 首选 | 混合 | 被动可接受 | 首选 |
| **谨慎表达** | 中等 | 保守 | 详细 | 极少（声称成果） |
| **图表整合** | 紧密 | 系统 | 详细 | 密集，页内 |

### 不同场所的评估重点

| 场所 | 关键评估标准 |
|------|-------------|
| **Nature/Science** | 非专业人士能否理解？广泛重要性是否清晰？故事是否引人？ |
| **医学** | 临床相关性是否明显？专业语调？方法是否充分？ |
| **专业期刊** | 技术精确性？领域专业知识是否体现？方法是否详细？ |
| **ML会议** | 贡献是否清晰？实验是否支持论断？可复现性？ |

**常见退稿原因：**
- 写作质量差/表述不清
- 风格不适合场所
- 声称过度
- 方法不足以复现
- 缺少关键细节（ML的基线、消融；期刊的统计）

### 快速风格适应指南

| 从 → 到 | 关键变化 |
|--------|---------|
| **期刊 → ML会议** | 添加编号贡献；包括方程/伪代码；强调定量提升；精简论述 |
| **ML会议 → 期刊** | 移除贡献编号；扩展动机；分离结果/讨论；减少正文中的方程 |
| **专业 → 广泛** | 简化语言；强调广泛含义；解释技术概念；为非专家添加背景 |
| **广泛 → 专业** | 添加技术细节；自由使用领域术语；扩展机制讨论；引用领域文献 |
| **基础科学 → 临床** | 添加患者/临床背景；使用临床语言；强调结果/意义；引用临床证据 |

### 提交前风格清单

**所有场所：**
- [ ] 写作风格与该场所最近3-5篇论文匹配
- [ ] 句子长度适当
- [ ] 技术词汇水平正确
- [ ] 语调与场所一致
- [ ] 没有夸大的声称

**ML会议额外：**
- [ ] 贡献在引言中清晰编号
- [ ] 数学符号正确一致
- [ ] 伪代码/算法在适当位置包含
- [ ] 声称已量化（例如"快15%"，"准确率提升2.3%"）
- [ ] 已承认局限性

## 结语

有效的科学写作是一项通过实践发展的技能。关键原则：

1. **清晰性**优于复杂性
2. **简洁性**尊重读者时间
3. **准确性**建立可信度
4. **客观性**维护科学完整性
5. **一致性**帮助理解
6. **逻辑组织**引导读者
7. **适应期刊特定要求**最大化发表成功

**记住：** 目标不是用词汇或复杂性给读者留下深刻印象，而是清晰精确地传达你的科学，以便读者能够理解、评价并在此基础上进行构建。根据目标期刊的期望和受众调整你的写作风格。
