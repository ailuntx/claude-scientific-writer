# 图表与表格最佳实践

## 概述

图表与表格是科学论文的重要组成部分，用于展示数据模式、总结结果并为结论提供依据。有效的视觉呈现能够增强可读性，在展示文本难以传达的趋势、模式和关系时，能够保持读者的阅读兴趣。

最新的《自然细胞生物学》检查清单（2025年）强调，创建清晰且引人入胜的科学图表对于以清晰、可及和设计卓越的方式传达复杂数据至关重要。

## 何时使用表格 vs. 图表

### 使用表格的情况：
- 呈现需要读者参考的精确数值
- 比较多个变量之间的精确测量值
- 显示详细的统计输出
- 数据无法在1-2句话内充分总结
- 读者需要访问特定数据点
- 显示人口统计学或基线特征
- 呈现多个相关的统计检验

**示例用例：**
- 基线参与者特征（年龄、性别、诊断等）
- 详细统计模型输出（系数、p值、置信区间）
- 具有精确值的剂量反应数据
- 特定基因的表达水平
- 化学成分或浓度

### 使用图表的情况：
- 显示随时间的变化趋势
- 展示关系或相关性
- 视觉上比较组别
- 说明分布
- 展示数值中不易看到的模式
- 显示图像（显微镜、放射学等）
- 展示工作流程、图表或示意图

**示例用例：**
- 增长曲线或时间序列
- 剂量反应曲线
- 显示相关性的散点图
- 比较治疗组的条形图
- 显示分布的直方图
- 跨条件显示模式的热图
- 显微镜图像或Western blot

### 一般决策规则

**信息是否可以用1-2句文字传达？**
- 是 → 仅使用文字
- 否，且需要精确值 → 使用表格
- 否，且模式/趋势最重要 → 使用图表

## 核心设计原则

### 1. 自明性展示项

**每个图表或表格必须能够独立存在，无需参考正文。**

**必要元素：**
- 完整、描述性的标题
- 所有缩写词已定义（在标题或脚注中）
- 测量单位清晰标明
- 样本量（n）已报告
- 统计显著性标注已解释
- 图例已包含（对于包含多个数据系列的图表）

**自明性标题示例：**
```
图1. 干预组和对照组12周内的平均收缩压（SBP）。误差条表示均值的标准误（SEM）。
星号表示各时间点两组之间的显著差异（*p < 0.05，**p < 0.01，***p < 0.001，双侧t检验）。
每组n = 48。BP = 血压；SEM = 均值的标准误。
```

### 2. 避免重复

**不要在正文、表格和图表之间重复信息。**

**不良做法：**
```
"平均年龄：A组45.2岁，B组47.8岁。平均BMI：A组26.3，B组28.1。
平均收缩压：A组132 mmHg..."
[亦显示于表1]
```

**良好做法：**
```
"两组基线特征相似（表1），年龄、BMI或血压无显著差异（均p > 0.15）。"
[详细信息见表1]
```

**核心原则：** 正文应突出表格/图表的主要发现，而非重复所有数据。

### 3. 一致性

**保持所有展示项的格式统一：**
- 字体类型和大小
- 颜色方案
- 术语和缩写
- 轴标签和单位
- 统计标注方法
- 图表样式（所有折线图应看起来相似）

**应避免的不一致示例：**
- 图1使用"standard error"，而图2使用"SE"
- 图1采用蓝/红色方案，而图2使用绿/黄色
- 表1报告p值为"p = 0.023"，而表2使用"p-value = .023"

### 4. 最优数量

**遵循"每1000字一个展示项"的指南。**

**典型手稿：**
- 3000-4000字 → 总共3-4个表格/图表
- 5000-6000字 → 总共5-6个表格/图表

**质量胜于数量：** 几个精心设计、信息丰富的展示项优于许多冗余或设计糟糕的展示项。

### 5. 清晰与简洁

**避免杂乱或过于复杂的展示：**
- 不要在一个图表中包含太多变量
- 使用清晰、可读的字体（最终尺寸至少8-10 pt）
- 元素之间提供足够的间距
- 使用高对比度（尤其是对色盲人士的可及性）
- 移除不必要的网格线、边框或装饰
- 最大化数据墨水比（Tufte原则：最小化非数据墨水）

## 图表类型及其使用场景

### 条形图

**最适用于：**
- 比较离散类别或组
- 显示计数或频率
- 展示带误差条的均值

**设计指南：**
- y轴从零开始（除非显示大数值中的小差异）
- 按逻辑顺序排列条形（按大小、字母顺序或时间顺序）
- 一致地使用误差条（SD、SEM或CI）
- 包含样本量
- 避免3D效果（会扭曲感知）

**常见错误：**
- 不从零开始（可能夸大差异）
- 类别太多（考虑用表格）
- 缺少误差条

**示例应用：**
- 跨组织类型的基因表达均值
- 治疗组比较
- 不良事件频率

### 折线图

**最适用于：**
- 显示连续变量（通常为时间）上的趋势
- 在同一坐标轴上显示多个组
- 展示剂量-反应关系

**设计指南：**
- 对不同组使用不同的线条样式或颜色
- 对稀疏数据包含数据点标记
- 显示误差条或阴影置信区间
- 清晰标注轴及单位
- x轴使用一致的间隔

**常见错误：**
- 连接不应连接的离散数据点
- 线条太多导致图表无法阅读
- 时间间隔不一致且无说明

**示例应用：**
- 增长曲线
- 时间进程实验
- 生存曲线（Kaplan-Meier图）
- 药代动力学曲线

### 散点图

**最适用于：**
- 显示两个连续变量之间的关系
- 展示相关性
- 识别异常值

**设计指南：**
- 包含趋势线或回归线，包括方程和R²
- 报告相关系数和p值
- 如果数据重叠，使用半透明点
- 对宽范围考虑使用对数刻度
- 如相关，标记异常值

**常见错误：**
- 不显示单个数据点
- 对分类数据使用散点图
- 缺少相关性统计

**示例应用：**
- 生物标志物之间的相关性
- 剂量与反应之间的关系
- 方法比较（Bland-Altman图）

### 箱线图（箱须图）

**最适用于：**
- 显示分布和离散程度
- 跨组比较分布
- 识别异常值

**设计指南：**
- 清晰定义箱体元素（中位数、四分位数、须）
- 明确显示或注明异常值
- 对小样本考虑使用小提琴图
- 当n < 20时叠加单个数据点

**常见错误：**
- 未定义须代表什么
- 对非常小的样本使用且不显示原始数据
- 未标记异常值

**示例应用：**
- 跨治疗组比较分布
- 显示测量的变异性
- 质量控制数据

### 热图

**最适用于：**
- 显示数据矩阵
- 跨许多条件显示模式
- 表示聚类或分组

**设计指南：**
- 使用感知均匀的颜色刻度
- 包含带单位的颜色刻度条
- 考虑对行/列进行层次聚类
- 使用适当的颜色方案（发散 vs. 顺序）
- 使轴标签可读

**常见错误：**
- 颜色选择不当（彩虹刻度通常具有误导性）
- 行/列太多导致标签无法阅读
- 无颜色刻度条

**示例应用：**
- 跨样本的基因表达
- 相关矩阵
- 跨多个变量的时间序列数据

### 图像（显微镜、凝胶、印迹）

**最适用于：**
- 显示代表性示例
- 展示形态或定位
- 呈现凝胶电泳或Western blot

**设计指南：**
- 包含比例尺（而非标题中的放大倍数）
- 在单独面板中显示代表性图像和量化
- 用箭头或标签标注重要特征
- 确保足够的分辨率（通常300+ dpi）
- 显示完整、未处理的图像，并注明裁剪
- 包含所有相关对照

**常见错误：**
- 无比例尺
- 过度处理或处理的图像
- 仅挑选最佳图像而无量化
- 分辨率不足

**示例应用：**
- 组织学切片
- 免疫荧光
- Western blot
- 凝胶电泳

### 森林图

**最适用于：**
- 显示荟萃分析结果
- 展示效应量和置信区间
- 比较多个研究或亚组

**设计指南：**
- 为每个研究包含点估计和CI
- 清晰显示总体合并估计
- 包含无效应线（通常为1.0或0）
- 列出研究详情或权重

**示例应用：**
- 荟萃分析
- 系统综述
- 亚组分析

### 流程图

**最适用于：**
- 研究参与者流程（CONSORT图）
- 系统综述检索过程（PRISMA图）
- 实验工作流程

**设计指南：**
- 遵循报告指南模板（CONSORT、PRISMA）
- 使用一致的形状和连接符
- 在每个阶段包含数字
- 清晰显示入组和排除

## 表格设计指南

### 结构

**基本组成：**
1. **表格编号和标题**（位于表格上方）
2. **列标题**（含单位）
3. **行标签**
4. **数据单元格**（适当的精度）
5. **脚注**（位于表格下方，用于缩写词、统计、说明）

### 格式最佳实践

**列标题：**
- 使用清晰、简洁的标签
- 单位放在括号内
- 谨慎使用缩写（在脚注中定义）

**数据呈现：**
- 列中小数点对齐
- 使用一致的小数位（均值通常1-2位）
- 跨行/列报告相同精度
- 对"不适用"使用短横线（–）
- 使用适当的精度（不要过度报告）

**统计标注：**
- 使用上标字母（ᵃ, ᵇ, ᶜ）或符号（*, †, ‡）作为脚注
- 清晰定义p值阈值
- 尽可能报告精确p值（p = 0.032，而非p < 0.05）

**脚注：**
- 定义所有缩写词
- 解释所使用的统计检验
- 注明任何缺失数据
- 如非原始数据，注明数据来源

### 示例表格格式

```
表1. 研究参与者的基线特征

特征                    干预组 (n=50)    对照组 (n=48)    p值
─────────────────────────────────────────────────────────────────────────
年龄，岁                 45.3 ± 8.2       47.1 ± 9.1       0.28
男性，n (%)             28 (56)          25 (52)          0.71
BMI，kg/m²              26.3 ± 3.8       27.1 ± 4.2       0.32
当前吸烟者，n (%)       12 (24)          15 (31)          0.42
收缩压，mmHg            132 ± 15         134 ± 18         0.54
─────────────────────────────────────────────────────────────────────────

数据以均值 ± SD或n (%)呈现。p值来自连续变量的独立t检验和分类变量的χ²检验。
BMI = 体质指数；BP = 血压；SD = 标准差。
```

### 常见表格错误

1. **过度复杂**（行/列太多）
2. **背景信息不足**（缺少单位、缩写不清晰）
3. **过度精确**（p值报告5位小数）
4. **缺少样本量**
5. **该做统计比较时未做**
6. **跨多个表格格式不一致**
7. **与图表或正文重复信息**

## 图表与表格中的统计呈现

### 报告要求

**对于每个比较，报告：**
1. **点估计**（均值、中位数、比例）
2. **变异性度量**（SD、SEM、CI）
3. **样本量**（n）
4. **检验统计量**（t、F、χ²等）
5. **p值**（当p > 0.001时精确）
6. **效应量**（适当时）

### 误差条

**选择适当的度量：**

| 度量 | 含义 | 何时使用 |
|---------|---------|-------------|
| **SD（标准差）** | 数据中的变异性 | 显示数据分布 |
| **SEM（均值的标准误）** | 均值估计的精度 | 显示测量精度 |
| **95% CI（置信区间）** | 可能包含真实均值的范围 | 显示统计显著性 |

**核心规则：** 始终说明显示的是哪种度量。

**示例标题：**
```
"误差条表示95%置信区间。"
而非："误差条表示标准误。"
```

**推荐：** 首选95% CI，因为非重叠的CI表示显著差异。

### 显著性指示符

**常见符号：**
```
* p < 0.05
** p < 0.01
*** p < 0.001
n.s.或NS = 不显著
```

**替代方案：** 在表格或标题中显示精确p值

**最佳实践：** 在每个图表标题或表格脚注中定义显著性指示符。

## 可及性考虑

### 色盲友好设计

**建议：**
- 使用专为色盲设计的配色方案
- 不仅依赖颜色（添加图案、形状或标签）
- 在灰度模式下测试图表
- 避免红绿组合

**色盲安全配色：**
- 蓝-橙
- 紫-黄
- Colorbrewer2.org配色
- Viridis、Plasma、Inferno（用于热图）

### 高对比度

**确保可读性：**
- 深色文字配浅色背景（或反之）
- 避免低对比度颜色组合（灰配灰）
- 使用足够粗的线条（至少0.5-1 pt）
- 足够大的文字（缩放后至少8-10 pt）

### 屏幕和打印兼容性

**为两种媒介设计：**
- 尽可能使用矢量格式（PDF、EPS、SVG）
- 光栅图像至少300 dpi（TIFF、PNG）
- 在最终打印尺寸下测试外观
- 确保彩色图表在灰度打印时可用

## 技术要求

### 文件格式

**矢量格式**（图表和图表首选）：
- **PDF**：通用，保持质量
- **EPS**：封装PostScript，出版标准
- **SVG**：可缩放矢量图形，网页友好

**光栅格式**（用于照片和图像）：
- **TIFF**：未压缩，高质量，文件大
- **PNG**：无损压缩，适合屏幕
- **JPEG**：有损压缩，避免用于数据图表

**避免：**
- 低分辨率截图
- 从演示文稿复制的图表（通常分辨率过低）
- 严重压缩的JPEG（伪影）

### 分辨率要求

**最低标准：**
- **线条艺术**（图表、图表）：300-600 dpi
- **半色调**（照片、灰度）：300 dpi
- **组合**（带标签的图像）：300-600 dpi

**最佳实践：** 以最终尺寸和分辨率创建图表。

### 尺寸

**检查期刊要求：**
- **单栏**：通常宽8-9厘米（3-3.5英寸）
- **双栏**：通常宽17-18厘米（6.5-7英寸）
- **整页**：因期刊而异

**建议：** 设计图表时尽可能适合单栏。

### 图像处理

**允许：**
- 应用于整个图像的亮度/对比度调整
- 色彩平衡调整
- 裁剪（注明）
- 旋转

**不允许：**
- 选择性编辑（例如增强凝胶中的条带）
- 去除背景伪影
- 拼接图像而无明确指示
- 任何掩盖、消除或歪曲数据的处理

**伦理要求：** 在方法部分报告所有图像调整。

## 图表与表格编号

### 编号系统

**图表：**
- 按首次在正文中提及的顺序连续编号
- 使用阿拉伯数字：图1、图2、图3...
- 补充图：图S1、图S2...

**表格：**
- 与图表分开编号
- 使用阿拉伯数字：表1、表2、表3...
- 补充表：表S1、表S2...

### 正文引用格式

```
"结果如图1所示。"
"参与者特征见表2。"
"多项分析确认了这一发现（图3-5）。"
```

**不要：**
```
"图1显示..."（避免"上方"或"下方"——分页可能会改变）
"该图显示..."（始终使用具体编号）
```

## 标题

### 标题结构

**对于图表：**
```
图1. [一句话标题]。[提供背景、定义缩写、解释面板、描述统计检验和注明样本量的额外描述句子]。
```

**对于表格：**
```
表1. [描述性标题]
[表格内容]
[定义缩写、统计方法并提供额外背景的脚注]
```

### 标题内容

**必要信息：**
1. 显示的内容（简要标题）
2. 内容的详细描述
3. 所有缩写和符号的定义
4. 样本量
5. 使用的统计检验
6. 误差条或标注的含义
7. 面板标签的解释（如果是多面板）

**示例完整标题：**
```
图3. 认知功能随治疗在12周内改善。（A）治疗组（蓝色）和安慰剂组（灰色）在基线、6周和12周时的平均简易精神状态检查（MMSE）分数。（B）治疗组个体参与者轨迹。误差条表示95%置信区间。星号表示组间显著差异（*p < 0.05，**p < 0.01，***p < 0.001；重复测量方差分析，Bonferroni校正）。n = 42治疗组，n = 40安慰剂组。MMSE分数范围0-30，分数越高表示认知功能越好。
```

## 期刊特定要求

### 创建图表/表格之前

**检查期刊指南的：**
- 首选文件格式
- 分辨率要求
- 颜色规格（RGB vs. CMYK）
- 展示项最大数量
- 尺寸要求
- 字体限制
- 是否将图表嵌入稿件或单独提交

### 提交期间

**准备检查清单：**
- [ ] 所有图表/表格正确编号
- [ ] 按顺序在正文中全部引用
- [ ] 标题完整且自明
- [ ] 缩写词已定义
- [ ] 正确的文件格式和分辨率
- [ ] 适当的尺寸/尺寸
- [ ] 质量足以打印
- [ ] 色盲友好（如使用颜色）
- [ ] 已获得许可（如改编他人作品）

## 应避免的常见陷阱

### 内容问题
1. 正文、表格和图表之间的**重复**
2. **背景信息不足**（不清楚显示的内容）
3. **一个展示项中信息过多**
4. **缺少关键信息**（样本量、单位、统计）
5. **挑选数据**而不显示全貌

### 设计问题
6. **颜色选择不当**（非色盲友好）
7. **跨展示项格式不一致**
8. **杂乱或繁忙的设计**
9. **最终尺寸文字过小**
10. **误导性可视化**（截断轴、3D扭曲）

### 技术问题
11. **分辨率不足**（打印时像素化）
12. **文件格式错误**（有损压缩、非矢量图表）
13. **不当的图像处理**（未声明的编辑）
14. **图像上缺少比例尺**
15. **灰度模式下不适用的图表**（如期刊黑白印刷）

## 期刊特定的图表与表格要求

### 理解期刊期望

**不同期刊对图表和表格的要求差异很大。** 在创建展示项之前，始终查阅目标期刊的作者指南以了解具体要求。

### 常见期刊特定差异

| 方面 | 期刊差异 | 示例期刊 |
|--------|---------------------|------------------|
| **允许数量** | 研究文章4-10个展示项 | Nature（4-6），PLOS ONE（无限制），Science（4-5） |
| **文件格式** | TIFF、EPS、AI或特定格式 | Nature（EPS/PDF用于线条艺术），Cell（首选TIFF） |
| **分辨率** | 300-1000 dpi取决于类型 | JAMA（300-600 dpi），Nature（300+ dpi） |
| **颜色** | RGB vs. CMYK | 印刷期刊：CMYK；在线：RGB |
| **尺寸** | 单栏 vs. 双栏宽度 | Nature（89mm或183mm），Science（特定模板） |
| **图例** | 长度限制，特定格式 | 一些期刊：每个图例最多150词 |
| **表格格式** | 可编辑 vs. 图像 | 大多数偏好可编辑表格，而非图像 |

### 场地特定要求总结

| 场地类型 | 展示限制 | 格式 | 分辨率 | 关键特点 |
|-----------|--------------|--------|------------|--------------|
| **Nature/Science** | 4-6个主要 | EPS/PDF/TIFF | 300+ dpi | 允许扩展数据；多面板图表 |
| **医学期刊** | 3-5个 | TIFF/EPS | 300-600 dpi | CONSORT图；保守设计 |
| **PLOS ONE** | 无限制 | TIFF/EPS/PDF | 300+ dpi | 必须在灰度下可用 |
| **ML会议** | 8页限制中4-6个 | PDF（首选矢量） | 打印质量 | 紧凑设计；信息密集图表 |

**ML会议图表要求：**

**NeurIPS/ICML/ICLR：**
- 图表计入页数限制（通常包括参考文献8页）
- 矢量图形（PDF）用于图表首选
- 预期高信息密度
- 补充材料用于额外图表
- 提供LaTeX模板（使用neurips_2024.sty或等效模板）
- 图表在灰度打印时必须清晰

**计算机视觉（CVPR/ICCV/ECCV）：**
- 定性结果图表至关重要
- 并排比较是标准
- 必须展示失败案例
- 补充材料用于视频/额外示例
- 8页论文中通常6-8个主要图表

**关键ML会议图表实践：**
- **消融研究**：紧凑的表格/图显示组件贡献
- **架构图**：清晰、专业的模块图
- **性能图**：包含误差条/置信区间
- **定性示例**：显示多样化、代表性样本
- **比较表格**：简洁，突出最佳结果

### 各场地评估标准

**审查者检查的内容：**
- **必要性**：每个图表/表格支持结论
- **质量**：专业外观，足够分辨率
- **清晰度**：标题自明且正确标注
- **统计**：误差条、样本量、显著性指示符
- **一致性**：格式在所有展示项中统一

**常见拒绝原因：**
- 分辨率或图像质量差
- 缺少误差条或样本量
- 标注不清晰或缺失
- 图表太多（超出场地限制）
- 图表重复正文信息

**ML会议特定评估：**
- **消融研究**：必须展示组件贡献
- **基线**：需要与相关先前工作的比较
- **误差条**：预期置信区间/标准差
- **架构图**：必须清晰且信息丰富
- **空间效率**：重视信息密度（页数限制严格）

### 各场地标题/图例样式

| 场地类型 | 样式 | 示例特点 |
|-----------|-------|------------------|
| **Nature/Science** | 简洁 | 简短；*P<0.05；最少方法 |
| **医学** | 正式 | 首字母大写；95% CI；拼出统计检验 |
| **PLOS/BMC** | 详细 | 完整句子；定义所有缩写 |
| **ML会议** | 技术 | 架构细节；超参数；数据集信息 |

**ML会议标题示例：**
```
图1. 所提模型的架构。（a）12层Transformer编码器。（b）注意力可视化。（c）ImageNet上与基线的性能比较（误差条：3次运行的95% CI）。
```
- 技术精度
- 相关时包含超参数
- 数据集/实验设置详情
- 紧凑以节省空间

### 快速适应指南

**改变场地时：**
- **期刊 → ML会议**：压缩图表；增加信息密度；在标题中添加超参数
- **ML会议 → 期刊**：扩展标题；分离密集图表；添加更多方法学细节
- **专业 → 广泛期刊**：简化；添加解释面板；在标题中定义术语
- **广泛 → 专业期刊**：添加技术细节；使用领域标准图表类型
