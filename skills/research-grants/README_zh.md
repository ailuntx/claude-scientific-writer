# 研究资助技能

## 概述

用于撰写具有竞争力研究资助提案的综合技能，涵盖四个主要美国资助机构：

- **NSF**（美国国家科学基金会）
- **NIH**（美国国立卫生研究院）
- **DOE**（美国能源部）
- **DARPA**（美国国防高级研究计划局）

## 本技能提供的内容

### 机构特定指南

每个资助机构的详细参考资料，包括：

- 使命与优先事项
- 评审标准与评分
- 提案结构与页数限制
- 预算要求
- 提交流程
- 具有竞争力申请的建议

### 核心组件

- **具体目标页**（NIH）：关键1页目标页的模板与详细指南
- **项目摘要**（NSF）：概述、学术价值与社会影响的必需模板
- **社会影响**：NSF同等权重评审标准的综合策略
- **预算说明**：人员、设备、旅行与材料的模板与示例
- **评审标准**：了解每个机构评审者关注的内容

### 模板

即用型模板：

- NSF项目摘要
- NIH具体目标页
- 预算说明
-（更多模板开发中）

## 如何使用本技能

## 主要特点

### NSF提案

- **学术价值 + 社会影响**（同等权重）
- 实质性、可衡量社会影响的策略
- 研究与教育整合
- 扩大STEM参与度
- 15页项目描述限制（大多数项目）

### NIH提案

- **具体目标页**：最关键的一页（包含详细1页指南）
- **研究策略**：重要性、创新性、方法部分
- **初步数据**：R01申请的必要条件
- 严谨性与可重复性要求
- 模块化与详细预算
- 重新提交策略（A1申请）

### DOE提案

- **能源相关性**与DOE使命一致性
- **技术就绪水平**（TRL）
- 国家实验室合作
- 成本分摊要求（尤其是ARPA-E）
- 商业化路径
- 用户设施访问

### DARPA提案

- **DARPA难题**：高风险、高回报
- **Heilmeier问答**：8个关键问题
- 项目经理互动（至关重要！）
- 基于阶段的结构与里程碑
- 技术转化规划
- 演示与原型

## 成功指标

各机构典型成功率：

- **NSF**：15-30%（因项目而异）
- **NIH R01**：总体约20%（早期研究者约27%）
- **DOE科学办公室**：20-40%（因项目而异）
- **ARPA-E**：2-5%（概念论文至获奖）
- **DARPA**：因项目差异很大

## 最佳实践

### 及早开始

- NSF/NIH提案：截止日期前3-6个月开始
- DOE/DARPA提案：4-6个月（特别是涉及国家实验室时）

### 获取反馈

- 模拟评审会议
- 领域内外的同事
- 机构资助支持办公室
- 项目官员（适当时）

### 了解评审标准

- NSF：学术价值 + 社会影响（同等权重）
- NIH：重要性、研究者、创新性、方法、环境（评分1-9）
- DOE：技术优点、资质、预算、相关性
- DARPA：创新、影响、团队、可行性、转化

### 常见成功因素

✅ 清晰、引人注目的重要性与创新性
✅ 强劲的初步数据（NIH、DOE）
✅ 详细、严谨的方法论
✅ 现实的进度安排与预算
✅ 具体、可衡量的成果
✅ 具有相关专业知识的强大团队
✅ 社会影响整合（NSF）
✅ 技术转化计划（DOE、DARPA）

## 最大化效果的建议

### NSF提案

1. 从具体目标/目标开始（虽然不是必需的）
2. 以与研究计划相同的严谨度开发社会影响
3. 大量使用图表和图示（使其易于浏览）
4. 明确解决两个评审标准
5. 获取本领域之外人员的反馈

### NIH提案

1. 首先完善你的具体目标页（10+次修改）
2. 包含实质性的初步数据
3. 明确解决严谨性与可重复性
4. 主动识别潜在问题及替代方案
5. 确保你的目标是独立的但具有协同作用

### DOE提案

1. 强调能源相关性与影响
2. 包含量化指标（成本、效率、排放）
3. 开发部署或商业化路径
4. 考虑国家实验室合作
5. 解决技术就绪水平

### DARPA提案

1. 及早联系项目经理（至关重要！）
2. 参加提案日活动
3. 专注于突破性创新（10倍，而非10%）
4. 明确回答Heilmeier问答
5. 制定清晰的转化策略

## 问题或疑问？

本技能设计全面，但可能无法涵盖所有具体情况。使用本技能时：

1. **具体说明**你的机构、项目与资助类型
2. **提供背景**关于你的研究领域与职业阶段
3. **提出后续问题**以获取澄清
4. **请求示例**用于你正在撰写的具体部分

**请记住**：资助撰写既是一门艺术也是一门科学。本技能提供了框架、策略和最佳实践——但你独特的研究愿景、初步数据和团队专业知识才是最终赢得资助的关键。及早开始，寻求反馈，反复修改，不要因拒绝而气馁。即使最成功的科学家在获得资助之前也会面临许多被拒绝的提案。
