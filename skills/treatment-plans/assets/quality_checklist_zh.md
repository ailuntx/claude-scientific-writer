# 治疗计划质量保证检查清单

## 概述

使用此检查清单确保治疗计划符合专业标准的完整性、质量、安全性和合规性。在最终确定计划之前，请审查每个部分。

---

## 第一部分：完整性 - 必需组件

### ☐ 患者信息
- [ ] 患者标识符（如需共享则去标识化）
- [ ] 年龄范围（不要使用具体出生日期）
- [ ] 性别和相关人口统计学信息
- [ ] 计划创建日期
- [ ] 医疗机构提供者姓名和资质
- [ ] 机构/诊所名称
- [ ] 已包含 HIPAA 去标识化通知

### ☐ 诊断和评估
- [ ] 明确说明主要诊断
- [ ] 包含 ICD-10 代码
- [ ] 列出次要诊断和合并症
- [ ] 记录疾病严重程度/分期
- [ ] 评估基线功能状态
- [ ] 进行风险分层（如适用）

### ☐ 治疗目标
- [ ] 存在短期目标（1-3 个月）
- [ ] 存在长期目标（6-12 个月）
- [ ] 目标符合 SMART 标准（见第二部分）
- [ ] 包含以患者为中心的目标
- [ ] 目标已优先排序或组织

### ☐ 干预措施
- [ ] 指定药物干预措施
- [ ] 包含非药物干预措施
- [ ] 记录程序性干预或转诊
- [ ] 每项干预措施都有明确的理由
- [ ] 基于循证或符合指南

### ☐ 时间表和安排
- [ ] 定义治疗阶段及持续时间
- [ ] 指定预约频率
- [ ] 安排里程碑评估
- [ ] 说明预期总治疗持续时间

### ☐ 监测参数
- [ ] 确定要追踪的临床结果
- [ ] 记录基线值
- [ ] 指定目标值
- [ ] 定义监测频率
- [ ] 命名评估工具/量表

### ☐ 预期结果
- [ ] 说明主要结果指标
- [ ] 定义成功标准
- [ ] 指出改善时间表
- [ ] 记录治疗修改标准

### ☐ 随访计划
- [ ] 已安排下次预约
- [ ] 指定随访频率
- [ ] 概述沟通计划
- [ ] 包含紧急联系程序

### ☐ 患者教育
- [ ] 记录疾病教育
- [ ] 记录自我管理技能培训
- [ ] 传达警告信号
- [ ] 列出资源和支持

### ☐ 风险缓解和安全
- [ ] 确定潜在不良影响
- [ ] 包含安全监测计划
- [ ] 概述紧急程序
- [ ] 解决并发症预防问题

### ☐ 签名和日期
- [ ] 医疗机构提供者签名行
- [ ] 医疗机构提供者姓名和资质
- [ ] 计划日期
- [ ] 患者确认（如适用）

---

## 第二部分：SMART 目标质量

对于每个治疗目标，验证其是否符合 SMART 标准：

### ☐ 具体
- [ ] 目标明确界定将完成的内容
- [ ] 无模糊语言（例如"改善"、"更好"）
- [ ] 说明具体结果

**示例**："将 HbA1c 从 8.5% 降至 <7%" ✓  
**不符合**："改善糖尿病控制" ✗

### ☐ 可衡量
- [ ] 包含可量化的指标或可观察的标准
- [ ] 记录基线值
- [ ] 指定目标值

**示例**："独立使用助行器步行 300 英尺" ✓  
**不符合**："走得更远" ✗

### ☐ 可实现
- [ ] 考虑到患者状况和能力是现实的
- [ ] 可用资源支持目标
- [ ] 时间范围合理
- [ ] 治疗效果支持目标

**示例**："6 周内将疼痛从 7/10 降至 4/10" ✓  
**不符合**："1 周内消除所有疼痛" ✗

### ☐ 相关
- [ ] 与患者价值观和优先事项一致
- [ ] 具有临床意义
- [ ] 解决患者的功能限制
- [ ] 与整体治疗目标相结合

**示例**："3 个月内带修改条件重返工作" ✓  
**不符合**："实验室数值改善"（如果患者不在意）✗

### ☐ 有时限
- [ ] 说明具体的截止日期或时间范围
- [ ] 定义再评估间隔
- [ ] 指定行动频率（如适用）

**示例**："8 周内"或"到第 3 个月" ✓  
**不符合**："最终"或"很快" ✗

---

## 第三部分：临床质量

### ☐ 循证实践
- [ ] 干预措施基于当前证据
- [ ] 遵循临床实践指南
- [ ] 解释和说明偏离指南的原因
- [ ] 引用文献或证据（如为正式计划）

### ☐ 药物文档（如适用）
- [ ] 使用通用药名
- [ ] 记录具体剂量、途径、频率
- [ ] 为每种药物提供适应症/理由
- [ ] 注明要监测的不良影响
- [ ] 考虑药物相互作用
- [ ] 如适用，包含滴定计划

### ☐ 评估工具
- [ ] 在可用时使用经过验证的评估工具
- [ ] 工具适合病症（PHQ-9、FIM、Berg 等）
- [ ] 记录基线分数
- [ ] 指定目标分数
- [ ] 定义再评估计划

### ☐ 多学科协调（如适用）
- [ ] 定义团队成员的角色
- [ ] 指定提供者之间的沟通计划
- [ ] 解决护理过渡问题
- [ ] 整合专科建议

### ☐ 预防性护理整合
- [ ] 包含年龄适当的筛查
- [ ] 记录疫苗接种时间表
- [ ] 记录生活方式咨询
- [ ] 解决健康维护问题

---

## 第四部分：以患者为中心的护理

### ☐ 共同决策
- [ ] 记录患者偏好
- [ ] 讨论治疗选择
- [ ] 解释风险和收益
- [ ] 将患者价值观纳入目标
- [ ] 考虑替代治疗

### ☐ 健康素养
- [ ] 语言适合患者理解
- [ ] 解释或避免医学术语
- [ ] 使用或计划使用回授法
- [ ] 书面材料在适当的阅读水平

### ☐ 文化能力
- [ ] 考虑文化信仰和实践
- [ ] 解决语言障碍（如需要口译员）
- [ ] 适当进行文化调整
- [ ] 尊重宗教/精神偏好

### ☐ 健康的社会决定因素
- [ ] 筛查社会需求（食物、住房、交通）
- [ ] 确定护理障碍
- [ ] 提供社区资源
- [ ] 解决经济问题（药物费用等）

### ☐ 患者参与
- [ ] 患者积极参与目标设定
- [ ] 提供自我管理支持
- [ ] 根据个人定制患者教育
- [ ] 考虑随访偏好

---

## 第五部分：安全和风险管理

### ☐ 药物安全
- [ ] 记录过敏史
- [ ] 审查多重用药（考虑减药）
- [ ] 适当监测高风险药物
- [ ] 检查药物-药物相互作用
- [ ] 如需要，进行肾脏/肝脏剂量调整

### ☐ 跌倒预防（如相关）
- [ ] 评估跌倒风险
- [ ] 包含跌倒预防策略
- [ ] 推荐环境改造
- [ ] 处方辅助设备

### ☐ 感染预防（如相关）
- [ ] 免疫接种是最新的
- [ ] 如有指征，预防性使用抗生素
- [ ] 患者教育了解感染体征和症状

### ☐ 应急准备
- [ ] 清楚列出紧急警告信号
- [ ] 指定何时拨打 911
- [ ] 定义何时联系提供者
- [ ] 提供紧急联系电话

### ☐ 自杀/暴力风险（心理健康计划）
- [ ] 记录风险评估
- [ ] 如有自杀意念，制定安全计划
- [ ] 处理获取途径限制
- [ ] 提供危机资源（988 生命热线）
- [ ] 随访频率适合风险水平

### ☐ 阿片类药物安全（疼痛管理计划）
- [ ] 完成阿片类药物风险评估（ORT、SOAPP）
- [ ] 记录知情同意讨论
- [ ] 签署治疗协议
- [ ] 检查 PDMP
- [ ] 共同处方纳洛酮
- [ ] 包含 UDS 计划

---

## 第六部分：合规性

### ☐ HIPAA 合规性
- [ ] 保护受保护健康信息（PHI）
- [ ] 如共享则按照安全港方法进行去标识化
- [ ] 如去标识化则移除所有 18 个 HIPAA 标识符
- [ ] 遵循最小必要原则

### ☐ 知情同意
- [ ] 记录同意讨论
- [ ] 验证患者理解
- [ ] 解释风险和收益
- [ ] 讨论替代治疗
- [ ] 记录患者同意

### ☐ 医疗必要性
- [ ] 治疗对诊断是医学必需的
- [ ] 干预措施适合严重程度
- [ ] 证据支持治疗选择
- [ ] 证明频率和持续时间的合理性

### ☐ 计费和编码
- [ ] 包含 ICD-10 诊断代码
- [ ] CPT 手术代码（如计划进行手术）
- [ ] 文档支持计费级别
- [ ] 证明服务的医疗必要性

### ☐ 质量指标支持
- [ ] 元素支持质量报告（HEDIS、MIPS）
- [ ] 遵循慢性病管理协议
- [ ] 记录预防性护理
- [ ] 解决患者安全指标

### ☐ 专科特定法规
- [ ] 遵守 42 CFR Part 2（如为物质使用障碍治疗）
- [ ] 遵循 CDC 阿片类药物指南（如处方阿片类药物）
- [ ] 符合联合委员会标准（如适用）
- [ ] 满足州特定要求

---

## 第七部分：文档标准

### ☐ 清晰性和精确性
- [ ] 适当使用专业医学术语
- [ ] 首次使用时定义缩写
- [ ] 无模糊语言
- [ ] 具体而非模糊的描述

### ☐ 准确性
- [ ] 信息事实正确
- [ ] 基于当前循证建议
- [ ] 药物剂量和频率正确
- [ ] ICD-10 和 CPT 编码正确

### ☐ 组织性
- [ ] 逻辑流程和结构
- [ ] 格式一致
- [ ] 易于定位关键信息
- [ ] 标题和部分清晰标注

### ☐ 可读性（如手写或混合）
- [ ] 笔迹清晰
- [ ] 无不明确的缩写
- [ ] 打字部分清晰
- [ ] 签名清晰，附打印姓名

### ☐ 认证
- [ ] 清楚说明提供者姓名
- [ ] 包含资质
- [ ] 存在计划日期
- [ ] 获得签名（电子或手写）

## 第八部分：按计划类型的特殊注意事项

### 对于一般医疗计划：
- [ ] 遵循慢性病管理协议
- [ ] 使用基于指南的目标（HbA1c、BP、血脂）
- [ ] 优化药物治疗方案
- [ ] 解决合并症
- [ ] 整合预防性护理

### 对于康复计划：
- [ ] 使用经过验证的工具进行功能评估（FIM、Berg）
- [ ] 包含损伤、活动和参与目标
- [ ] 指定治疗频率和持续时间
- [ ] 记录家庭锻炼计划
- [ ] 列出 DME 和环境改造
- [ ] 定义出院标准

### 对于心理健康计划：
- [ ] 符合 DSM-5 诊断标准
- [ ] 评估症状严重程度（PHQ-9、GAD-7 等）
- [ ] 评估自杀/暴力风险
- [ ] 制定安全计划（如有指征）
- [ ] 指定循证心理治疗
- [ ] 记录药物试验和反应
- [ ] 包含功能和恢复导向目标

### 对于慢性病管理计划：
- [ ] 优先处理所有活跃病症
- [ ] 确定药物协同作用
- [ ] 解决多重用药问题
- [ ] 明确护理协调计划
- [ ] 注意登记/人群健康整合
- [ ] 包含过渡管理

### 对于围手术期计划：
- [ ] 术前风险评估（RCRI、ASA、Caprini）
- [ ] 记录医学优化
- [ ] 包含 ERAS 元素（如适用）
- [ ] 定义术后里程碑
- [ ] 指定出院标准
- [ ] 包含 VTE 预防计划

### 对于疼痛管理计划：
- [ ] 综合疼痛评估（位置、质量、强度、影响）
- [ ] 分类疼痛类型（伤害性、神经病理性、中枢性）
- [ ] 多模式镇痛方法
- [ ] 阿片类药物风险评估（如考虑使用阿片类药物）
- [ ] 强调功能目标（不仅仅是疼痛分数）
- [ ] 包含心理筛查和干预
- [ ] 遵循 CDC 阿片类药物指南（如处方）

## 评分和解释

**总项目数**：约 150 个（因计划类型而异）

### 评分：
- 统计已勾选的项目数量
- 计算百分比：（已勾选 / 总数）× 100

### 解释：
- **95-100%**：优秀 - 计划符合最高质量标准
- **85-94%**：良好 - 计划质量高，有小差距
- **70-84%**：可接受 - 计划足够，但有需要改进的方面
- **<70%**：需要改进 - 质量或合规性存在显著差距

### 关键项目（必须有）：
以下项目是关键项目，必须存在：
- ✓ 患者标识符和去标识化通知
- ✓ 主要诊断及 ICD-10 代码
- ✓ 至少 3 个 SMART 目标
- ✓ 带有理由的干预措施
- ✓ 监测计划
- ✓ 随访计划
- ✓ 患者教育
- ✓ 安全/风险缓解
- ✓ 紧急程序
- ✓ 提供者签名

如果缺少任何关键项目，在纠正之前不应最终确定计划。

## 使用说明

1. **系统地**审查每个部分
2. **勾选**符合标准的方框
3. **记录**需要纠正的缺陷
4. **计算**分数以评估整体质量
5. **在最终确定之前**解决差距
6. **记录**审查结果，包括审查者姓名和日期
