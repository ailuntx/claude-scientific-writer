# 治疗计划的目标设定框架

## 概述

有效的治疗目标是成功患者护理的基石。本参考资料提供了关于创建SMART目标、以患者为中心的结果选择以及跨所有医学专业的治疗计划共享决策流程的综合指导。

## SMART目标框架

### 定义

**SMART**是一个目标标准的助记符，确保目标清晰明确且可实现：

- **S**pecific（具体的）
- **M**easurable（可衡量的）
- **A**chievable（可实现的）
- **R**elevant（相关的）
- **T**ime-bound（有时限的）

### 1. 具体性

目标必须清晰、明确且无歧义。

**具体性组成要素**：

- **什么**：将要完成的具体内容
- **谁**：谁负责（患者、医护人员或两者）
- **在哪里**：相关情境或环境
- **哪个**：涉及的健康/功能的具体方面

**示例**：

| 较差的（模糊）| 较好的（具体）|
|--------------|-----------------|
| "感觉更好" | "根据PHQ-9评分减轻抑郁症状" |
| "改善糖尿病" | "将HbA1c从目前的8.5%降低到7%以下" |
| "变得更强壮" | "将右侧股四头肌力量从3/5提高到徒手肌力测试的4/5" |
| "减轻体重" | "将体重减少10磅（从210磅减至200磅）" |
| "多运动" | "每周5天，每天步行30分钟" |

### 2. 可衡量性

目标必须包含可量化的指标或可观察的标准以跟踪进度。

**测量类型**：

- **量化指标**：数字、百分比、评分量表
  - 实验室值：HbA1c、低密度脂蛋白胆固醇、估算肾小球滤过率
  - 生命体征：血压、心率、体重
  - 量表：疼痛（0-10数字评分量表）、PHQ-9、GAD-7、FIM
  - 功能指标：步行距离、关节活动度、肌力等级
  
- **可观察的定性指标**：可观察和验证的行为
  - "患者演示正确的胰岛素注射技术"
  - "患者可独立使用助行器行走150英尺"
  - "患者可执行2步指令"

**示例**：

| 不可衡量| 可衡量|
|----------------|------------|
| "血压更好" | "收缩压<130 mmHg且舒张压<80 mmHg" |
| "疼痛减轻" | "疼痛强度从7/10降至数字评分量表上的≤4/10" |
| "活动能力改善" | "使用前轮助行器行走300英尺，监督水平" |
| "定期服药" | "根据配药率衡量，药物依从性>90%" |
| "睡眠更好" | "每晚睡眠7-8小时，夜间醒来<2次" |

### 3. 可实现性

目标必须基于患者的能力、资源和情况切实可行。

**需要考虑的因素**：

- **患者能力**：身体、认知、心理承受能力
- **病情严重程度**：晚期疾病改善潜力可能有限
- **治疗效果**：根据现有治疗可实际实现的程度
- **资源**：获得护理、药物、设备、支持的机会
- **可用时间**：实现目标所需的时间
- **动机**：患者改变的意愿和参与度

**设定可实现目标**：

- 从基线评估开始
- 了解预期治疗效果（例如：二甲双胍可将HbA1c降低1-1.5%）
- 为大改变设定渐进目标（先减5磅，再减10磅，而不是直接减50磅）
- 挑战患者但不要让其不堪重负
- 根据进展调整目标

**示例**：

| 不可实现| 可实现|
|----------------|------------|
| "1个月后能跑马拉松"（久坐不动的70岁心肌梗死后患者）| "3个月内连续步行1英里" |
| "6周内HbA1c从12%降至<6%" | "3个月内HbA1c从12%降至<9%，6个月内<7%" |
| "术后第3天膝关节活动度达到0-140°"（全膝关节置换术后）| "第2周膝关节活动度达到0-90°，第6周达到0-110°" |
| "治愈慢性疼痛" | "将疼痛从7/10降至4/10并改善30%功能" |

### 4. 相关性

目标必须与患者的价值观、优先事项和总体治疗目标一致。

**相关性标准**：

- **以患者为中心**：对患者重要，反映其优先事项
- **临床有意义**：实现目标可改善健康或生活质量
- **与诊断一致**：目标针对所治疗的病症
- **时机适当**：当前治疗阶段的合适目标
- **整合性**：与其他治疗目标相配合

**评估相关性**：

- 询问患者："对您来说最重要的是什么？""您想能够做什么？"
- 确保目标解决对患者重要的功能限制
- 将临床指标与患者有意义的结果联系起来（例如："HbA1c<7%可降低视力丧失风险"）
- 避免与患者不产生共鸣的医护人员驱动的目标

**示例**：

| 较不相关| 更相关|
|---------------|---------------|
| "减少用药数量"（当药物控制症状良好时）| "简化用药方案以提高依从性"（如果因复杂而漏服）|
| "完美的血糖"（患者优先考虑的是精力）| "通过更好地控制血糖来改善精力水平" |
| "步行5英里"（患者只是想独立购物）| "无需帮助即可在杂货店中行走" |

### 5. 有时限性

目标必须有具体的完成截止日期或时间范围。

**时间范围考虑**：

- **短期目标**：数天至3个月
- **中期目标**：3-6个月
- **长期目标**：6-12个月或更长时间（针对慢性疾病）
- **重新评估间隔**：在既定间隔检查进度

**应包含的时间要素**：

- 目标日期或时间范围
- 进度检查的 checkpoint 日期
- 行动频率（例如："运动30分钟，每周5次"）

**示例**：

| 无时限| 有时限|
|----------------|------------|
| "最终减轻体重" | "在6个月内减轻15磅（约每周1-2磅）" |
| "参加物理治疗" | "在8周内完成12次物理治疗课程，每周1-2次" |
| "准备就绪后返回工作" | "术后12周内重返修改后的轻体力工作" |
| "改善抑郁症状" | "开始使用SSRI和CBT后8周内，将PHQ-9评分从18降至<10" |

## 创建SMART目标：分步流程

### 步骤1：评估基线

- 识别当前状态：症状、实验室值、功能水平
- 使用标准化评估工具（如果可用）
- 记录量化基线

### 步骤2：确定预期结果

- 需要改善什么？
- 让患者参与："您希望有什么不同？"
- 考虑临床需求和患者优先事项

### 步骤3：使其具体

- 确定确切的结果
- 消除模糊语言
- 包含所有相关细节

### 步骤4：添加衡量标准

- 如何跟踪进度？
- 什么指标或可观察的行为？
- 基线 → 目标值

### 步骤5：现实检查（可实现性？）

- 考虑到患者的病情、资源、治疗效果，这是否可能？
- 可能需要调整期望
- 如需要，设定渐进目标

### 步骤6：确保相关性

- 患者是否关心这个目标？
- 它是否有临床意义？
- 它是否与总体治疗计划一致？

### 步骤7：设定时间表

- 何时实现目标？
- 何时检查进度？
- 如需要，分为短期和长期目标

### 步骤8：记录和沟通

- 以清晰的SMART格式书写目标
- 与患者和护理团队分享
- 确保患者理解

## 目标层级和水平

### ICF框架（国际功能、残疾和健康分类）

适用于康复和功能目标：

1. **损伤水平目标**：身体结构/功能
   - 示例："将肩关节屈曲活动度从90°提高到140°"
   
2. **活动水平目标**：任务表现
   - 示例："独立穿脱上半身衣物"
   
3. **参与水平目标**：生活角色参与
   - 示例："重返教师工作"

### 医学结果水平

1. **生物/临床目标**：实验室值、生命体征、疾病标志物
   - 示例："HbA1c<7%，血压<130/80，低密度脂蛋白<70 mg/dL"
   
2. **症状目标**：患者报告的症状
   - 示例："疼痛≤4/10，日常活动无呼吸困难"
   
3. **功能目标**：患者能做什么
   - 示例："步行1英里，爬2层楼梯"
   
4. **生活质量目标**：总体幸福感
   - 示例："重返爱好，改善睡眠质量"

## 以患者为中心的结果指标（PCOMs）

### 定义

超越传统临床指标的对患者最重要的结果。

### 常见的PCOMs

**患者报告的结果指标（PROMs）**：

- SF-36或SF-12（与健康相关的生活质量）
- PROMIS（患者报告的结果测量信息系统）
- 疾病特异性生活质量量表（例如：堪萨斯城心肌病问卷用于心力衰竭）

**功能结果**：

- 日常生活活动（ADLs）：洗澡、穿衣、如厕、转移、进食、排便控制
- 工具性日常生活活动（IADLs）：购物、烹饪、家务管理、财务管理、交通
- 职业/教育功能
- 社会功能和关系
- 娱乐和休闲参与

**患者优先事项**：

- 对个别患者最重要的事项
- 可能与临床医生的优先事项不同
- 示例："和孙辈玩耍"、"参加女儿的婚礼"、"避免入住养老院"

### 将PCOMs整合到目标中

**方法**：

1. 在评估早期询问患者优先事项
2. 将临床目标与患者有意义的结果联系起来
3. 至少包含一些直接解决患者优先事项的目标
4. 如可能，在记录目标时使用患者的语言

**整合示例**：

- **临床目标**："在3个月内将HbA1c从8.5%降至<7%"
- **相关联的以患者为中心的目标**："改善精力水平，以便和孙辈玩耍而不感到疲劳"
- 两个目标都被记录，跟踪两者的进度

## 目标设定中的共享决策

### 什么是共享决策（SDM）？

临床医生和患者共同参与的协作流程：

- 讨论治疗选择
- 权衡风险和收益
- 考虑患者的价值观和偏好
- 共同做出决定

### 治疗目标设定中的SDM

**步骤**：

1. **选择意识**：承认多个可能的目标/方法
   - "我们可以专注于积极的HbA1c降低或最小化低血糖风险。您更看重哪个？"

2. **选项呈现**：呈现带利弊的目标选项
   - "选项A：强化血压控制（<120/80）可降低中风风险，但需要更多药物。选项B：标准控制（<140/90）更容易，但中风风险略高。"

3. **价值观澄清**：了解患者优先事项
   - "您对服用多种药物有什么看法？""避免注射对您有多重要？"

4. **偏好整合**：将偏好纳入目标
   - 如果患者优先避免用药 → "尽可能通过生活方式改变和一种药物控制血压"

5. **决定**：共同商定目标
   - "听起来您想尝试3个月的强化生活方式改变，然后再加用另一种药物。让我们为此制定计划。"

6. **记录**：记录共享决策过程
   - "目标通过共享决策建立。患者表达了对......的偏好。"

### 决策辅助工具

促进SDM的工具：

- 比较各方法的选项网格
- 数值化风险/收益数据
- 患者故事/证词
- 视觉辅助（图片、图表）
- "对您来说重要的是什么"工作表

## 不同人群的特殊考虑

### 老年人

- 功能独立性通常优先于疾病特异性指标
- 平衡积极治疗与治疗负担
- 考虑预期寿命和受益时间
- 预防跌倒、减少多重用药可能是关键目标
- 生活质量优先于生命长度

### 儿童

- 发育阶段适当的目标
- 以家庭为中心（让父母/照护者参与）
- 生长发育里程碑
- 学校/社交功能
- 过渡计划（从儿科到成人护理）

### 慢性疾病

- 长期可持续目标
- 雄心与现实期望平衡
- 并发症预防
- 维持生活质量
- 适应和接受与改善并重

### 姑息治疗/临终

- 舒适和症状管理是首要
- 关注有价值活动的功能目标
- 心理社会和精神需求
- 照护者支持
- 尊严和自主权

### 复杂的多发病

- 优先考虑最具影响力的目标
- 跨病症协调目标（当治疗重叠时更好）
- 避免相互冲突的治疗
- 最小化治疗负担
- 对多种病症的现实期望

## 常见目标设定陷阱

### 陷阱1：以医护人员为中心的目标

**问题**：目标反映医护人员认为重要的，而非患者优先事项  
**解决方案**：在就诊早期询问患者希望实现什么，使用他们的语言

### 陷阱2：目标过多

**问题**：用10+个目标让患者不堪重负  
**解决方案**：优先考虑3-5个关键目标，通过成功逐步建立

### 陷阱3：全或无思维

**问题**：目标是"治愈"或"完美"  
**解决方案**：渐进目标，重视有意义的改善

### 陷阱4：忽视障碍

**问题**：设定目标时未评估可行性（资源、支持、可及性）  
**解决方案**：在评估期间识别障碍，解决问题或调整目标

### 陷阱5：静态目标

**问题**：设定目标后从不重新审视  
**解决方案**：定期重新评估，根据患者进展或情况变化进行修改

### 陷阱6：纯临床指标

**问题**：所有目标都是实验室值，没有功能或生活质量目标  
**解决方案**：平衡临床标志物与功能、症状和生活质量结果

### 陷阱7：没有患者认同

**问题**：患者不相信目标可实现或重要  
**解决方案**：共享决策动机性访谈以探索矛盾心理

## 按病症分类的SMART目标示例

### 糖尿病

**短期目标**："通过启动二甲双胍1000mg每日两次并将碳水化合物摄入量降至每餐45-60克，在3个月内将HbA1c从8.5%降至<7.5%。"

**长期目标**："保持HbA1c<7%超过6个月，预防微血管并发症，并通过每天步行30分钟提高精力水平以参与日常活动。"

### 心力衰竭

**短期目标**："通过优化呋塞米剂量和限制钠摄入<2000mg/天，在2周内实现正常容量（无水肿，体重稳定在2磅内）。"

**长期目标**："保持NYHA II级功能状态，预防心力衰竭住院，并在3个月内无呼吸困难地步行1/4英里。"

### 抑郁

**短期目标**："通过开始使用艾司西酞普兰10mg每日一次并参加每周CBT治疗，在8周内将PHQ-9评分从18降至<10。"

**长期目标**："实现抑郁缓解（PHQ-9<5），全职重返工作，并在4个月内每周2-3次重新参与社交活动。"

### 卒中后康复

**短期目标**："通过每周5次PT/OT，在4周内将右上肢力量从2/5提高至3+/5，并将功能独立度量表（FIM）评分从85提高至100。"

**长期目标**："实现所有日常生活活动独立，在平坦地面上使用手杖行走500英尺，并在3个月内回家（而非护理机构）。"

### 慢性腰背痛

**短期目标**："使用多模式镇痛（SNRI、NSAID、PT），在6周内将疼痛强度从7/10降至4/10，并将步行耐受时间从10分钟提高至30分钟。"

**长期目标**："在3个月内重返修改后的轻体力工作，参与爱好（钓鱼、适应性园艺），并将日常疼痛干扰减少50%（简明疼痛量表）。"

### 高血压

**短期目标**："通过启动赖诺普利10mg每日一次并将钠摄入量降至<2300mg/天，在4周内将血压从152/94降至<140/90 mmHg。"

**长期目标**："实现并保持血压<130/80 mmHg，将10年ASCVD风险从15%降至<10%，并预防心血管事件。"

## 工具和资源

### 目标设定模板

- SMART目标工作表（填空格式）
- 患者目标跟踪表
- 动机性访谈"改变对话"以引出目标

### 评估工具

- 目标实现量表（GAS）：个性化结果指标
- 加拿大职业表现measure（COPM）：患者确定的功能目标
- 患者报告的结果测量信息系统（PROMIS）

### 患者教育

- "设定健康目标"讲义
- 目标可视化练习
- 跟踪应用程序和日志
