# 基于循证的干预指南

## 概述

本参考指南提供关于在药物、非药物和治疗程序治疗模式中选择、实施和记录循证干预的综合指导。这些指南支持使用当前最佳实践和临床建议制定治疗方案。

## 证据等级

### 证据级别

**一级证据：最高质量**
- 随机对照试验（RCT）的系统评价和荟萃分析
- 大型多中心RCT

**二级证据：高质量**
- 单个RCT
- 观察性研究的系统评价

**三级证据：中等质量**
- 队列研究
- 病例对照研究
- 设计良好的观察性研究

**四级证据：较低质量**
- 病例系列
- 病例报告
- 专家意见

**推荐强度**：
- **A级**：强推荐，高质量证据
- **B级**：中等推荐，中等质量证据
- **C级**：弱推荐，低质量证据
- **D级**：反对推荐（存在危害或无益的证据）

## 药物干预

### 药物选择原则

#### 1. 循证处方
- 使用对适应症有已证实疗效的药物
- 遵循临床实践指南
- 考虑比较有效性数据
- 疗效相当时，优先选择安全性更好的药物

#### 2. 患者特定因素
- 合并症和禁忌症
- 器官功能（肾脏、肝脏）
- 药物过敏和不耐受
- 合并用药（药物相互作用）
- 年龄、妊娠状态
- 遗传因素（如有可用的药物基因组学）
- 费用和保险覆盖

#### 3. 药物安全
- 起始低剂量，缓慢加量（尤其是老年人、多种合并症）
- 根据反应和耐受性滴定至目标剂量
- 监测不良反应
- 避免潜在不适当药物（老年人Beers标准）
- 尽可能减少多重用药

### 按适应症的常见药物类别

#### 高血压

**一线药物**（根据JNC-8、ACC/AHA指南）：
- **ACE抑制剂**（赖诺普利、依那普利）：如有糖尿病、慢性肾病或心力衰竭为首选
- **ARB**（氯沙坦、缬沙坦）：如不能耐受ACE的替代选择
- **钙通道阻滞剂**（氨氯地平）：对老年人、黑人患者尤其有效
- **噻嗪类利尿剂**（氯噻酮、氢氯噻嗪）：具有成本效益，心血管结局良好

**给药策略**：
- 从低剂量单药开始
- 加用第二种药物前先滴定至最大耐受剂量
- 通常需要联合治疗（2-3种药物）
- 监测血压反应，每2-4周调整一次

#### 2型糖尿病

**一线治疗**（ADA诊疗标准）：
- **二甲双胍**：除非有禁忌症（eGFR <30），适用于所有患者的一线治疗
  - 起始500-850mg每日一次或两次，滴定至每日总量2000mg

**二线治疗**（根据合并症个体化）：
- **SGLT2抑制剂**（恩格列净、达格列净）：如有心力衰竭或慢性肾病（强心肾获益）
- **GLP-1受体激动剂**（司美格鲁肽、度拉糖肽）：如有动脉粥样硬化性心血管疾病（ASCVD）或高风险、需要减重
- **DPP-4抑制剂**（西格列汀）：如希望低低血糖风险
- **磺脲类**（格列吡嗪）：具有成本效益但有低血糖风险
- **胰岛素**：如HbA1c非常高（>10%）或高血糖症状

#### 抑郁

**一线SSRI**（APA指南）：
- 舍曲林、艾司西酞普兰、氟西汀、西酞普兰、帕罗西汀
- 低剂量起始（如舍曲林50mg，艾司西酞普兰10mg）
- 如部分反应，2-4周后滴定
- 足量试验：治疗剂量下6-8周
- 缓解后继续治疗6-12个月（复发则延长）

**二线治疗**：
- **SNRI**（文拉法辛、度洛西汀）：尤其如有慢性疼痛合并症
- **安非他酮**：如有性功能障碍问题、戒烟
- **米氮平**：如有失眠/食欲刺激需要

**增强治疗**（如部分反应）：
- 来自不同类别的第二种抗抑郁药
- 非典型抗精神病药（阿立哌唑、喹硫平）- FDA批准的增强治疗
- 锂、甲状腺激素（三碘甲状腺原氨酸）

#### 慢性疼痛

**多模式镇痛**（WHO疼痛阶梯，CDC阿片类药物指南）：

**非阿片类镇痛药**：
- **对乙酰氨基酚**：每日3-4g分次服用，肝功能正常时安全
- **NSAIDs**：布洛芬、萘普生、美洛昔康 - 短期或长期使用需保护胃肠道
  - 监测：肾功能、血压、胃肠道出血风险

**神经病理性疼痛的辅助镇痛药**：
- **加巴喷丁**：300mg滴定至每日1800-3600mg，分3次服用
- **普瑞巴林**：75mg每日两次滴定至150-300mg每日两次（生物利用度优于加巴喷丁）
- **SNRI**（度洛西汀）：每日60mg，用于糖尿病神经病变、慢性肌肉骨骼疼痛
- **TCAs**（阿米替林、去甲替林）：低剂量（10-75mg睡前）- 因副作用作为二线

**局部用药**：
- 5%利多卡因贴膏、双氯芬酸凝胶、辣椒素软膏
- 局部作用，系统吸收极少

**阿片类药物**（CDC指南 - 谨慎使用）：
- 仅在非阿片类多模式治疗不足后使用
- 最低有效剂量，初期首选短效制剂
- 如可能避免>90 MME/天
- 尿液药物监测、处方药监测项目监测、纳洛酮联合处方
- 频繁重新评估，如未达到功能目标则减量

#### 射血分数降低的心力衰竭（HFrEF）

**指南指导的医学治疗（GDMT）** - "基础四联"：

1. **ACE抑制剂或ARB或ARNI**
   - ACE：赖诺普利每日20-40mg，依那普利每日10-20mg每日两次
   - ARNI（沙库比曲/缬沙坦）：24/26mg每日两次 → 97/103mg每日两次（优于ACE/ARB）
   - 监测：血压、肾功能、钾

2. **β受体阻滞剂**
   - 卡维地洛3.125-6.25mg每日两次 → 25mg每日两次（目标）
   - 琥珀酸美托洛尔12.5-25mg每日 → 200mg每日
   - 比索洛尔1.25mg → 10mg每日
   - 缓慢滴定，监测心率、血压

3. **盐皮质激素受体拮抗剂（MRA）**
   - 螺内酯12.5-25mg每日（可达50mg）
   - 依普利酮25mg每日 → 50mg每日
   - 监测：钾、肾功能（高钾血症风险）

4. **SGLT2抑制剂**
   - 达格列净10mg每日或恩格列净10mg每日
   - 减少心力衰竭住院和死亡率
   - 对糖尿病和慢性肾病也有益处

**其他治疗**：
- 祥利尿剂（呋塞米）用于容量管理（无死亡率获益）
- 肼屈嗪-硝酸异山梨酯（如果是非裔美国人或不能耐受ACE/ARB）
- 伊伐布雷定（如果射血分数≤35%，最大β受体阻滞剂下心率>70次/分）
- 地高辛（症状获益，减少住院）

### 药物记录最佳实践

**治疗方案中应包括**：
- 通用名（品牌名可选）
- 剂量、给药途径、频次
- 适应症/理由
- 如适用，滴定计划
- 预期起效时间
- 需要监测的关键副作用
- 药物相互作用
- 何时调整或停药

**示例**："赖诺普利10mg口服每日一次 - ACE抑制剂用于高血压和糖尿病肾脏保护。如血压未达标且耐受良好，2-4周后滴定至20mg（监测咳嗽、高钾血症）。目标血压<130/80mmHg。"

## 非药物干预

### 生活方式调整

#### 饮食与营养

**地中海饮食**（证据：多项RCT，PREDIMED试验）：
- **适应症**：心血管疾病预防、糖尿病管理
- **组成**：
  - 高摄入：水果、蔬菜、全谷物、豆类、坚果、橄榄油
  - 适量：鱼、禽
  - 低：红肉、甜食
- **证据**：减少30%的心血管事件，改善血糖控制
- **实施**：转介营养师进行医学营养治疗

**DASH饮食**（停止高血压的饮食方法）：
- **适应症**：高血压
- **组成**：高水果/蔬菜、低脂乳制品、减少钠摄入（<2300mg，最好<1500mg）
- **证据**：降低收缩压8-14mmHg
- **实施**：DASH饮食计划教育、钠摄入追踪

**碳水化合物计数**（用于糖尿病）：
- 稳定的碳水化合物摄入：每餐45-60g
- 可调整胰岛素剂量
- 防止血糖波动
- 营养师教授碳水化合物计数技能

**体重管理**：
- 热量赤字：每日500-750千卡，每周减重1-2磅
- 行为改变策略：自我监测、刺激控制、目标设定
- 结构化项目（Weight Watchers、MOVE!等）比自我指导更有效
- 药物治疗（GLP-1激动剂、奥利司他）或减肥手术用于BMI≥30-35且有合并症

#### 体力活动和运动

**有氧运动**：
- **建议**：每周150分钟中等强度或75分钟高强度
- **中等强度**：快走、骑自行车、游泳 - 能说话但不能唱歌
- **高强度**：跑步、快骑自行车 - 只能说几个词就要停下来
- **获益**：心血管健康、血糖控制、体重管理、情绪
- **实施**：从10分钟开始，逐渐增加

**阻力训练**：
- **建议**：每周2-3次，覆盖所有主要肌群
- **获益**：肌肉力量、骨密度、代谢率、血糖控制
- **实施**：自重训练、阻力带、自由重量、器械

**平衡和柔韧性**：
- 对老年人跌倒预防重要
- 瑜伽、太极拳
- 拉伸训练

**运动处方**：
- FITT原则：**F**requency（频率）、**I**ntensity（强度）、**T**ime（时间）、**T**ype（类型）
- 根据 fitness水平、合并症、目标个体化
- 如有需要心脏 clearance（使用ACSM或ACC/AHA指南）

**示例**："有氧运动：中等强度步行30分钟，每周5天（目标心率50-70%最大心率）。阻力训练：上下肢运动每周2次，每次2组10-12次重复。"

#### 戒烟

**证据**：对COPD、心血管疾病、癌症预防最强的干预措施

**5A方法**：
1. **询问**：筛查所有患者的烟草使用
2. **建议**：敦促所有烟草使用者戒烟
3. **评估**：戒烟意愿
4. **协助**：帮助戒烟（咨询+药物）
5. **安排**：随访联系

**药物治疗**（戒烟成功率翻倍）：
- **尼古丁替代疗法**：贴片、口香糖、锭剂 - 非处方、安全
- **伐尼克兰**：最有效（Chantix），戒烟日期前1周开始
- **安非他酮**：替代方案，也可治疗抑郁症
- **联合**：NRT + 伐尼克兰/安非他酮更有效

**咨询**：
- 戒烟热线：1-800-QUIT-NOW
- 个人或团体咨询
- 认知行为技术

**实施**：30天内设定戒烟日期，处方药物治疗 + 转介咨询，戒烟日期后1周内随访。

#### 睡眠卫生

**适应症**：失眠、睡眠质量差

**组成部分**：
- 固定的睡眠-觉醒时间（相同就寝/起床时间）
- 卧室：黑暗、安静、凉爽（60-67°F/15-19°C）
- 避免：下午2点后咖啡因、酒精、睡前大量进食
- 屏幕时间：睡前1小时停止
- 放松程序：阅读、洗澡、放松
- 只用床睡觉（不是看电视、工作）
- 如20分钟后仍无法入睡，起床做安静活动

**证据**：对慢性失眠有效，常与失眠认知行为治疗（CBT-I）联合使用

#### 压力管理

**技术**：
- **正念冥想**：每日10-20分钟，减少焦虑、抑郁
- **渐进性肌肉放松**：系统性地绷紧和放松肌肉群
- **深呼吸**：膈式呼吸，4-7-8技术
- **瑜伽、太极拳**：身心练习
- **认知重构**：挑战导致压力的想法

**证据**：减少压力激素，改善情绪、疼痛感知

### 行为干预

#### 认知行为治疗（CBT）

**适应症**：抑郁、焦虑、失眠、慢性疼痛、物质使用

**核心组成部分**：
- 心理教育
- 认知重构（识别和挑战扭曲思维）
- 行为激活（增加奖励性活动）
- 解决问题技能
- 复发预防

**证据**：对轻中度抑郁与抗抑郁药效果相当，焦虑、失眠的一线治疗

**实施**：与训练有素的治疗师进行12-16次每周50分钟的治疗，期间布置家庭作业

**变体**：
- **CBT-I**（失眠）：睡眠限制、刺激控制、认知睡眠治疗
- **CBT-CP**（慢性疼痛）：疼痛教育、活动节奏、疼痛灾难化认知重构

#### 动机访谈（MI）

**适应症**：对行为改变（饮食、运动、物质使用、药物依从性）的矛盾心理

**原则**：
- 表达同理心
- 发展差异（当前行为与目标/价值观之间）
- 避免阻抗（不要争论）
- 支持自我效能

**技术**：
- 开放式问题
- 肯定
- 反思性倾听
- 总结
- 引出"改变谈话"

**证据**：在多个领域有效启动行为改变

### 患者教育和自我管理

**组成部分**：
- 疾病教育（病理生理、自然病程、治疗）
- 自我监测技能（血糖、血压、体重、症状）
- 药物管理（目的、剂量、副作用）
- 症状识别和行动计划
- 生活方式改变技能
- 解决问题
- 何时就医

**证据**：自我管理教育改善糖尿病、哮喘、心力衰竭、慢性疼痛的结局

**实施方式**：
- 临床医生或教育者的个体化教育
- 结构化项目（糖尿病的DSMES、心脏病的心脏康复）
- 小组课程
- 书面材料、视频、应用程序

## 程序和治疗性干预

### 康复治疗

#### 物理治疗

**适应症**：肌肉骨骼损伤、术后康复、平衡/步态障碍、慢性疼痛

**干预**：
- 治疗性运动：力量、拉伸、耐力
- 手法治疗：软组织动员、关节动员
- 步态和平衡训练
- 物理因子：热、冰、超声、电刺激、TENS
- 功能训练：日常生活活动再训练、力学

**证据**：对特定疾病有强证据（如膝关节骨关节炎的物理治疗减少疼痛、改善功能，效果与NSAIDs相当）

**处方**：频次（如每周2-3次）、持续时间（如4-8周）、具体干预/目标

#### 作业治疗

**适应症**：日常生活活动受限、上肢功能障碍、认知-知觉障碍、工作相关损伤

**干预**：
- 日常生活活动/工具性日常生活活动训练
- 辅助设备和环境改造
- 上肢力量和协调
- 节能技术
- 认知康复
- 工作强化/适应性训练

**证据**：改善中风后、损伤后、慢性疾病后的独立性

#### 言语-语言病理学

**适应症**：吞咽困难、失语症、构音障碍、认知-沟通障碍

**干预**：
- 吞咽治疗和饮食调整
- 语言治疗（失语症）
- 构音治疗
- 认知-语言治疗
- 增强和替代沟通（AAC）

### 介入性疼痛治疗

#### 硬膜外类固醇注射（ESI）

**适应症**：椎间盘突出或椎管狭窄引起的放射性疼痛

**证据**：短期疼痛缓解（3-6周至3个月）的中等质量证据，长期获益可变

**入路**：透视引导下，经椎间孔、椎板间或骶管

**频次**：每年最多3-4次注射

**风险**：感染、出血、神经损伤（罕见）、硬膜穿刺

#### 射频消融（RFA）

**适应症**：关节突关节介导的疼痛（阳性诊断性内侧支阻滞后）

**证据**：对腰椎关节突关节疼痛有良好证据，6-12个月缓解

**程序**：对支配关节突关节的内侧支神经进行热消融

**可重复**：疼痛复发时可重复

#### 脊髓刺激（SCS）

**适应症**：难治性慢性神经病理性疼痛（椎间盘术后综合征、CRPS、糖尿病神经病变）

**证据**：50-60%达到≥50%疼痛缓解，改善功能

**程序**：试验性电极置入（5-7天），如成功 → 永久植入

**技术**：传统、高频、爆刺激、背根神经节（DRG）

### 外科干预

**何时转介手术**：
- 保守治疗失败（充分试验 - 通常至少6-12周）
- 进行性神经功能缺损
- 马尾综合征（急诊）
- 严重影响生活质量的严重功能限制
- 结构病变适合手术矫正
- 讨论风险/收益后患者偏好

**共同决策**：讨论手术与非手术治疗、风险、获益、预期结局、恢复

## 整合和补充疗法

### 针灸

**证据**：
- **中等证据**：慢性腰痛、膝关节骨关节炎疼痛、紧张性头痛、偏头痛
- **机制**：不清楚（内啡肽释放、门控理论、安慰剂）

**实施**：由持证针灸师进行8-12次治疗

### 按摩治疗

**证据**：对慢性腰痛、焦虑、癌症相关症状有轻微获益

**类型**：瑞典式、深层组织、肌筋膜松解

**实施**：每周1-2次，每次30-60分钟

### 瑜伽

**证据**：改善腰痛、平衡、柔韧性，减少压力和焦虑

**类型**：哈他（温和）、流瑜伽（流动）、艾扬格（正位聚焦）

**实施**：小组课程或居家练习，每周2-3次

### 正念减压（MBSR）

**证据**：减少压力、焦虑、抑郁、慢性疼痛

**项目**：8周结构化项目，每周2.5小时课程，每日居家练习

**组成部分**：冥想、身体扫描、正念运动（瑜伽）

### 脊骨神经科治疗

**证据**：急性和慢性腰痛、颈痛有效

**技术**：脊柱操作、动员、软组织治疗

**安全性**：通常安全，如有骨质疏松、脊柱不稳定则避免高速操作

## 干预选择和记录

### 治疗方法

1. **诊断特异性**：遵循疾病的循证指南
2. **严重程度适当**：轻度 → 保守；重度 → 积极
3. **逐步强化**：从一线开始，如反应不足则加用或更换
4. **多模式**：结合互补干预（药物+非药物）
5. **个体化**：根据患者因素（合并症、偏好、资源）调整

### 记录模板

对于每项干预，记录：
- **干预**：具体名称/类型
- **适应症**：为什么为此患者选择此干预
- **证据**：基于指南的、支持使用的RCT数据
- **剂量/频次/持续时间**：具体参数
- **预期获益**：应该改善什么、改善多少、何时
- **监测**：如何评估反应
- **风险/副作用**：需要监测的关键问题
- **考虑的替代方案**：还考虑了哪些、为何未选择
