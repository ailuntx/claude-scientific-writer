## 解释/备注

[如适用，提供临床解释]

**贫血示例：**
```
正细胞性贫血伴白细胞升高。鉴别诊断包括慢性病贫血、近期失血或溶血。
建议进行网织红细胞计数、铁谱检查和外周血涂片进一步评估。建议结合临床。
```

**电解质异常示例：**
```
检出低钾血症（K+ 3.2 mEq/L）。常见原因包括利尿剂使用、胃肠道丢失或摄入不足。
建议补钾并复查。中度氮质血症，提示急性肾损伤或慢性肾脏病。建议结合患者病史
及既往结果进行临床判断。
```
