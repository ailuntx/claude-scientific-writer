# 临床报告的HIPAA合规检查清单

## 18个HIPAA标识符 - 去标识化检查清单

确认以下所有标识符均已移除或更改：

- [ ] **1. 姓名** - 患者姓名、家庭成员、医疗服务提供者（除非必要并获得同意）

- [ ] **2. 小于州的地理细分**
  - 无街道地址
  - 无城市（除非人口超过20,000，且如果人口超过20,000可保留部分邮编）
  - 无县
  - 邮编前三位仅在地理单位人口超过20,000时才可接受
  - 移除邮编的所有其他部分

- [ ] **3. 日期**（年份除外）
  - 无确切出生日期（仅接受年份；89岁以上者的出生年份必须汇总）
  - 无入院日期
  - 无出院日期
  - 无服务日期
  - 无死亡日期
  - 使用相对时间段（例如"3个月前"）或仅使用年份

- [ ] **4. 电话号码**
  - 无任何类型的电话号码
  - 包括患者、家庭、服务提供者联系方式

- [ ] **5.传真号码**
  - 无传真号码

- [ ] **6. 电子邮件地址**
  - 无患者或相关人员的电子邮件地址

- [ ] **7. 社会安全号码**
  - 无SSN或部分SSN

- [ ] **8. 病历号码**
  - 无MRN、医院ID或诊所编号
  - 如需要，使用编码的研究ID或病例编号

- [ ] **9. 健康计划受益人号码**
  - 无保险ID号码
  - 无保单号码

- [ ] **10. 账户号码**
  - 无账单账户号码
  - 无财务账户信息

- [ ] **11. 证书/执照号码**
  - 无驾照号码
  - 无专业执照号码（除非用于作者资质）

- [ ] **12. 车辆标识符和序列号**
  - 无车牌号码
  - 无VIN号码

- [ ] **13. 设备标识符和序列号**
  - 无心脏起搏器序列号
  - 无植入设备序列号
  - 可接受通用设备描述（例如"植入式心律转复除颤器"）

- [ ] **14. 网络URL**
  - 无个人网站
  - 无识别个人的URL

- [ ] **15. IP地址**
  - 无IP地址

- [ ] **16. 生物识别标识符**
  - 无指纹
  - 无声纹
  - 无视网膜扫描
  - 无其他生物识别数据

- [ ] **17. 完整面部照片和类似图像**
  - 未经同意无完整面部照片
  - 如需展示则裁剪或模糊面部
  - 移除识别特征（珠宝、纹身、胎记，如果不是临床相关的）
  - 眼部黑条不足以遮盖
  - 确保无反光或背景可识别

- [ ] **18. 任何其他唯一识别特征或代码**
  - 无可识别个人的唯一特征
  - 无可识别的罕见疾病组合
  - 考虑剩余数据点的组合是否可识别个人

## 其他去标识化注意事项

### 年龄和日期

- [ ] 年龄≤89岁的患者：可接受确切年龄或年龄范围
- [ ] 年龄>89岁的患者：必须汇总为"90岁或以上"或">89岁"
- [ ] 日期：仅使用年份或使用相对时间段
  - 示例：使用"就诊前3个月"而非"2023年1月15日"
  - 示例：使用"2023年入院"而非"2023年3月10日入院"

### 地理信息

- [ ] 州或国家可接受
  - 移除具体城市（除非人口超过20,000且无其他识别信息）
- [ ] 移除医院/诊所名称
- [ ] 使用通用描述："中西部的一家社区医院"或"一家三级医疗中心"

### 罕见疾病和组合

- [ ] 考虑非常罕见的疾病本身是否可识别患者
- [ ] 考虑以下组合是否可识别患者：
  - 年龄 + 诊断 + 地理区域 + 时间段
- [ ] 可能需要对某些独特细节保持模糊
- [ ] 在提供临床信息和保护隐私之间取得平衡

### 图像和图表

- [ ] 移除图像标题/元数据中的所有患者标识符
- [ ] 剥离DICOM数据
- [ ] 移除图像中的日期
- [ ] 移除病历号码
- [ ] 面部裁剪、模糊或遮挡
- [ ] 移除或遮挡识别标记：
  - 纹身
  - 珠宝
  - 胎记或独特疤痕（如果不是临床相关的）
- [ ] 标尺和注释不含识别信息
- [ ] 背景环境去标识化（房间号、姓名牌等）

### 声音和视频

- [ ] 无患者声音的录音（除非获得同意）
- [ ] 无显示可识别特征的视频（除非获得同意）
- [ ] 如有必要使用视频，必须遮挡面部

## 知情同意检查清单（用于病例报告/发表）

### 同意要求

- [ ] 在提交发表前获得知情同意
- [ ] 直接从患者处获得同意（如患者有行为能力）
- [ ] 如患者已故或无行为能力，从法定代表人或直系亲属处获得同意
- [ ] 对于儿科病例，获得父母/监护人同意

### 同意书要素

知情同意书必须包括：

- [ ] 发表目的（教育、医学知识）
- [ ] 将发表的内容（病例详情、图像、结果）
- [ ] 期刊或发表场所（如已知）
- [ ] 开放获取 vs 订阅（公众可获取性）
- [ ] 解释去标识化努力
- [ ] 承认再识别的可能性
- [ ] 不影响临床护理
- [ ] 撤回同意的权利（时间限制）
- [ ] 联系方式以便咨询
- [ ] 患者签名和日期
- [ ] 见证人签名（如需要）

### 同意文档

- [ ] 签署的同意书存档
- [ ] 提供副本给患者
- [ ] 同意书可供编辑审查
- [ ] 手稿中声明确认已获得同意

**手稿示例声明：**
"已获得患者书面知情同意，同意发表本病例报告及任何附随图像。本书面同意的副本可供本刊编辑委员会主任根据要求审查。"

## 安全港 vs 专家确定

### 安全港方法

- [ ] 移除全部18个标识符
- [ ] 无实际知识表明剩余信息可识别个人
- 最简单直接的方法
- [ ] 推荐用于大多数临床报告

### 专家确定方法

- [ ] 合格统计学家/专家确定再识别风险非常小
- [ ] 记录方法论
- [ ] 指定分析方法
- [ ] 记录结论
- [ ] 可能允许保留某些数据元素
- [ ] 需要统计学专业知识

**使用的方法：** [ ] 安全港 [ ] 专家确定

---

## 最小必要标准

### 使用和披露

- [ ] 仅使用目的所需的最小PHI
- [ ] 明确界定披露目的
- [ ] 仅限于相关信息
- [ ] 考虑将去标识化数据或有限数据集作为替代方案

### 最小必要例外

最小必要不适用于：
- 治疗目的（提供者可能需要完整信息）
- 患者授权的披露
- 法律要求的披露
- 向HHS披露用于合规调查

---

## PHI使用/披露授权

### 需要授权的情况

需要授权的情况：
- [ ] 研究（除非获得IRB豁免）
- [ ] 营销目的
- [ ] PHI销售
- [ ] 心理治疗记录
- [ ] 超出治疗、支付、运营（TPO）的用途

### 授权要素

如需要授权，必须包括：

- [ ] 将使用/披露的PHI的具体描述
- [ ] 被授权进行披露的人
- [ ] 将接收信息的人
- [ ] 披露目的
- - [ ] 到期日期或事件
- [ ] 撤销权利及方式
- [ ] 拒绝签字的权利
- [ ] 接收方再披露的可能性
- [ ] 患者签名和日期

---

## 有限数据集

### 有限数据集选项

有限数据集移除18个标识符中的16个，但可能保留：
- [ ] 日期（入院、出院、服务、出生、死亡）
- [ ] 地理信息（城市、州、邮编）

### 有限数据集要求

- [ ] 需要数据使用协议（DUA）
- [ ] DUA指定允许的用途
- [ ] 仅用于研究、公共卫生或医疗运营
- [ ] 接收方同意不再识别
- [ ] 接收方同意保护数据

---

## 安全防护检查清单

### 行政管理保障

- [ ] 实施安全管理制度
- [ ] 员工安全措施
- [ ] 访问管理（基于角色）
- [ ] 员工安全培训
- [ ] 事件响应程序

### 物理安全保障

- [ ] 设施访问控制
- [ ] 工作站使用策略
- [ ] 工作站安全措施
- [ ] 设备和介质控制
- [ ] 安全处置程序

### 技术安全保障

- [ ] 访问控制（唯一用户ID、密码）
- [ ] 审计控制和日志记录
- [ ] 完整性控制
- [ ] 传输安全（加密）
- [ ] 不活动后自动注销

---

## 违规通知检查清单

### 如发生未经授权的披露

- [ ] 确定是否发生违规（未经授权的访问/使用/披露）
- [ ] 评估对个人的伤害风险
- [ ] 如违规影响<500人：
  - 在60天内通知个人
  - 每年向HHS报告
- [ ] 如违规影响≥500人：
  - 在60天内通知个人
  - 在60天内通知HHS
  - 如在一个州/司法管辖区影响≥500人，通知媒体
- [ ] 记录违规和响应
- [ ] 实施纠正措施

### 违规通知内容

通知必须包括：
- [ ] 违规描述
- [ ] 涉及的信息类型
- [ ] 个人应采取的步骤
- [ ] 组织正在做什么
- [ ] 联系方式以便咨询

---

## 研究特定合规

### IRB/隐私委员会注意事项

- [ ] 获得IRB批准（如为研究）
- [ ] 获得HIPAA授权或获得豁免
- [ ] 记录豁免理由：
  - 对隐私的风险最小
  - 没有豁免则无法实际进行研究
  - 没有PHI则无法实际进行研究
  - 保护标识符的计划
  - 适当时候销毁标识符的计划

### 临床试验报告

- [ ] 受试者仅用ID编号识别
- [ ] 监管提交中无姓名
- [ ] 仅在监管机构要求时使用首字母
- [ ] 日期仅限于年份或相对时间
- [ ] 方案包括隐私保护

---

## 特殊人群

### 儿科病例

- [ ] 获得父母/监护人同意
- [ ] 获得儿童同意（如年龄适当）
- [ ] 对可识别照片格外谨慎
- [ ] 移除学校信息

### 已故患者

- [ ] HIPAA保护适用于死亡后50年
- [ ] 发表需直系亲属同意
- [ ] 尸检信息去标识化

### 心理健康和药物滥用

- [ ] 根据42 CFR Part 2提供额外保护
- [ ] 披露需明确同意
- [ ] 未经同意不得再披露
