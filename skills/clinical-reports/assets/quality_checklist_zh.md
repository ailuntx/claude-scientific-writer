# 临床报告质量保证核查清单

## 通用质量标准

### 完整性
- [ ] 所有必需章节已包含
- [ ] 无空白字段或缺失信息
- [ ] 已包含所有相关临床信息
- [ ] 事件时间线清晰完整
- [ ] 已记录所有诊断检查和结果
- [ ] 已记录所有治疗和干预措施
- [ ] 已指定随访计划

### 准确性
- [ ] 患者人口学信息正确
- [ ] 日期和时间准确
- [ ] 实验室数值包含正确的单位和参考范围
- [ ] 药物名称、剂量和频率正确
- [ ] 诊断编码正确（ICD-10）
- [ ] 操作编码正确（CPT，如适用）
- [ ] 无矛盾信息

### 清晰性
- [ ] 语言清晰、专业
- [ ] 医学术语使用得当
- [ ] 缩略语已定义或仅使用标准缩略语
- [ ] 组织逻辑清晰、流程顺畅
- [ ] 字迹清晰可读（如手写）
- [ ] 无模糊陈述
- [ ] 临床推理解释清晰

### 及时性
- [ ] 在实际发生或检查后不久记录
- [ ] 出院小结在24-48小时内完成
- [ ] 危急结果立即传达
- [ ] 符合监管报告截止日期

## 病例报告质量核查清单

### CARE指南合规性
- [ ] 标题包含"病例报告"
- [ ] 提供关键词（2-5个MeSH术语）
- [ ] 结构化摘要包含所有要素
- [ ] 引言说明创新性
- [ ] 患者信息已提供并去标识
- [ ] 已记录临床发现
- [ ] 提供时间线（表格或图表）
- [ ] 详细说明诊断评估
- [ ] 描述治疗干预措施
- [ ] 报告随访和结局
- [ ] 讨论部分包含文献综述
- [ ] 包含患者观点（如可能）
- [ ] 包含知情同意声明

### 隐私和伦理
- [ ] 已获得并记录知情同意
- [ ] 已移除所有18项HIPAA标识符
- [ ] 日期已移除或近似
- [ ] 年龄报告适当（>89岁聚合）
- [ ] 地理信息限制在州级
- [ ] 图像已去标识或获得同意
- [ ] 如适用，已获得IRB批准

### 科学质量
- [ ] 创新性明确确立
- [ ] 文献检索全面
- [ ] 已考虑鉴别诊断
- [ ] 已解决因果关系问题
- [ ] 已承认局限性
- [ ] 学习点可操作
- [ ] 参考文献当前且相关

## 文档完整性核查清单

### 法律和伦理标准

- [ ] 必要时引用患者原话
- [ ] 客观记录不合规情况
- [ ] 不更改原始记录
- [ ] 使用附录进行更正
- [ ] 附录清晰标记
- [ ] 所有条目已签字和注明日期

### 计费和编码支持
- [ ] 已记录医疗必要性
- [ ] 已记录护理复杂性
- [ ] 已记录时间（如基于时间的计费）
- [ ] ICD-10代码适当且具体

- [ ] 修饰语适当
- [ ] 文档支持所计费服务级别

## 最终审核核查清单

在完成任何临床报告之前：

- [ ] 通读全文
- [ ] 检查完整性
- [ ] 验证所有数据准确性
- [ ] 确保逻辑流畅
- [ ] 检查拼写和语法
- [ ] 验证患者标识符正确（或如已去标识则已移除）
- [ ] 确保符合监管要求
- [ ] 确认所有必要签字
- [ ] 验证分发正确
- [ ] 适当存档副本
