# 临床试验报告标准

## ICH-E3：临床研究报告的结构和内容

国际人用药品注册技术协调会（ICH）E3指南定义了用于监管申报的临床研究报告（CSRs）的结构和内容。

### CSR概述

**目的：**
- 提供研究设计、执行和结果的全面描述
- 支持监管决策
- 记录安全性和有效性的证据

**受众：**
- 监管机构（FDA、EMA、PMDA等）
- 医学审评员
- 统计审评员
- 临床药理学审评员

**篇幅：**通常50-300页（正文），附有大量附录

### ICH-E3 CSR的主要章节

#### 第1章：封面页

**必需要素：**
- 全研究标题
- 方案编号和版本
- 申办者名称和地址
- 化合物/药物名称和代
- 研究阶段
- 适应症
- 报告日期和版本号
- 报告作者
- 保密声明

#### 第2章：摘要

**篇幅：**5-15页

**内容：**
- 整个CSR的简要总结
- 必须可作为独立文件理解
- 涵盖所有主要章节

**标准摘要要素：**
1. 研究标识符和标题
2. 研究目标
3. 方法学：
   - 研究设计
   - 患者数量和描述
   - 诊断和主要入组标准
   - 研究治疗
   - 治疗持续时间
   - 评估标准
   - 统计学方法
4. 结果：
   - 入组、完成和退出患者数量
   - 有效性结果
   - 安全性结果
5. 结论

#### 第3章：伦理学

**3.1 独立伦理委员会/机构审查委员会**
- 所有IRB的名称和位置
- 初始批准日期
- 方案修正案批准日期
- 持续审查文件

**3.2 研究的伦理行为**
- 符合GCP和赫尔辛基宣言的声明
- 方案依从性
- 知情同意过程

**3.3 患者信息和同意**
- 知情同意程序描述
- 使用的同意书版本
- 重新同意程序（如适用）

#### 第4章：研究者和研究行政结构

**4.1 研究者**
- 各中心主要研究者名单
- 中心地址和入组数
- 协调研究者（如适用）

**4.2 行政结构**
- 申办者人员及其职责
- CRO参与（如适用）
- 监查程序
- 数据管理机构
- 统计分析机构

**4.3 研究监查和质量保证**
- 监查程序和频率
- 源文件核实
- 质量控制程序
- 进行的审计

#### 第5章：引言

**5.1 背景**
- 研究疾病或状况
- 当前治疗格局
- 未满足的医疗需求

**5.2 试验用药品**
- 药理学作用机制
- 非临床发现
- 既往临床经验
- 已知安全性特征

**5.3 非试验用治疗**
- 对照药物或安慰剂
- 允许/禁止的合并用药

#### 第6章：研究目标

**6.1 主要目标**
- 主要研究问题
- 清晰表述且具体
- 示例："评估药物X与安慰剂相比在2型糖尿病患者中治疗24周时降低HbA1c的有效性"

**6.2 次要目标**
- 额外研究问题
- 支持性有效性终点
- 安全性目标
- 探索性目标

**6.3 终点**
- 主要终点定义和测量
- 次要终点
- 安全性终点
- 药代动力学终点（如适用）
- 生物标志物终点（如适用）

#### 第7章：试验计划

**7.1 整体研究设计和计划**
- 研究设计类型（平行、交叉、析因等）
- 随机化和设盲
- 研究阶段或时期
- 治疗和随访持续时间
- 给药方案
- 研究流程图（患者流程图）

**7.2 样本量**
- 目标入组量
- 样本量依据
- 把握度计算假设：
  - 预期效应量
  - 变异度估计
  - I类错误（alpha）
  - 把握度（1-beta）
  - 脱落率假设

**7.3 统计学方法**
- 分析人群（ITT、PP、安全性）
- 缺失数据处理
- 期中分析（如计划）
- 多重性校正
- 亚组分析
- 敏感性分析

**7.4 方案变更**
- 方案修正案及其理由
- 对研究执行和分析的影响

#### 第8章：研究患者

**8.1 入组和排除标准**
- 关键入组标准
- 关键排除标准
- 标准依据

**8.2 人口统计学和基线特征**
- 年龄、性别、种族/民族
- 疾病严重程度或分期
- 既往治疗
- 关键终点基线值
- 治疗组间可比性

**8.3 患者处置**
- 筛查数量
- 随机化数量
- 完成研究数量
- 退出数量（按原因）
- 失访数量
- CONSORT流程图

**8.4 方案偏离**
- 重大方案偏离
- 轻微方案偏离
- 对有效性和安全性分析的影响
- 采取的纠正措施

**8.5 人口统计学和其他基线特征**
- 详细人口统计学表格
- 基线疾病特征
- 分层因素
- 病史
- 既往/合并用药

#### 第9章：有效性评估

**9.1 分析数据集**
- 意向性治疗（ITT）人群
- 符合方案（PP）人群
- 改良ITT
- 其他分析集
- 人群定义依据

**9.2 人口统计学和基线特征**
- 各分析人群的人口统计学
- 基线可比性

**9.3 治疗依从性测量**
- 药物清点
- 片剂计数或日记依从性
- 血浆药物水平（如测量）
- 计划剂量接收百分比

**9.4 有效性结果**

**9.4.1 主要终点**
- 主要终点结果
- 统计分析
- 效应量和置信区间
- P值
- 亚组分析

**9.4.2 次要终点**
- 各次要终点结果
- 统计分析
- 检验顺序（如适用）

**9.4.3 其他有效性终点**
- 探索性终点
- 事后分析
- 应答者分析

**9.5 脱落和缺失数据**
- 缺失数据模式
- 脱落原因
- 缺失数据敏感性分析

#### 第10章：安全性评估

**10.1 暴露程度**
- 暴露持续时间
- 剂量强度
- 剂量延迟或减少
- 因不良事件导致的治疗停止

**10.2 不良事件**

**10.2.1 不良事件概述**
- 汇总表（任何AE、治疗相关、严重、导致停药）
- 有AE的患者百分比
- 治疗组间比较

**10.2.2 常见不良事件**
- 发生率≥5%或≥10%的AE
- 按频率排序
- 首选术语和系统器官分类（MedDRA）

**10.2.3 严重不良事件**
- SAE定义
- SAE汇总表
- 各SAE的个别叙述
- 因果关系评估
- 结果

**10.2.4 导致停药的不良事件**
- 导致研究药物停药的AE
- 频率和类型
- 与研究药物的关系

**10.2.5 死亡**
- 研究期间和随访期间的所有死亡
- 各死亡的详细叙述
- 与研究药物的关系
- 尸检结果（如有）

**10.3 临床实验室评估**
- 实验室异常
- 转向表（正常转异常、异常转正常）
- 基线变化均值
- 符合方案定义标准的实验室值
- 肝毒性监测（如适用）

**10.4 生命体征和体格检查**
- 生命体征（BP、HR、温度、呼吸频率）
- 基线变化均值
- 有临床意义的改变
- 体格检查发现

**10.5 心电图评估**
- QTc间期变化
- 其他心电图异常
- 有临床意义的心电图发现

**10.6 特殊安全性评估**
- 免疫原性（如生物制品）
- 妊娠结局（如适用）
- 滥用潜力（如适用）
- 停药或反跳效应
- 依赖潜力

#### 第11章：讨论和总体结论

**11.1 有效性讨论**
- 有效性结果解读
- 发现的临床意义
- 与既往研究的一致性
- 局限性

**11.2 安全性讨论**
- 安全性特征概述
- 值得注意的安全性发现
- 与已知安全性特征比较
- 风险-获益评估

**11.3 获益-风险评估**
- 总体获益-风险结论
- 获益-风险有利/不利的亚组
- 给药或患者选择的建议

**11.4 临床意义**
- 在治疗中的地位
- 目标患者人群
- 与现有治疗的比较

#### 第12章：表格、图形和图表

有效性和安全性数据的全套表格和图形。

**常见表格：**
- 人口统计学和基线特征
- 患者处置
- 暴露程度
- 有效性结果（主要和次要终点）
- 不良事件汇总
- 常见不良事件
- 严重不良事件
- 死亡
- 实验室异常
- 生命体征

**常见图形：**
- 研究设计示意图
- 患者处置流程图（CONSORT）
- Kaplan-Meier曲线（生存、事件发生时间）
- 森林图（亚组分析）
- 随时间变化均值图

#### 第13章：参考文献

- CSR中引用的出版物
- 相关文献
- 监管指南
- 既往研究报告

#### 第14章：附录

**必需附录：**
- 研究方案和修正案
- 样版病例报告表
- 含IRB信息的研究者名单
- 患者信息和知情同意书
- 接受研究药物的患者名单
- 随机化方案
- 审计证书（如适用）
- 统计学方法文件
- 基于研究出版的出版物

**可选附录：**
- 个别患者数据列表
- SAE叙述
- 实验室正常值和换算因子
- 研究者签名

### 统计分析计划（SAP）

**SAP组成部分：**
- 分析人群
- 缺失数据处理
- 将使用的统计检验
- 多重性校正
- 期中分析计划
- 亚组分析
- 敏感性分析
- 安全性分析

**SAP时间：**
- 在数据库锁定前完成
- 附有理赔的修正案

## CONSORT（报告试验的综合标准）

CONSORT指南促进随机对照试验的透明和完整报告。

### CONSORT 2010清单

#### 标题和摘要
- **1a. 标题**：在标题中识别为随机试验
- **1b. 摘要**：涵盖试验设计、方法、结果、结论的结构化摘要

#### 引言
- **2a. 背景**：科学背景和理由解释
- **2b. 目标**：具体目标或假设

#### 方法-参与者
- **3a. 资格**：参与者的资格标准
- **3b. 场所**：数据收集的场所和位置

#### 方法-干预措施
- **4a. 干预措施**：各组干预措施的详细信息
- **4b. 详细信息**：足以允许复制的详细信息

#### 方法-终点
- **5. 终点**：明确定义的主要和次要终点指标
- **6a. 样本量**：如何确定样本量
- **6b. 期中分析**：如适用，期中分析的说明

#### 方法-随机化
- **7a. 序列生成**：随机序列生成方法
- **7b. 分配隐藏**：分配隐藏机制
- **8a. 实施**：谁生成分配、纳入和分配参与者
- **8b. 设盲**：参与者、护理提供者、结果评估者是否设盲

#### 方法-统计
- **9. 统计学方法**：主要和次要终点的统计学方法
- **10. 额外分析**：亚组或校正分析

#### 结果-参与者流程
- **11a. 纳入**：筛查、随机化、分配的数量
- **11b. 损失和排除**：每组随机化后的损失和排除
- **12. 纳入**：定义纳入和随访期的日期
- **13a. 基线**：基线人口统计学和临床特征
- **13b. 基线可比性**：每组分析的患者数量

#### 结果-终点和估计
- **14a. 终点**：主要和次要终点，每组的结果
- **14b. 二分类终点**：二分类终点，效应量和置信区间
- **15. 辅助分析**：进行的其他分析的结果

#### 结果-危害
- **16. 危害**：每组的所有重要危害或非预期效应

#### 讨论
- **17a. 局限性**：试验局限性，解决偏倚、不精确性
- **17b. 可推广性**：试验发现的可推广性（外部效度）
- **18. 解读**：与结果一致的解读，平衡获益和危害
- **19. 注册**：试验注册号和名称
- **20. 方案**：可获取完整试验方案的地点
- **21. 资金**：资金来源、资金方作用

### CONSORT流程图

显示患者通过试验流程的标准格式：
```
评估资格 (n=)
    ↓
随机化 (n=)
    ├─ 分配至干预组 (n=)
    │   ├─ 接受干预 (n=)
    │   └─ 未接受干预 (n=)
    │       说明原因
    ├─ 分配至对照组 (n=)
    │   ├─ 接受对照 (n=)
    │   └─ 未接受对照 (n=)
    │       说明原因
    ↓
失访 (n=)
    说明原因
停止干预 (n=)
    说明原因
    ↓
分析 (n=)
排除分析 (n=)
    说明原因
```

## 严重不良事件（SAE）报告

### 严重不良事件定义

不良事件或疑似不良反应如果符合以下条件被视为严重：
- 导致死亡
- 危及生命
- 需要住院治疗或延长现有住院时间
- 导致持久或显著残疾/丧失能力
- 是先天性异常/出生缺陷
- 需要干预以防止永久性损伤或损害（器械相关）
- 其他医学重要事件（根据医学判断）

### SAE报告组成部分

**1. 管理信息**
- 报告类型（初始、随访、最终）
- 报告编号
- 报告日期
- 报告者信息
- 申办者信息
- 研究标识符（方案编号、NCT编号）

**2. 患者信息（去标识化）**
- 受试者ID或随机化编号
- 姓名首字母（如允许）
- 年龄或出生日期（仅年份）
- 性别
- 种族/民族
- 体重
- 身高

**3. 研究信息**
- 研究阶段（I、II、III、IV）
- 研究设计（随机、开放标签等）
- 治疗组或随机化
- 首次研究药物日期
- 最后一次研究药物日期

**4. 事件信息**
- 报告术语（原始术语）
- MedDRA首选术语
- 系统器官分类
- 开始日期
- 开始时间（如相关）
- 解决日期（或进行中）
- 持续时间

**5. 严重性标准**
- 死亡：是/否
- 危及生命：是/否
- 需要住院：是/否
- 住院延长：是/否
- 丧失能力/无能力：是/否
- 先天性异常：是/否
- 医学重要：是/否

**6. 严重程度**
- 轻度：可注意到但不影响日常活动
- 中度：影响日常活动但可管理
- 重度：阻止正常日常活动，需要干预

注意：严重程度≠严重性

**7. 结果**
- 恢复/解决
- 正在恢复/解决
- 未恢复/未解决
- 恢复/解决但有后遗症
- 死亡
- 未知

**8. 因果关系评估**
- 与研究药物的关系：
  - 无关
  - 可能无关
  - 可能相关
  - 可能相关
  - 确定相关
- 与研究程序的关系
- 与基础疾病的关系
- 与合并用药的关系
- 判断理由

**9. 预期性**
- 预期（根据研究者手册或方案）
- 意外（不在IB中或比记录的更严重）

**10. 对研究药物采取的措施**
- 无变化
- 剂量减少
- 剂量增加
- 药物暂停（暂时停用）
- 药物停用
- 不适用（事件发生在停药后）

**11. 事件的治疗/干预**
- 给与的药物
- 进行的程序
- 住院详情
- ICU入院
- 手术干预

**12. 事件叙述**
- 事件的详细描述
- 事件时间表
- 临床过程
- 相关病史
- 合并用药
- 诊断检查结果
- 治疗和反应
- 结果和当前状态

**示例叙述：**
```
一位58岁男性（受试者ID：12345）入组研究XYZ-301，这是一项药物X与安慰剂对比治疗心力衰竭的3期随机试验。在治疗第42天（2024年2月15日），患者因突然发作严重胸痛、出汗和呼吸困难到急诊就诊。心电图显示V2-V4导联ST段抬高。肌钙蛋白I升高至12.5 ng/mL（正常<0.04）。患者被诊断为急性ST段抬高型心肌梗死，急诊心脏造影显示左前降支95%闭塞。成功进行经皮冠状动脉介入治疗并置入药物洗脱支架。患者入住心脏重症监护室。研究药物在第42天永久停用。患者恢复并于第47天（2024年2月20日）病情稳定出院。研究者评估此事件与研究药物可能无关，因为患者有显著的潜在冠状动脉疾病风险因素，包括糖尿病、高血压和吸烟史。
```

### 监管报告时间线

**FDA IND安全性报告（21 CFR 312.32）：**
- **致死或危及生命的意外SAE**：初步报告7个日历日，完整报告15天
- **其他严重的意外事件**：15个日历日
- **年度安全性报告**：在IND周年纪念后60天内

**EMA expedited报告：**
- **致死或危及生命的意外事件**：初步7天，完整报告额外8天
- **其他意外的严重事件**：15天

**IRB报告：**
- 根据机构政策
- 严重的意外事件通常5-10天
- 一些机构要求24-48小时内报告

### MedDRA编码

**MedDRA（监管活动医学词典）：**
- 监管沟通的标准化医学术语
- 分层结构：
  - SOC（系统器官分类）- 最高级别
  - HLGT（高级组术语）
  - HLT（高级术语）
  - PT（首选术语）- 用于编码AE
  - LLT（最低级别术语）- 原始术语

**示例：**
- 原始术语："bad headache"
- LLT：头痛
- PT：头痛
- HLT：头痛NEC
- HLGT：神经系统疾病NEC
- SOC：神经系统疾病

### 因果关系评估方法

**WHO-UMC因果关系类别：**
- **确定**：事件无法用其他因素解释
- **可能/很可能**：事件更可能与药物相关而非其他因素
- **可能**：事件可能与药物相关，但不能排除其他因素
- **不太可能**：事件可能用其他因素解释
- **有条件/未分类**：需要更多数据
- **无法评估/无法分类**：信息不足

**Naranjo算法（用于ADR）：**
基于10个问题的评分系统：
- 评分≥9：肯定
- 评分5-8：很可能
- 评分1-4：可能
- 评分≤0：不太可能

## 数据安全监测委员会（DSMB）

**目的：**
- 安全性数据的独立审查
- 监测获益-风险
- 关于研究继续的建议

**DSMB章程要素：**
- 成员资格和资质
- 角色和职责
- 会议频率
- 审查的数据
- 决策标准
- 沟通程序
- 保密

**DSMB报告：**
- 开放报告（所有各方可见）
- 封闭报告（仅DSMB和申办者）
- 建议：继续、修改或终止研究
