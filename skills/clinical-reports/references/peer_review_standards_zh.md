# 临床手稿同行评审标准

## 临床手稿同行评审概述

### 目的

同行评审确保临床手稿在发表前符合科学严谨性、伦理行为和清晰沟通的标准。

**目标：**
- 评估科学有效性及方法学
- 评价临床意义
- 验证伦理合规性
- 确保完整性和清晰性
- 提高手稿质量

**同行评审类型：**
- 单盲（审稿人知作者，作者不知审稿人）
- 双盲（双方匿名）
- 开放同行评审（双方均已知）
- 出版后同行评审

### 审稿人责任

**仅在以下情况下接受评审：**
- 在该领域具有资质
- 无利益冲突
- 有充足时间（通常2-3周）
- 能提供建设性、无偏见的评价

**保密：**
- 不得分享手稿内容
- 不得利用信息谋取个人利益
- 未经编辑许可不得让他人参与

**及时评审：**
- 在要求时间内完成
- 如无法完成应及时通知编辑

## 病例报告审核标准

### CARE指南合规性

**核对手稿包括：**
- [ ] 标题标明为病例报告
- [ ] 关键词（2-5个）
- [ ] 结构化或非结构化摘要
- [ ] 引言说明病例的创新性
- [ ] 患者信息（去标识化）
- [ ] 临床发现
- [ ] 事件时间线
- [ ] 诊断评估
- [ ] 治疗干预
- [ ] 随访和结局
- [ ] 讨论及文献综述
- [ ] 患者观点（如适用）
- [ ]知情同意声明

### 创新性和意义

**评估：**
- 此病例是否真正创新或为医学知识增添新内容？
- 报告此病例的价值是什么？
- 该疾病是否罕见或表现是否不寻常？
- 它是否挑战现有知识？
- 是否有可推广的临床经验？

**警示信号：**
- 常见疾病的常见表现
- 单一病例无独特特征
- 从单一病例过度概括
- 缺乏显示创新性的文献综述

### 隐私和伦理考量

**验证：**
- 获得并记录知情同意
- 充分去标识化（去除18个HIPAA标识符）
- 无明确同意不得使用可识别图像
- 删除或近似日期
- 地理信息限于州/国家
- 年龄适当（具体年龄或范围）
- 去除机构标识符

**伦理问题：**
- 缺少同意文件
- 存在可识别信息
- 缺乏回顾性图表审查的IRB批准（如适用）
- 弱势群体缺乏额外保护

### 临床质量

**诊断过程：**
- 针对症状进行适当检查
- 考虑鉴别诊断
- 逻辑递进至最终诊断
- 充分记录发现

**治疗：**
- 基于证据的干预
- 治疗选择的理由
- 考虑替代治疗
- 适当的监测和随访

**结局：**
- 清晰描述临床结局
- 随访时间适当
- 记录并发症
- 如有长期结局

### 文献综述

**评估：**
- 充分检索现有文献
- 识别并讨论类似病例
- 综述对该疾病的当前理解
- 适当定位病例
- 参考当前且相关
- 与既往病例比较

### 写作质量

**结构：**
- 逻辑流程和组织
- 遵循CARE指南结构
- 清晰、简洁的写作
- 适当的医学术语

**清晰性：**
- 解释医学术语
- 时间线清晰易读
- 事件按时间顺序逻辑清晰
- 结论有病例细节支持

## 临床试验手稿审核标准

### 研究设计和方法学

**评估：**
- 针对研究问题采用适当的研究设计
- 清晰的目标和假设
- 明确定义的主要和次要终点
- 适当的样本量及效能计算
- 随机化和盲法适当
- 对照组适当

**警示信号：**
- 事后更改终点
- 效能不足的研究声称等效性
- 不适当的统计方法
- 可行时未使用盲法
- 纳入时存在选择偏倚

### CONSORT合规性

**验证：**
- 标题标明为随机试验
- 结构化摘要
- 提供试验注册号
- 方案可获取
- 包含CONSORT流程图
- 基线特征表
- 报告所有结局（不仅是显著的）
- 报告不良事件
- 披露资金来源
- 声明利益冲突

### 随机化和分配

**评估：**
- 适当的序列生成方法
- 分配隐藏适当
- 基线特征均衡
- 指定分层因素
- 记录交叉和方案偏离

### 受试者流程

**验证：**
- 报告筛选人数
- 提供排除原因
- 明确随机化人数
- 记录脱落及原因
- 最小化并解释失访
- 指定ITT和符合方案分析
- CONSORT图完整准确

### 结局指标

**主要结局：**
- 明确定义为先验
- 具有临床意义
- 适合研究问题
- 可靠且有效地测量
- 统计分析适当

**次要结局：**
- 在方案中预先指定
- 适当分析
- 如需要则进行多重比较校正
- 如效能不足则不过度解读

**探索性结局：**
- 清楚标明为探索性或事后
- 不与主要结局同等权重
- 生成假说，非验证性

### 统计分析

**评估：**
- 揭盲前指定分析计划
- 适当的统计检验
- 验证假设（正态性等）
- 适当处理缺失数据
- 需要时进行多重性校正
- 提供置信区间
- 报告效应量

**常见问题：**
- p值黑客（选择性报告）
- 未校正的多重检验
- 不适当的亚组分析
- 在ITT和符合方案分析之间切换
- 缺失数据被忽略或处理不当

### 安全性报告

**验证：**
- 报告所有不良事件
- 详细描述严重不良事件
- 充分描述死亡
- 评估因果关系
- 报告实验室异常
- 记录因不良事件停药

### 临床意义

**评估：**
- 统计学意义与临床意义
- 效应量有临床意义
- 如适用计算需要治疗的人数（NNT）
- 获益-风险比有利
- 外推至临床实践
- 考虑成本效益

## 诊断研究审核标准

### STARD指南（诊断准确性研究报告标准）

**评估合规性：**
- 描述研究设计
- 参与者选择标准
- 抽样方法
- 数据收集程序
- 定义参考标准
- 详细描述待测试验
- 解决盲法问题
- 参与者流程清晰
- 提供2×2表格
- 估计诊断准确性

### 参考标准

**验证：**
- 使用适当的金标准
- 所有参与者使用相同参考标准
- 无论待测试验结果如何均执行参考标准
- 待测试验与参考标准之间的时间适当
- 待测试验和参考标准独立解释

### 试验性能

**必需指标：**
- 敏感度和特异度
- 阳性和阴性预测值（含患病率）
- 似然比
- ROC曲线和AUC（如为连续结局）
- 所有估计的95%置信区间

**考虑：**
- 验前和验后概率
- 除准确性外的临床效用
- 与现有试验比较
- 成本和可及性

### 谱系和验证偏倚

**评估：**
- 包含疾病严重程度谱系
- 避免谱系偏倚（仅严重病例）
- 避免验证偏倚（所有参与者均获得参考标准）
- 避免差异验证（不同参与者使用不同参考标准）

## 观察性研究审核标准

### STROBE指南（加强流行病学观察性研究报告）

**对于队列、病例对照或横断面研究，验证：**
- 标题和摘要标明研究设计
- 背景和理由清晰
- 指定目标
- 方法中包含研究设计
- 描述研究场景
- 描述参与者
- 明确定义变量
- 详细描述数据来源和测量
- 解决偏倚
- 论证研究规模
- 描述统计方法
- 报告结果含效应量和CI

### 暴露和结局评估

**评估：**
- 明确定义暴露
- 明确定义结局
- 测量方法有效可靠
- 尽可能盲法评估者
- 各组测量一致
- 暴露和结局之间的时间关系适当

### 混杂和偏倚

**验证：**
- 识别潜在混杂因素
- 在分析中调整混杂因素
- 讨论残余混杂
- 解决选择偏倚
- 考虑信息偏倚
- 进行敏感性分析

### 因果关系

**考虑Bradford Hill标准：**
- 关联强度
- 跨研究一致性
- 特异性
- 时间性（暴露先于结局）
- 生物梯度（剂量-反应）
- 合理性
- 与现有知识一致性
- 实验证据
- 类比

**避免：**
- 无强力证据的观察性研究使用因果语言
- 混淆关联与因果

## 系统综述和Meta分析审核标准

### PRISMA指南

**验证：**
- 标题标明为系统综述/meta分析
- 结构化摘要
- 研究问题（PICO格式）
- 方案和注册（PROSPERO）
- 全面的检索策略
- 描述研究选择过程
- 数据提取过程
- 纳入研究的质量评估
- 适当的综合方法
- 结果含森林图
- 异质性评估
- 发表偏倚评估
- 证据确定性（GRADE）

### 检索策略

**评估：**
- 检索多个数据库
- 检索词全面
- 论证限制和筛选器
- 考虑灰色文献
- 手工检索参考文献
- 联系作者获取缺失数据
- 检索可重复

### 研究选择

**验证：**
- 预先指定纳入/排除标准
- 由≥2名审稿人独立筛选
- 适当解决分歧
- PRISMA流程图完整
- 列出排除的研究及原因

### 质量评估

**评估：**
- 使用适当的质量评估工具
  - 随机对照试验：Cochrane风险偏倚工具
  - 观察性研究：Newcastle-Ottawa量表
  - 诊断研究：QUADAS-2
- 独立进行质量评估
- 报告质量评估结果
- 将质量纳入综合

### 统计方法

**对于meta分析：**
- 论证固定效应vs随机效应模型
- 评估异质性（I²、Q统计量）
- 提供森林图
- 评估发表偏倚（漏斗图、Egger检验）
- 进行敏感性分析
- 预先指定亚组分析

### GRADE评估

**证据确定性：**
- 高：对效应估计非常有信心
- 中等：中等有信心
- 低：信心有限
- 极低：信心非常不足

**降低确定性的因素：**
- 偏倚风险
- 不一致性
- 间接性
- 不精确性
- 发表偏倚

## 手稿质量评估

### 结构和组织

**评估：**
- 从引言到讨论的逻辑流程
- 各部分组织适当
- 图表支持正文
- 补充材料适当

### 写作质量

**清晰性：**
- 清晰、简洁的语言
- 最小化并定义术语
- 首次使用时定义缩写
- 术语一致

**语法和风格：**
- 正确的语法和拼写
- 适当的动词时态（研究结果用过去式，既定事实用现在式）
- 适当使用主动语态
- 简洁但不牺牲清晰性

### 参考文献

**验证：**
- 足够的参考文献数量
- 包含当前文献
- 引用关键论文
- 参考文献格式正确
- 所有引文在参考文献列表中，反之亦然
- 不过度自引

### 表格和图表

**评估：**
- 适合数据类型
- 标签和图例清晰
- 图像高质量
- 可独立使用
- 不与正文重复
- 统计符号正确

## 评审中的伦理考量

### 利益冲突

**如有以下情况则披露并回避：**
- 与作者的个人关系
- 对结果的经济利益
- 竞争性研究
- 对主题的强烈偏见
- 机构冲突

### 公正和建设性评审

**提供：**
- 对优势和劣势的平衡评估
- 具体、可执行的建议
- 尊重的语气
- 客观评价
- 认可研究设计的局限性

**避免：**
- 人身攻击
- 轻视的语言
- 要求不合理的修改
- 期望完美研究
- 将个人偏好置于标准之上

### 保密

**保持：**
- 不分享手稿
- 未经许可不与同事讨论
- 不使用想法或数据
- 评审后销毁副本

## 建议类别

**接受：**
- 手稿符合发表标准
- 只需小幅编辑

**小修：**
- 可解决的小问题
- 不需要额外数据
- 通常一轮修改

**大修：**
- 需要实质性更改的重大问题
- 可能需要额外分析
- 可能需要额外数据或实验
- 通常需要再审

**拒绝：**
- 无法纠正的根本性缺陷
- 创新性或意义不足
- 不伦理行为
- 数据欺诈

**拒绝再投：**
- 研究有潜力但需要大量工作
- 重大更改后实质上是新投稿

## 撰写评审报告

### 结构

**摘要：**
- 简要概述（2-3句）
- 总体评估
- 主要优势（2-3点）
- 主要劣势（2-3点）
- 建议

**主要意见：**
- 编号
- 影响有效性、解释性或意义的重要问题
- 具体且可执行
- 按优先级排序

**次要意见：**
- 编号
- 编辑、格式或澄清问题
- 具体的行意见
- 表格/图表意见

### 语气和语言

**使用：**
- 专业、同行化的语气
- 用"作者指出..."而非"你指出..."
- 用"本研究表明..."而非"你的研究表明..."
- 建设性批评
- 改进建议

**避免：**
- 严厉或轻视的语言
- 人称代词
- 讽刺
- 模糊的批评
- 不合理的要求

### 具体且可执行的反馈

**好的示例：**
"样本量计算（第8页）未考虑预期脱落率。请修订以纳入预期脱落并解释这如何影响招募目标。"

**差的示例：**
"样本量不足。"

**好的示例：**
"图2如果用95%置信区间而非标准误表示误差线会更清晰。请修订并更新图例。"

**差的示例：**
"图2令人困惑。"
