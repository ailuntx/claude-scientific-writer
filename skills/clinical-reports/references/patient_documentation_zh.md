# 患者文档标准

## SOAP 病程记录

SOAP（主观、客观、评估、计划）是临床实践中病程记录的标准格式。

### 目的和使用

**何时使用SOAP记录：**
- 住院每日病程记录
- 门诊就诊记录
- 亚专科会诊
- 随访记录
- 记录治疗反应

**优点：**
- 结构标准化
- 临床思维清晰
- 促进沟通
- 支持计费和编码
- 法律文件

### SOAP 组成部分

#### S - 主观

**定义：** 患者报告的信息（症状、concerns、病史）

**包含要素：**
- 主诉或就诊原因
- 现病史（HPI）
- 与就诊相关的系统回顾（ROS）
- 患者对症状的描述
- 对既往治疗的反应
- 功能影响
- 患者的问题或疑问

**HPI要素（使用OPQRST评估疼痛/症状）：**
- **O**nset（起病）：何时开始？突然还是逐渐？
- **P**rovocation/Palliation（诱发/缓解）：什么会加重或减轻？
- **Q**uality（性质）：感觉如何？（锐痛、钝痛、灼烧感等）
- **R**egion/Radiation（部位/放射）：在哪里？是否放射？
- **S**everity（严重程度）：有多严重？（0-10分制）
- **T**iming（时间特点）：持续性还是间歇性？持续时间？频率？

**伴随症状：**
- 与主诉同时出现的其他症状
- 相关内容阴性（预期症状的缺失）

**治疗反应：**
- 服用的药物及效果
- 既往干预措施及结果
- 治疗方案的依从性

**主观部分示例：**
```
S: 患者报告持续咳嗽5天，咳黄色痰。伴发热至101.5°F，昨晚在家测量。否认呼吸困难、胸痛或咯血。2天前在急诊诊所开始使用阿奇霉素，自觉改善不明显。报告食欲下降但能保持水分补充。否认近期旅行或传染病接触史。
```

#### O - 客观

**定义：** 可测量、可观察的临床数据

**包含要素：**

**生命体征：**
- 体温（°F或°C）
- 血压（mmHg）
- 心率（bpm）
- 呼吸频率（次/分）
- 氧饱和度（%）
- 身高和体重（计算BMI）
- 如适用疼痛评分

**一般外观：**
- 整体外观（良好、病重、痛苦）
- 年龄适宜性
- 营养状态
- 卫生状况
- 情感和行为

**按系统的体格检查：**
- 按从头到脚或按系统组织
- 与主诉相关的发现
- 包含相关内容阳性和阴性

**标准检查系统：**
1. **HEENT**（头、眼、耳、鼻、喉）
2. **Neck**（甲状腺、淋巴结、颈静脉怒张、颈动脉）
3. **Cardiovascular**（心音、杂音、周围脉搏、水肿）
4. **Pulmonary/Respiratory**（呼吸音、呼吸功）
5. **Abdomen**（肠音、触痛、肝脾肿大、肿块）
6. **Extremities**（水肿、脉搏、关节活动度、畸形）
7. **Neurological**（精神状态、颅神经、运动、感觉、反射、步态）
8. **Skin**（皮疹、伤口、创面）
9. **Psychiatric**（情感、情感、思维过程/内容）

**实验室和影像学结果：**
- 相关检查结果
- 异常值包含参考范围
- 注明检查时间与就诊的关系

**客观部分示例：**
```
O: 生命体征：体温100.8°F，血压128/82，心率92，呼吸18，室内空气下氧饱和度96%
一般：清醒，轻度呼吸困难，看起来稍有病态
HEENT：咽部无红斑或渗出，双侧鼓膜清晰
Neck：无淋巴结肿大，无颈静脉怒张
Cardiovascular：心律规整，无杂音
Pulmonary：右肺下叶呼吸音减低，叩诊浊音，存在羊鸣音。无哮鸣音。
Abdomen：软，无触痛，无肝脾肿大
Extremities：无水肿，双侧脉搏2+
Neurological：清醒，定向力x3，无局灶性缺损

今日化验：
白细胞14.2 x10³/μL (H) [参考 4.5-11.0]
血红蛋白13.5 g/dL
血小板245 x10³/μL
C反应蛋白8.5 mg/dL (H) [参考 <0.5]

胸片：右肺下叶实变，符合肺炎
```

#### A - 评估

**定义：** 临床印象、诊断和患者状态评价

**包含要素：**
- 主要诊断或问题
- 次要诊断或问题
- 不确定时的鉴别诊断
- 严重程度评估
- 治疗目标进展
- 并发症或新问题

**格式：**
- 问题列表（编号）
- 每个问题附简要评估
- 适当情况下包含ICD-10代码用于计费

**评估部分示例：**
```
A: 
1. 社区获得性肺炎，右肺下叶（J18.1）
   - 中度严重（CURB-65评分1）
   - 适合门诊管理
   - 阿奇霉素改善不明显，可能为细菌性病因
   
2. 轻度脱水（E86.0）
   - 继发于口服摄入减少
   
3. 2型糖尿病（E11.9）
   - 控制良好，继续在家用药
```

#### P - 计划

**定义：** 诊断和治疗干预措施

**包含要素：**
- 诊断计划（进一步检查、影像学、转诊）
- 治疗计划（药物、手术、治疗）
- 患者教育和咨询
- 随访安排
- 给患者的具体指示
- 返回注意事项（何时就医）

**药物记录：**
- 药物名称（首选通用名）
- 剂量和途径
- 频率
- 疗程
- 适应症

**计划组织：**
- 按问题（与评估匹配）
- 按干预类型（诊断、治疗、教育）

**计划部分示例：**
```
P:
1. 社区获得性肺炎：
   诊断：目前无需进一步检查
   治疗：
   - 停用阿奇霉素
   - 开始阿莫西林克拉维酸钾875/125 mg口服，每日2次 x 7天
   - 支持治疗：充足水分、休息、对乙酰氨基酚退热
   教育： 
   - 解释细菌性肺炎诊断及抗生素更换
   - 讨论预期48-72小时内改善
   - 返回注意事项：呼吸困难加重、发热>103°F、意识模糊
   随访：48小时电话评估反应，1周后门诊就诊
   
2. 脱水：
   - 鼓励口服液体，目标2升/天
   - 运动饮料或电解质溶液均可
   
3. 2型糖尿病：
   - 继续二甲双胍1000 mg口服每日2次
   - 在家监测血糖
   - 按计划随访内分泌科

患者表示理解并同意计划。
```

### SOAP记录最佳实践

**文档标准：**
- 如手写则需字迹清晰
- 仅使用标准缩写
- 记录每次entry的日期和时间
- 签名并注明资质
- 实时或尽快记录
- 避免复制转发错误
- 审查并更新问题列表

**计费注意事项：**
- 记录医疗必要性
- 文档与计费级别匹配
- 包含E/M编码所需要素
- 时间计费需记录时间

**法律注意事项：**
- 记录事实，而非观点或判断
- 适当时引用患者原话
- 客观记录不依从情况
- 不得修改记录
- 更正需使用附录

## 病史和体格检查（H&P）

### 目的

- 综合基线评估
- 记录入院或首次就诊时的患者状态
- 指导诊断和治疗计划
- 入院后24小时内完成（TJC要求）

### H&P组成部分

#### 头部信息

- 患者姓名、出生日期、病历号
- 检查日期和时间
- 入院诊断
- 主治医师
- 科室
- 位置（急诊、病房、ICU）

#### 主诉（CC）

**定义：** 简要说明患者就诊原因

**格式：**
- 一句话
- 使用患者原话（加引号）
- 示例：CC："我喘不上气来"

#### 现病史（HPI）

**目的：** 当前问题的详细时间叙述

**必需要素（用于计费）：**
- 部位
- 性质
- 严重程度
- 持续时间
- 时间特点
- 背景
- 缓解因素
- 伴随体征/症状

**结构：**
- 开场陈述（人口统计学资料、就诊问题）
- 按时间顺序描述
- 症状特征
- 既往检查或治疗
- 促使现在就诊的原因

**示例：**
```
HPI：Smith先生是一位65岁男性，有充血性心力衰竭病史（左室射血分数35%），因3天来活动后进行性呼吸困难就诊。患者报告现在步行10英尺即出现呼吸困难（平时可走1-2个街区）。伴端坐呼吸（现在需要3个枕头，平时1个）和下肢水肿。否认胸痛、心悸或晕厥。报告按时服药但指出2天前利尿剂用完了。过去一周体重增加8磅。未曾在家监测每日体重。因呼吸困难加重且休息时也出现呼吸困难，今日就诊于急诊。
```

#### 既往病史（PMH）

**包含：**
- 慢性疾病
- 既往住院
- 重大疾病
- 损伤
- 儿童期疾病（如相关）

**格式：**
```
PMH:
1. 射血分数降低型心力衰竭（2018），6个月前超声心动图示射血分数35%
2. 冠状动脉疾病，冠状动脉旁路移植术后（2019）
3. 2型糖尿病（2010）
4. 高血压（2005）
5. 慢性肾脏病3期（基线肌酐1.8 mg/dL）
6. 高脂血症
```

#### 既往手术史（PSH）

**包含：**
- 所有手术和操作
- 日期（日期不详时年份也可）
- 如有并发症

**格式：**
```
PSH:
1. 冠状动脉旁路移植术x4（2019），术后并发房颤
2. 胆囊切除术（2015）
3. 阑尾切除术（儿童期）
```

#### 用药

**记录：**
- 首选通用名
- 剂量、途径、频率
- 如不明确则注明适应症
- 包含非处方药
- 草药补充剂
- 如患者无法提供列表则注明

**格式：**
```
用药：
1. 呋塞米40 mg口服每日（2天前用完）
2. 卡维地洛12.5 mg口服每日2次
3. 赖诺普利20 mg口服每日
4. 螺内酯25 mg口服每日
5. 二甲双胍1000 mg口服每日2次
6. 阿托伐他汀40 mg口服每日
7. 阿司匹林81 mg口服每日
8. 每日复合维生素
```

#### 过敏史

**记录：**
- 药物过敏及反应
- 食物过敏
- 环境过敏
- 如无已知过敏则注明NKDA

**格式：**
```
过敏史：
1. 青霉素→过敏性休克（儿童期）
2. 贝类→荨麻疹
3. ACE抑制剂→血管性水肿
```

#### 家族史（FH）

**包含：**
- 一级亲属（父母、兄弟姐妹、子女）
- 年龄及健康状况或死亡年龄及死因
- 相关遗传性疾病
- 如相关则包含就诊疾病的家族史

**格式：**
```
家族史：
父亲：冠心病，58岁时心梗，85岁在世
母亲：乳腺癌，72岁去世
兄弟姐妹：2型糖尿病
妹妹：健康
子女：2个儿子，均健康
```

#### 个人史（SH）

**包含：**
- 烟草使用（当前、既往、从不；如适用计算年包数）
- 酒精使用（每周饮酒量，如提示则用CAGE问题）
- 非法药物使用（当前、既往、从不；类型和途径）
- 职业
- 居住情况（独居、与家人、辅助生活等）
- 婚姻状况
- 性生活史（如相关）
- 运动习惯
- 饮食
- 功能状态

**格式：**
```
个人史：
烟草：既往吸烟者，10年前戒除（30年包史）
酒精：每周2-3瓶啤酒，否认酗酒
非法药物：否认
职业：退休电工
居住情况：与妻子同住，2层房子，卧室在楼上
婚姻状况：已婚
运动：因呼吸困难无法运动
饮食：低钠饮食（通常遵从）
功能状态：基线日常生活自理
```

#### 系统回顾（ROS）

**目的：** 按身体系统进行症状的系统筛查

**要求：**
- 综合检查至少10个系统
- 相关阳性和阴性
- 如已记录"其他所有系统回顾阴性"也可

**系统：**
1. **Constitutional**：发热、寒战、夜间出汗、体重变化、疲劳
2. **Eyes**：视力变化、疼痛、分泌物
3. **ENT**：听力下降、耳鸣、鼻窦问题、咽喉痛
4. **Cardiovascular**：胸痛、心悸、水肿、间歇性跛行
5. **Respiratory**：咳嗽、呼吸困难、哮鸣、咯血
6. **Gastrointestinal**：恶心、呕吐、腹泻、便秘、腹痛
7. **Genitourinary**：排尿困难、尿频、血尿、尿失禁
8. **Musculoskeletal**：关节疼痛、肿胀、僵硬、无力
9. **Skin**：皮疹、伤口、瘙痒、痣的变化
10. **Neurological**：头痛、头晕、晕厥、癫痫发作、无力、麻木
11. **Psychiatric**：情感变化、抑郁、焦虑、睡眠障碍
12. **Endocrine**：耐热/耐寒障碍、多尿、多饮
13. **Hematologic/Lymphatic**：易 bruising、出血、淋巴结肿大
14. **Allergic/Immunologic**：季节性过敏、频繁感染

**格式：**
```
ROS：
Constitutional：否认发热、寒战。报告疲劳和体重增加（8磅）。
Cardiovascular：报告呼吸困难、端坐呼吸、下肢水肿。否认胸痛、心悸、晕厥。
Respiratory：否认咳嗽、哮鸣、咯血。
Gastrointestinal：否认恶心、呕吐、腹泻、便秘、腹痛。
其他所有系统回顾阴性。
```

#### 体格检查

**总体组织：**
- 先记录生命体征
- 一般外观
- 按从头到脚系统检查

**生命体征：**
```
生命体征：体温98.2°F，血压142/88，心率105，呼吸24，室内空气下氧饱和度88% → 2L鼻导管吸氧下95%
身高：5'10"，体重：195磅（基线187磅），BMI 28
```

**系统检查：**

**一般：** 发育良好，中度呼吸困难的肥胖男性，端坐位于床上

**HEENT：**
- 头部：头颅形态正常，无外伤
- 眼睛：双侧瞳孔等大等圆，对光反应灵敏，无巩膜黄染
- 耳朵：双侧鼓膜清晰
- 鼻腔：鼻孔开放，无分泌物
- 咽喉：咽部无红斑或渗出

**Neck：** 柔软，无淋巴结肿大，颈静脉怒张至12 cm，无甲状腺肿大

**Cardiovascular：**
- 望诊：未可见心尖搏动
- 触诊：心尖搏动向外侧移位
- 听诊：心动过速但心律规整，S3奔马律存在，心尖部2/6全收缩期杂音，向腋下放射
- 周围脉搏：桡动脉2+，足背动脉1+双侧

**Pulmonary：**
- 望诊：呼吸功增加，使用辅助呼吸肌
- 触诊：触觉语颤对称
- 叩诊：双肺底叩诊浊音
- 听诊：双肺野中部可闻及湿啰音，无哮鸣音

**Abdomen：**
- 望诊：肥胖，无腹胀
- 听诊：肠音正常
- 叩诊：鼓音
- 触诊：软，无触痛，无肿块，无肝脾肿大

**Extremities：** 双侧胫前3+凹陷性水肿至小腿中部，无发绀或杵状指

**Skin：** 温暖干燥，无皮疹

**Neurological：**
- 精神状态：清醒，定向力完整
- 颅神经：II-XII完好
- 运动：四肢肌力5/5
- 感觉：浅触觉正常
- 反射：2+对称
- 因呼吸困难暂缓步态检查
- 小脑：指鼻试验正常

**Psychiatric：** 情感焦虑是与疾病相适应的，思维过程正常

#### 实验室和影像学

**包含：**
- 所有相关化验及参考范围
- 影像学检查及主要发现
- 心电图结果
- 其他诊断性检查

**示例：**
```
实验室数据：
全血细胞计数：白细胞8.5，血红蛋白11.2（L），红细胞压积34%，血小板245
基本代谢面板：钠132（L），钾3.2（L），氯98，二氧化碳30，尿素氮45（H），肌酐2.1（H，基线1.8），血糖145
肌钙蛋白：<0.04（正常）
脑钠肽：1250 pg/mL（H，升高）

影像学：
胸片：心脏扩大，双侧胸腔积液，肺血管充血，符合容量超负荷

心电图：窦性心动过速105bpm，左室肥厚，无急性ST-T改变
```

#### 评估和计划

**格式：** 基于问题的问题列表

**示例：**
```
评估和计划：

65岁男性，有充血性心力衰竭病史（左室射血分数35%），因急性失代偿性心力衰竭就诊。

1. 急性失代偿性心力衰竭（I50.23）
   - NYHA IV级症状
   - 查体和影像学显示容量超负荷
   - 因药物不依从诱发（呋塞米用完）
   - 脑钠肽升高至1250
   诊断：
   - 超声心动图评估当前射血分数和瓣膜功能
   - 每日体重和严格出入量记录
   治疗：
   - 呋塞米40 mg静脉每日2次，目标每日负平衡1-2升
   - 继续卡维地洛、赖诺普利、螺内酯
   - 2L鼻导管吸氧，目标氧饱和度>92%
   - 低钠饮食（<2g/天），液体限制1.5L/天
   - 心电监护
   随访：利尿后重新评估，目标3-5天出院

2. 慢性肾脏病3期基础上急性肾损伤（N17.9, N18.3）
   - 肌酐2.1（基线1.8），可能因前向流不佳导致肾前性
   - 每日监测，利尿后预期改善
   - 停用肾毒性药物

3. 低钾血症（E87.6）
   - 钾3.2，可能与既往利尿剂使用有关
   - 口服补钾40 mEq一次，然后重新评估
   - 继续螺内酯保钾

4. 低钠血症（E87.1）
   - 钠132，可能为容量超负荷导致的稀释性
   - 利尿后预期改善
   - 按上述限制液体

5. 2型糖尿病（E11.9）
   - 控制良好
   - 继续在家用二甲双胍
   - 住院期间监测血糖

6. 冠状动脉疾病（I25.10）
   - 稳定，无急性冠脉综合征
   - 继续阿司匹林、他汀、β受体阻滞剂

代码状态：完全复苏
安排：入院心电监护病房
```

## 出院小结

### 目的

- 向门诊医生传达住院治疗情况
- 记录住院过程和结果
- 确保连续性治疗
- 满足监管要求（TJC、CMS）

### 时限

**要求：**
- 出院后30天内完成（CMS）
- 许多医院要求24-48小时内完成
- 随访预约时可用

### 组成部分

#### 头部信息

- 患者人口统计学资料
- 入院日期和出院日期
- 住院时长
- 主治医师
- 会诊科室
- 主治医生

#### 入院诊断

住院的主要目的

#### 出院诊断

**格式：** 编号列表，优先排序

**示例：**
```
出院诊断：
1. 急性失代偿性心力衰竭
2. 慢性肾脏病3期基础上急性肾损伤
3. 低钾血症
4. 低钠血症
5. 冠状动脉疾病
6. 2型糖尿病
```

#### 住院过程

**内容：**
- 按时间顺序叙述或基于问题
- 关键事件和干预措施
- 治疗反应
- 执行的手术
- 会诊
- 并发症
- 重要检查结果

**示例（简要）：**
```
住院过程：
Smith先生因药物不依从导致急性失代偿性心力衰竭入院。静脉注射呋塞米利尿，3天内净负平衡5升，呼吸困难和下肢水肿显著改善。超声心动图示射血分数持续降低至30%，与之前相似。利尿后肾功能恢复至基线。电解质已补充并恢复正常。患者于住院第3天过渡至口服呋塞米，病情稳定。出院时室内空气下步行无呼吸困难。已进行全面的心力衰竭教育。
```

```
手术操作：
1. 超声心动图经胸（住院第1天）
```

#### 出院用药

**格式：**

- **新**用药突出显示
- **更改**用药注明
- **停用**用药列出

**示例：**

### 随访计划

**包含：**
- 已预约的就诊
- 建议的随访时间
- 出院时待完成的检查或研究
- 已进行的转诊

**示例：**
```
随访：
1. [日期] [时间]与Jones医生心脏病学预约

4. 1周后复查基本代谢面板（已安排，化验单已提供）
```

#### 患者指导

**包含：**
- 活动限制
- 饮食限制
- 伤口护理（如适用）
- 设备或家庭服务
- 监测指导（每日体重、血糖、血压）
- 返回注意事项

**示例：**
```
患者指导：
1. 每天早晨称体重，如1天内体重增加>2磅或1周内>5磅则致电医生
2. 低钠饮食（<2克/天）
3. 液体限制2升/天
4. 按医嘱服用所有药物，不要断药
5. 活动：根据耐受恢复日常活动
6. 如出现严重呼吸困难、胸痛、严重水肿或其他令人担忧的症状则返回急诊或拨打911
```
