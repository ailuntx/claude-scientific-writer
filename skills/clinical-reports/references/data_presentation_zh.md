# 临床报告中的数据呈现

## 临床数据表格

### 表格设计原则

**一般指南：**
- 清晰、简洁的标题描述表格内容
- 列标题带单位
- 行标签左对齐，数据适当对齐（数字右对齐，文本左对齐）
- 脚注用于缩写、统计符号、特殊情况
- 一致的小数位数（百分比通常1-2位，连续变量1-3位）
- 整篇文档格式一致

**标题位置：**
- 表格上方
- 顺序编号（表1、表2等）
- 足够描述性以独立存在

**脚注符号（按顺序）：**
- *, †, ‡, §, ||, ¶, #
- 或使用上标字母（a, b, c...）
- 或使用上标数字（如果不与参考文献混淆）

**关键要素：**
- 每组的样本量（N=）
- 连续变量：平均值（SD）、中位数（IQR）、范围
- 分类变量：n (%)
- 基线比较不显示p值（有争议但 generally not recommended）

**示例：**

  95% CI                    (-2.0, -1.6)   (-0.8, -0.4)  (-1.5, -0.9)

次要终点
FPG变化，mg/dL
  平均值（标准误）          -42.5 (3.2)     -15.2 (3.4)   -27.3       <0.001
  95% CI                    (-48.8, -36.2) (-21.9, -8.5) (-36.4, -18.2)

达到HbA1c <7%的人数
  n (%)                     78 (52.0)       25 (17.2)     -           <0.001
  95% CI                    (43.9, 60.1)   (11.4, 24.5)

体重变化，kg
  平均值（标准误）          -3.2 (0.4)      -0.5 (0.4)    -2.7        <0.001
  95% CI                    (-4.0, -2.4)   (-1.3, 0.3)   (-3.8, -1.6)
──────────────────────────────────────────────────────────────────────────────

**统计呈现：**
- 点估计值伴精确度指标（SE或CI）
- p值（考虑多重性校正）
- 效应量（差值或比值）伴95% CI
- 说明显著性水平（例如p<0.05, p<0.01, p<0.001）

## 临床数据图表

### 图表设计原则

**一般指南：**
- 清晰、简洁的图注/图例置于图下方
- 顺序编号（图1、图2等）
- 坐标轴标签带单位
- 字体大小可读（最小8-10磅）
- 高分辨率（印刷300 dpi，网络150 dpi）
- 适合色盲的配色方案
- 兼容黑白（使用不同符号/图案）

**图注：**
- 描述显示内容
- 解释符号、误差棒、统计标注
- 定义缩写
- 提供解读背景

### Kaplan-Meier生存曲线

**目的：** 显示事件时间数据

**要素：**
- X轴：时间（周、月、年）
- Y轴：无事件生存概率（0到1或0%到100%）
- 每组单独的曲线
- 删失观测标记（通常用垂直刻度线标记）
- 风险人数表置于图下方
- 显示中位生存时间
- log-rank p值
- 风险比伴95% CI

**图注示例：**
```
图1. 总生存的Kaplan-Meier曲线

治疗组和对照组的总生存Kaplan-Meier估计。刻度线表示删失观测。
风险人数示于图下方。Log-rank p<0.001。中位生存时间：治疗组24.5个月
（95% CI: 22.1-26.8），对照组18.2个月（95% CI: 16.5-20.1）。
风险比0.68（95% CI: 0.55-0.84）。
```

### 森林图

**目的：** 显示亚组分析或Meta分析结果

**要素：**
- 点估计值（方形或菱形）
- 符号大小与精确度成比例（逆方差）或样本量
- 显示95% CI的水平线
- 无效应垂直线（HR=1.0, OR=1.0, 或差值=0）
- 亚组标签在左侧
- 效应量值在右侧
- 总体估计（如果是Meta分析）
- 异质性统计量（I², p值）

**图注示例：**
```
图2. 亚组治疗效应的森林图

治疗组与对照组对主要终点的效应在预设亚组中的比较。
方形表示点估计值；水平线表示95%置信区间。方形大小与亚组样本量成比例。
总体效应显示为菱形。交互作用检验的p值用于检验亚组间治疗效应的异质性。
```

### 箱线图

**目的：** 显示连续变量的分布

**要素：**
- 箱体：四分位距（第25至第75百分位数）
- 箱内线：中位数
- 须线：延伸至1.5×四分位距范围内的最极端数据点
- 异常值：须线外的点（通常显示为圆圈）
- X轴：组或时间点
- Y轴：带单位的连续变量

### 带回归的散点图

**目的：** 显示两个连续变量之间的关系

**要素：**
- X轴：自变量
- Y轴：因变量
- 单个数据点
- 回归线（如果适当）
- 回归方程
- R²值
- 斜率的p值
- 回归线的95%置信区间（可选，显示为阴影区域）

### 折线图

**目的：** 显示个体随时间的变化轨迹

**要素：**
- X轴：时间
- Y轴：结局变量
- 单个患者线条（通常半透明）
- 平均轨迹（粗线）
- 治疗组用不同颜色区分

### 柱状图

**目的：** 比较各组间的比例或均值

**要素：**
- 柱子之间清晰分隔
- 误差棒（SEM或95% CI）
- Y轴从0开始（柱状图不要截断）
- X轴上组标签
- Y轴上带单位的值标签
- 标明统计显著性（p值或星号）

**避免：**
- 3D柱状图（扭曲感知）
- 过度装饰
- 柱状图的Y轴截断

### 线图

**目的：** 显示随时间的变化

**要素：**
- X轴：时间（间隔一致）
- Y轴：结局变量
- 每组单独的线（不同颜色/图案）
- 标记数据点（圆圈、正方形、三角形）
- 每个时间点误差棒（SE或95% CI）
- 识别各组的图例
- 网格线（可选，浅灰色）

### 直方图

**目的：** 显示连续变量的分布

**要素：**
- X轴：变量（分箱）
- Y轴：频数或密度
- 适当的箱宽（不太少，不太 多）
- 叠加正态分布曲线（如果要检验正态性）

## 临床数据的特殊考虑

### 呈现比例

**分子和分母：**
- 同时提供两者：25/100 (25%)
- 不仅显示百分比（25%）

**百分比：**
- 如果n<100，无小数位
- 如果n≥100，1位小数
- 百分比永不超过1位小数

**比例的置信区间：**
- Wilson得分区间或精确二项式（优于小样本的Wald）
- 与百分比同时报告

### 呈现连续数据

**集中趋势指标：**
- 正态分布数据用平均值
- 偏态数据或有序数据用中位数
- 如果分布不清楚，两者都报告

**离散程度指标：**
- **标准差（SD）**：描述数据的变异性
- **标准误（SE）**：描述均值的精确度
- **95%置信区间**：推论统计的首选
- **四分位距（IQR）**：与偏态数据的中位数一起使用
- **范围**：最小值至最大值

**何时使用：**
- 描述性统计 → 平均值（SD）或中位数（IQR）
- 推论统计 → 平均值（95% CI）或平均值（SE）
- 不使用±而不说明SD、SE或CI

### 呈现P值

**报告指南：**
- 报告精确p值至2-3位小数（p=0.042）
- 对于非常小的p值，使用p<0.001（不是p=0.000）
- 不报告为"NS"或"p=NS"
- 对于不显著的结果，报告精确p值（p=0.18，不是p>0.05）
- 说明双尾除非预设单尾
- 适当校正多重比较
- 报告使用的显著性阈值（α=0.05是标准）

**避免：**
- p<0.05（报告精确值）
- p=0.00（不可能）
- 多位小数（p=0.04235891）

### 统计显著性指示符

**选项：**
1. 在表中报告p值
2. 使用带图例的星号：
   - *p<0.05
   - **p<0.01
   - ***p<0.001
3. 使用置信区间（首选）

### 置信区间

**报告：**
- 95% CI是标准
- 格式：（下限，上限）
- 或：下限至上限
- 或：下限-上限

**解释：**
- 如果差值的CI排除0 → 显著
- 如果比值的CI排除1 → 显著
- CI宽度表示精确度

### 缺失数据

**清楚说明：**
- 脚注解释缺失数据
- 清楚说明是否为完整病例分析
- 如果使用插补方法，描述方法
- 报告每个变量的缺失数据量

### 小数位和四舍五入

**一般规则：**
- 报告至测量精确度水平
- 表内小数位一致
- p值四舍五入至2-3位小数
- 百分比四舍五入至0-1位小数
- 平均值/中位数四舍五入至1-2位小数
- 包括适当有效数字

## 创建图表的软件

**统计软件：**
- R (ggplot2) - 高度可定制
- GraphPad Prism - 生物医学用户友好
- SAS、Stata、SPSS - 综合统计软件包
- Python (matplotlib, seaborn) - 灵活强大

**通用图形软件：**
- Adobe Illustrator - 专业出版质量
- Inkscape - 免费矢量图形编辑器
- PowerPoint - 基本图表，易于使用
- BioRender - 生物示意图和图表

## 配色方案

**适合色盲的配色：**
- 避免红绿组合
- 使用蓝-橙、蓝-黄
- 包括形状/图案差异
- 在灰度下测试图表

**推荐配色：**
- ColorBrewer（为数据可视化设计）
- Viridis（感知均匀）
- IBM色盲安全配色

## 图像质量标准

**分辨率：**
- 印刷出版300 dpi
- 网络/屏幕150 dpi
- 矢量图形（PDF、SVG）更适合图表

**文件格式：**
- TIFF或EPS用于印刷
- PNG用于网络
- PDF用于矢量图形
- JPEG可用于照片（高质量）

**图像处理：**
- 不更改数据的处理
- 仅可接受的调整：亮度、对比度、应用于整个图像的色彩平衡
- 记录所有调整
- 如需要提供原始图像
