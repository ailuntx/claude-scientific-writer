# 诊断报告标准

## 放射学报告标准

### 美国放射学会（ACR）指南

ACR 提供全面的诊断影像报告实践参数，以确保质量、一致性和沟通有效性。

#### 核心放射学报告组成部分

**1. 患者人口统计学信息**
- 患者姓名和/或唯一标识符
- 出生日期或年龄
- 性别
- 病历号
- 检查日期和时间
- 转诊医生

**2. 检查/操作**
- 执行的特定检查
- 解剖区域
- 偏侧性（右、左、双侧）
- 技术和方案
- 示例："颅脑MRI平扫及增强"

**3. 临床指征**
- 检查原因
- 相关临床病史
- 具体临床问题
- ICD-10编码（如需要）
- 示例："头痛和视觉障碍。排除颅内肿块。"

**4. 对比**
- 既往相关影像学检查
- 既往检查的日期
- 既往检查的 modalities
- 是否可对比
- 示例："对比：6个月前的颅脑CT平扫（2023年1月15日）"

**5. 技术**
- 影像学参数和方案
- 对比剂使用详情：
  - 类型（含碘、钆）
  - 途径（静脉、口服、直肠）
  - 给药剂量
  - 成像时机
- 技术质量声明
- 辐射剂量（CT）
- 局限性或技术问题
- 示例：
  ```
  技术：获取了多平面T1和T2加权序列，颅脑平扫及静脉增强。15 mL钆对比剂静脉给药。技术质量合格。
  ```

**6. 发现**
- 影像学发现的系统描述
- 按解剖区域或器官系统组织
- 异常的大小测量（尺寸、体积）
- 具体的描述性术语
- 相关阳性发现
- 相关阴性发现
- 与既往检查对比（如有）

**组织方式：**
- 按器官（腹部/盆腔）
- 按区域（胸部）
- 按系统（脊柱）
- 按间隙（肌肉骨骼）

**7. 意见/结论**
- 关键发现的总结
- 诊断或鉴别诊断
- 临床问题的答案
- 关注程度或紧急程度
- 与既往对比（改善、稳定、恶化）
- 进一步影像学检查或临床管理的建议
- 清晰简洁（通常为编号列表）

示例：
```
意见：
1. 右侧额叶3.2 cm强化肿块伴周围血管性水肿，最符合高级别胶质瘤。不能排除转移。建议临床相关检查和组织取样。
2. 无急性颅内出血或脑疝。
3. 建议神经外科会诊。
```

**8. 危急结果沟通**
- 需要立即处理的紧急或意外发现
- 记录与开单医生的直接沟通
- 口头沟通的时间、日期和接收人
- 示例："危急结果：急性肺栓塞。Smith医生于[日期]14:35被呼叫。"

### 结构化报告系统

#### Lung-RADS（肺部CT筛查报告和数据系统）

用于肺癌筛查CT解读。

**分类：**
- **Lang-RADS 0**：不完整——需要额外影像学检查
- **Lung-RADS 1**：阴性——无结节，肯定为良性结节
- **Lung-RADS 2**：良性表现或行为——恶性可能性极低的结节
- **Lung-RADS 3**：可能良性——建议短间隔随访
- **Lung-RADS 4A**：可疑——3个月随访或PET/CT
- **Lung-RADS 4B**：非常可疑——3个月随访或PET/CT，考虑活检
- **Lung-RADS 4X**：非常可疑伴额外特征，考虑活检

**每个分类均包含管理建议**

#### BI-RADS（乳腺影像报告和数据系统）

乳腺影像学标准化术语（乳腺X线摄影、超声、MRI）。

**分类：**
- **BI-RADS 0**：不完整——需要额外影像学检查
- **BI-RADS 1**：阴性——无异常
- **BI-RADS 2**：良性发现
- **BI-RADS 3**：可能良性——短间隔随访（6个月）
- **BI-RADS 4**：可疑——建议活检
  - 4A：低度可疑
  - 4B：中度可疑
  - 4C：高度可疑
- **BI-RADS 5**：高度提示恶性——建议活检
- **BI-RADS 6**：已知活检证实恶性

**描述符：**
- 肿块：形状、边缘、密度
- 钙化：形态、分布
- 不对称：类型和特征
- 相关特征

#### LI-RADS（肝脏影像报告和数据系统）

用于报告肝细胞癌风险患者的肝脏观察结果。

**分类：**
- **LI-RADS 1**：肯定良性
- **LI-RADS 2**：可能良性
- **LI-RADS 3**：恶性中间概率
- **LI-RADS 4**：可能为HCC
- **LI-RADS 5**：肯定为HCC
- **LI-RADS M**：可能或肯定为恶性，非HCC特异性
- **LI-RADS TIV**：静脉肿瘤

**评估的主要特征：**
- 大小
- 强化模式（动脉期强化、洗脱）
- 包膜外观
- 阈值增长

#### PI-RADS（前列腺影像报告和数据系统）

用于前列腺多参数MRI。

**评估分类：**
- **PI-RADS 1**：非常低——临床显著癌可能性极低
- **PI-RADS 2**：低——临床显著癌不太可能
- **PI-RADS 3**：中间——不确定
- **PI-RADS 4**：高——临床显著癌可能
- **PI-RADS 5**：非常高——临床显著癌可能性极高

**评估：**
- 外周带：DWI/ADC为主要决定因素
- 过渡带：T2加权为主要决定因素
- DCE（动态对比增强）：用于外周带的PI-RADS 3病变

### RadLex和标准化术语

**RadLex** 是由北美放射学会（RSNA）开发的放射学综合术语库。

**优势：**
- 标准化术语
- 改善沟通
- 支持数据挖掘和分析
- 促进决策支持系统
- 一致的报告结构

**常用RadLex术语：**
- 解剖结构
- 影像学观察
- 疾病实体
- 操作程序

### 放射学测量

**线性测量：**
- 使用二维（长×宽）或三维（长×宽×高）
- 报告结节/肿块的最大径
- 随访时使用一致的测量方法
- 尽可能使用垂直测量

**体积测量：**
- 对于不规则病变的随访更准确
- 自动或半自动软件
- 对肺结节特别有用

**疗效评估：**
- RECIST 1.1（实体瘤疗效评估标准）
  - 靶病灶：最长径之和（最多5个病灶，每个器官最多2个）
  - 完全缓解、部分缓解、疾病稳定、疾病进展

## 病理报告标准

### 美国病理学家学会（CAP）方案

CAP癌症方案为癌症标本提供标准化的摘要报告模板。

#### 摘要报告要素

**所有癌症标本的核心要素：**

**1. 标本信息**
- 操作类型（活检、切除、切除术）
- 标本偏侧性
- 标本完整性和充分性

**2. 肿瘤部位**
- 解剖部位和亚部位
- 器官内的精确位置

**3. 肿瘤大小**
- 最大径（cm）
- 如三维测量相关则提供额外尺寸
- 测量方法（大体 vs 镜下）

**4. 组织学类型**
- WHO分类
- 具体亚型
- 混合肿瘤中各成分的百分比

**5. 组织学分级**
- 使用的分级系统（如Nottingham、Fuhrman、Gleason）
- 分级类别（高分化、中分化、低分化或G1、G2、G3）
- 如适用各成分评分

**6. 侵犯范围**
- 侵犯深度（mm）
- 邻近结构受累
- 淋巴血管侵犯（存在/未识别）
- 神经周围侵犯（存在/未识别）

**7. 切缘**
- 最近切缘距离
- 每个评估切缘的切缘状态（阴性/阳性）
- 如为阳性则注明具体受累切缘

**8. 淋巴结**
- 检查的淋巴结数量
- 有转移的淋巴结数量
- 最大转移灶的大小
- 淋巴结外侵犯（存在/缺失）

**9. 病理分期（pTNM）**
- pT：原发肿瘤范围
- pN：区域淋巴结
- pM：远处转移（如已知）
- 使用的AJCC癌症分期手册版本

**10. 其他发现**
- 治疗效果（如新辅助治疗后）
- 相关病变（发育不良、原位癌）
- 背景组织（肝硬化、炎症）

**11. 辅助研究**
- 免疫组织化学结果
- 分子/基因检测结果
- 生物标志物状态（如乳腺癌的ER、PR、HER2；结肠癌的MSI）
- FISH或其他细胞遗传学结果

#### 器官特异性CAP方案

**乳腺癌：**
- 组织学类型（浸润性导管癌、浸润性小叶癌、特殊类型）
- Nottingham分级（腺管形成、核多形性、有丝分裂计数）
- ER/PR状态（百分比和强度）
- HER2状态（IHC评分，如需要则FISH）
- Ki-67增殖指数
- DCIS成分（如存在）
- 新辅助治疗反应（残余癌症负担）

**结直肠癌：**
- 组织学类型（腺癌、粘液癌等）
- 分级
- 侵犯深度（粘膜下层、肌层、结肠周围组织等）
- 肿瘤沉积
- 淋巴结（阳性数/检查总数）
- 切缘（近端、远端、径向/环周）
- MSI/MMR状态
- KRAS、NRAS、BRAF突变

**前列腺癌：**
- Gleason评分（主要+次要模式）
- 分级组（1-5）
- 受累组织百分比
- 前列腺外侵犯
- 精囊侵犯
- 手术切缘状态
- 如采样则包括淋巴结

**肺癌：**
- 组织学类型（腺癌、鳞癌、小细胞癌等）
- 分级（NSCLC）
- 侵犯深度
- 脏层胸膜侵犯
- 距切缘距离
- 淋巴结
- 分子标志物（EGFR、ALK、ROS1、PD-L1）

### 大体病理描述

**基本要素：**
- 标本标签和识别
- 标本类型
- 尺寸和重量
- 定向标记（如存在）
- 外部表面描述
- 切面外观
- 病变描述：
  - 尺寸（三维）
  - 位置
  - 颜色
  - 质地
  - 边界（边界清楚、浸润性）
  - 距切缘距离
- 采样方式（组织如何切开和提交）

**示例：**
```
大体描述：
新鲜接收，标注患者姓名和"左乳，肿块切除术"，为一定向肿块切除标本，尺寸8.5 x 6.0 x 4.0 cm，上切缘有缝线标记。墨水标记：上-蓝色，下-黑色，内侧-绿色，外侧-红色，前-橙色，后-黄色。连续切开显示一灰白色坚硬肿块，尺寸2.1 x 1.8 x 1.5 cm，位于上切缘2.5 cm、下切缘3.0 cm、内侧2.0 cm、外侧3.5 cm、前方1.5 cm、后方1.8 cm处。代表性切片提交如下：A1-A3肿瘤，A4上切缘，A5内侧切缘，A6后切缘。
```

### 镜下描述

**关键要素：**
- 生长模式
- 细胞特征
  - 细胞类型
  - 核特征（大小、形状、染色质、核仁）
  - 胞质特征
  - 有丝分裂活动
- 分化程度
- 侵犯模式
- 特殊特征（坏死、出血、钙化）
- 间质和背景组织
- 淋巴血管或神经周围侵犯
- 切缘（距离和状态）
- 淋巴结（转移描述）

### 冰冻切片报告

**适应症：**
- 术中诊断
- 切缘评估
- 淋巴结评估
- 组织分拣

**报告格式：**
- 清楚标注"冰冻切片诊断"
- 术中会诊记录
- 冰冻切片时间
- 标本描述
- 冰冻切片诊断
- 注明："常规切片待出"

**冰冻切片免责声明：**
- 冰冻伪影限制
- 常规切片的最终诊断
- 最终诊断以常规切片为准

### 诊断确定性语言

**确定性：**
- "符合..."
- "诊断为..."
- "阳性..."

**可能：**
- "符合..."
- "倾向..."
- "最可能..."

**可疑：**
- "提示..."
- "不能排除..."
- "鉴别诊断包括..."

**待定：**
- "待..."
- "建议..."
- "待附加研究..."

## 实验室报告标准

### 临床实验室标准协会（CLSI）指南

CLSI为实验室检测和报告提供标准。

#### 实验室报告组成部分

**1. 患者人口统计学信息**
- 患者姓名和标识符
- 出生日期或年龄
- 性别
- 开单医生

**2. 标本信息**
- 标本类型（血液、血清、血浆、尿液、脑脊液等）
- 采集日期和时间
- 接收日期和时间
- 标本状态
- 禁食状态（如相关）

**3. 检测信息**
- 检测名称（全称，而非仅缩写）
- 检测代码
- 方法学
- 编号或标本号

**4. 结果**
- 定量值及单位
- 定性结果（阳性/阴性、检测到/未检测到）
- 参考范围或区间
- 异常结果标记
  - H = 增高
  - L = 降低
  - 危急值或 panic值突出显示

**5. 参考区间**
- 年龄特异性
- 性别特异性
- 人群特异性（如相关）
- 方法特异性
- 单位清晰标注

**示例：**
```
检测：糖化血红蛋白
结果：8.2%  (H)
参考范围：4.0-5.6%（非糖尿病）
方法：HPLC
解释：符合糖尿病控制不佳
```

**6. 解释性注释**
- 当结果需要背景时
- 建议附加检测时
- 解释干扰或局限性时
- 提供临床指导时

**7. 质量控制**
- 差值检查（与先前值比较）
- 危急值和回读程序
- 标本质量问题（溶血、乳糜、黄疸）
- 执行的稀释
- 如需要重复检测

### LOINC（逻辑观察标识符名称和编码）

实验室和临床观察的标准编码系统。

**LOINC编码组成部分：**
- 成分（测定的分析物）
- 属性（质量、物质浓度等）
- 时间（点、时间、24小时）
- 系统（标本类型）
- 尺度（定量、有序、名义）
- 方法（如相关）

**示例：**
- 血液中糖化血红蛋白：4548-4
- 血清/血浆中葡萄糖：2345-7
- 血清/血浆中肌酐：2160-0

### 危急值报告

**定义：** 表明需要立即临床处理的危及生命状况的结果。

**危急值示例：**
- 葡萄糖：<40 mg/dL或>500 mg/dL
- 钾：<2.5 mEq/L或>6.5 mEq/L
- 钠：<120 mEq/L或>160 mEq/L
- 钙：<6.0 mg/dL或>13.0 mg/dL
- 白细胞：<1.0 ×10³/μL或>50 ×10³/μL
- 血红蛋白：<5.0 g/dL
- 血小板：<20 ×10³/μL
- INR：>5.0（华法林使用者）
- 血培养阳性
- 脑脊液培养或革兰染色阳性

**危急值程序：**
1. 实验室识别结果
2. 立即联系开单医生或指定人
3. 回读验证
4. 记录：
   - 日期和时间
   - 联系的人员
   - 接收通知的人员
   - 检测和结果
5. 如无法联系医生，按机构政策处理

### 微生物学报告

**培养报告：**
- 标本类型和来源
- 识别的生物
- 数量（少量、中等量、大量生长）
- 抗菌药物敏感性结果
- 解释（敏感、中介、耐药）
- 如适用则提供MIC值

**革兰染色报告：**
- 存在的细菌（革兰阳性/阴性、形态）
- 数量和细胞背景
- 存在的白细胞或其他细胞

**初步报告：**
- 在最终鉴定前发布
- 清楚标注"初步"
- 后续发出最终报告

**最终报告：**
- 确定的生物鉴定
- 完整的敏感性检测板
- 解释性注释

### 分子病理/基因组学报告

**组成部分：**
- 检测的基因
- 检测到的变异
- 分类（致病、可能致病、VUS、可能良性、良性）
- 等位基因频率
- 方法学（NGS、Sanger测序、PCR等）
- 参考序列
- 临床意义和解释
- 建议（治疗影响、家族检测）
- 检测的局限性

**示例：**
```
检测：BRCA1/BRCA2全基因测序
结果：检测到致病性变异
基因：BRCA1
变异：c.68_69delAG (p.Glu23ValfsTer17)
分类：致病性
解释：该变异与乳腺癌和卵巢癌风险增加相关。建议遗传咨询和风险降低策略。应考虑家族检测。
```

### 即时检测（POCT）

**要求：**
- 与中心实验室相同的质量标准
- 操作员能力记录
- 质量控制记录
- 维护记录
- 医疗记录中的结果记录

**常见POCT：**
- 血糖
- 血红蛋白/血细胞比容
- INR
- 血气
- 妊娠试验
- 尿分析
- 快速链球菌检测
- 流感

## 诊断报告质量指标

### 放射学质量指标

- 报告周转时间（常规 vs 急诊）
- 危急结果沟通时间
- 报告错误率
- 补充报告率
- 转诊医生满意度

**基准：**
- 常规报告：<24小时
- 急诊报告：<4小时
- 即时报告：<1小时
- 危急发现：立即口头沟通

### 病理质量指标

- 不同标本类型的周转时间（TAT）
- 冰冻切片准确性
- 修订率
- 标本充分率
- 免疫组织化学质量控制

**TAT基准：**
- 常规外科病理：2-3天
- 复杂外科病理：5-7天
- 细胞病理：1-2天
- 冰冻切片：术中15-20分钟

### 实验室质量指标

- 从采集到结果的TAT
- 危急值通知时间
- 标本拒收率
- 熟练测试性能
- 差值检查失败率

**TAT基准：**
- 急诊实验室：<60分钟
- 常规实验室：2-4小时
- 外送检测：按参考实验室时间
