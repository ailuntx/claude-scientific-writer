# 研究海报质量检查清单

在打印或展示研究海报之前使用此综合检查清单。

## 编译前检查

### 内容完整性
- [ ] 标题简洁且具有描述性（10-15个词）
- [ ] 所有作者姓名拼写正确
- [ ] 单位信息完整准确
- [ ] 包含联系邮箱
- [ ] 所有部分都已包含：引言、方法、结果、结论
- [ ] 引用文献（5-10个关键引用）
- [ ] 包含致谢（资助方、合作者）
- [ ] 没有剩余的占位符文本（TODO、Lorem ipsum等）

### 视觉内容
- [ ] 所有图片已准备且高分辨率（300+ DPI）
- [ ] 图片说明文字已编写且具有描述性
- [ ] 标志可用（大学、资助机构）
- [ ] 二维码已生成并测试
- [ ] 图标/图形已获取（如使用）

### LaTeX配置
- [ ] 指定正确的纸张尺寸（A0、A1、36×48英寸等）
- [ ] 正确的方向（竖版/横版）
- [ ] 配置最小边距（5-15毫米）
- [ ] 字体尺寸适当（标题72pt+，正文24pt+）
- [ ] 定义配色方案
- [ ] 所有包已安装并正常工作

## 编译检查

### 成功编译
- [ ] PDF编译无错误
- [ ] .log文件中无严重警告
- [ ] 所有引用已解析（无[?]标记）
- [ ] 所有交叉引用正常工作
- [ ] 参考文献正确生成（如使用BibTeX）

### 警告审查
在终端运行：`grep -i "warning\|overfull\|underfull" poster.log`

- [ ] 无overfull hbox警告（文本过宽）
- [ ] 无underfull hbox警告（间距过大）
- [ ] 无缺少图片警告
- [ ] 无缺少字体警告
- [ ] 无未定义引用警告

## PDF质量检查

### 自动检查

运行：`./scripts/review_poster.sh poster.pdf` 或手动验证：

#### 页面规格
```bash
pdfinfo poster.pdf | grep "Page size"
```
- [ ] 页面尺寸完全符合要求
- [ ] 单页文档（非多页）
- [ ] 方向正确

#### 字体嵌入
```bash
pdffonts poster.pdf
```
- [ ] 所有字体在"emb"列显示"yes"
- [ ] 无位图字体（应为Type 1或TrueType）

#### 图片质量
```bash
pdfimages -list poster.pdf
```
- [ ] 所有图片至少300 DPI
- [ ] 图片中无JPEG伪影
- [ ] 尽可能使用矢量图形

#### 文件大小
```bash
ls -lh poster.pdf
```
- [ ] 大小合理（通常2-50 MB）
- [ ] 如果数字分享，不过大（<50 MB）
- [ ] 不过小（<1 MB - 可能表明质量低）

## 视觉检查（100%缩放）

### 布局和间距
- [ ] 内容填充整个页面（无边距过大）
- [ ] 列间距一致（1-2厘米）
- [ ] 块间距一致（1-2厘米）
- [ ] 所有元素对齐网格
- [ ] 无文本或图片重叠
- [ ] 空白均匀分布（占总面积30-40%）
- [ ] 海报视觉平衡（无过重/空旷区域）

### 排版
- [ ] 标题清晰醒目（72-120pt）
- [ ] 部分标题清晰（48-72pt）
- [ ] 正文足够大（最小24-36pt，建议30pt+）
- [ ] 说明文字清晰可读（18-24pt）
- [ ] 无文本溢出边缘
- [ ] 全篇字体使用一致
- [ ] 行距适当（1.2-1.5×）
- [ ] 无尴尬的连字符或断词
- [ ] 所有特殊字符正确渲染（希腊字母、数学符号）

### 视觉元素
- [ ] 所有图片显示正确
- [ ] 无像素化或模糊的图片
- [ ] 图片分辨率高（放大到200%验证）
- [ ] 图片标签大而清晰
- [ ] 图表坐标轴标注单位
- [ ] 各图配色方案一致
- [ ] 图例清晰可读且位置得当
- [ ] 标志清晰专业
- [ ] 二维码清晰高对比度（最小2×2厘米）
- [ ] 无视觉伪影或渲染错误

### 颜色
- [ ] 颜色按预期渲染（不过淡）
- [ ] 文本与背景对比度高（≥4.5:1）
- [ ] 配色方案和谐
- [ ] 颜色适合打印（不过亮/霓虹色）
- [ ] 正确使用机构颜色
- [ ] 色盲友好调色板（避免仅红绿）

### 内容
- [ ] 标题完整且位置正确
- [ ] 所有作者姓名和单位可见
- [ ] 所有部分存在且标注
- [ ] 结果部分有图片/数据
- [ ] 结论清晰陈述
- [ ] 引用格式一致
- [ ] 联系信息清晰可见
- [ ] 无遗漏内容

## 缩小比例打印测试（关键）

### 测试打印准备
按25%比例打印海报：
- A0海报 → 用A4纸打印
- 36×48英寸海报 → 用Letter纸打印
- A1海报 → 用A5纸打印

### 远距离可读性

**距离6英尺（2米）：**
- [ ] 标题清晰可读
- [ ] 作者可识别
- [ ] 主要图片可见

**距离4英尺（1.2米）：**
- [ ] 部分标题可读
- [ ] 图片说明文字可读
- [ ] 关键结果可见

**距离2英尺（0.6米）：**
- [ ] 正文可读
- [ ] 引用可读
- [ ] 所有细节清晰

### 打印质量
- [ ] 颜色准确（与屏幕预期相符）
- [ ] 无条带或颜色偏移
- [ ] 边缘清晰（不模糊）
- [ ] 打印密度一致
- [ ] 无打印机伪影

## 快速参考：常见问题

| 问题 | 快速修复 |
|-------|-----------|
| 白色边距过大 | 在documentclass中减小边距：`margin=5mm` |
| 文本太小 | 在beamerposter中增加比例：`scale=1.5` |
| 图片模糊 | 使用矢量图形（PDF）或更高分辨率（600+ DPI） |
| 颜色错误 | 检查RGB vs CMYK，最终打印前测试 |
| 字体未嵌入 | 编译时使用：`pdflatex -dEmbedAllFonts=true` |
| 内容被截断 | 检查总宽度：列+间距+边距=页面宽度 |
| 二维码无法扫描 | 增加尺寸（最小2×2厘米），确保高对比度 |
| 文件太大 | 压缩：`gs -sDEVICE=pdfwrite -dPDFSETTINGS=/printer ...` |
