# Paper2Poster：学术海报生成

## 概述

Paper2Poster 可自动从研究论文生成专业学术海报。该系统提取关键内容、设计视觉效果佳的布局，并创建适合会议、学术研讨会和学术演示的可打印海报。

## 核心功能

### 1. 内容提取
- 识别关键发现和贡献
- 提取重要的图表
- 总结方法论
- 突出结果和结论
- 保留引文和参考文献

### 2. 布局设计
- 创建平衡、专业的布局
- 优化内容密度和留白
- 建立清晰的视觉层次
- 支持多种海报尺寸
- 适应不同内容类型

### 3. 视觉设计
- 应用配色方案和品牌标识
- 优化字体可读性
- 确保图表质量和大尺寸
- 创建统一的视觉标识
- 保持学术演示标准

## 使用方法

### 基本海报生成

```bash
python pipeline_all.py \
  --input-dir "path/to/papers" \
  --output-dir "path/to/output" \
  --model-choice 1 \
  --generate-poster
```

### 自定义海报尺寸

```bash
python pipeline_all.py \
  --input-dir "path/to/papers" \
  --output-dir "path/to/output" \
  --model-choice 2 \
  --generate-poster \
  --poster-width-inches 60 \
  --poster-height-inches 40
```

### 参数

**基本配置：**
- `--input-dir`：包含论文文件的目录
- `--output-dir`：生成海报的输出目录
- `--model-choice`：LLM模型选择（1=GPT-4，2=GPT-4.1）
- `--generate-poster`：启用海报生成

**海报尺寸：**
- `--poster-width-inches`：宽度（英寸），默认48
- `--poster-height-inches`：高度（英寸），默认36
- `--poster-orientation`：横向或纵向，默认横向
- `--poster-dpi`：DPI分辨率，默认300

**设计选项：**
- `--poster-template`：模板样式，默认现代
- `--color-scheme`：配色方案选择
- `--institution-branding`：包含机构颜色和标志
- `--font-family`：字体选择

## 标准海报尺寸

### 会议标准尺寸
- **4' × 3'** (48" × 36")：最常见的会议海报
- **5' × 4'** (60" × 48")：大型会议的大型格式
- **3' × 4'** (36" × 48")：纵向排列，适用于窄空间
- **A0** (841mm × 1189mm)：国际标准
- **A1** (594mm × 841mm)：紧凑型会议海报

### 自定义尺寸
系统支持任何自定义尺寸。使用以下方式指定：
```bash
--poster-width-inches [width] --poster-height-inches [height]
```

## 输入要求

### 支持的输入格式
1. **LaTeX源文件**（首选）
   - 包含完整论文的主 `.tex` 文件
   - 所有图表已引用
   - 编译成功

2. **PDF**
   - 嵌入字体的高质量PDF
   - 可选择文本（非扫描）
   - 高分辨率图表

### 必需的内容元素
- 标题和作者
- 摘要或总结
- 方法论描述
- 关键结果
- 结论
- 参考文献（可选但建议包含）

### 建议的资源
- 高分辨率图表（最低300 DPI）
- 矢量图形（PDF、SVG、EPS）
- 机构标志
- 作者照片（可选）
- 指向网站/仓库的二维码

## 输出结构

```
output/paper_name/poster/

    └── content_map.json     # 内容组织
```

## 海报布局版块

### 标准版块
1. **标题栏**
   - 标题（大而醒目）
   - 作者和单位
   - 机构标志
   - 会议信息

2. **引言/背景**
   - 问题陈述
   - 研究动机
   - 简要文献背景

3. **方法**
   - 实验设计
   - 关键程序
   - 重要参数
   - 视觉流程图

4. **结果**
   - 关键发现（最大版块）
   - 主要图表
   - 统计摘要
   - 可视化数据展示

5. **结论**
   - 主要要点
   - 意义
   - 未来工作

6. **参考文献与联系方式**
   - 精选关键参考文献
   - 作者联系方式
   - 论文/网站的二维码
   - 致谢

## 配色方案

### 预定义方案
- **机构**：使用机构品牌颜色
- **专业**：海军蓝和灰色配色
- **鲜艳**：大胆、吸睛的颜色
- **自然**：绿色和大地色调
- **科技**：现代蓝色和青色
- **暖色**：橙色和红色点缀
- **冷色**：蓝色和紫色调

### 自定义配色方案
在配置中指定自定义颜色：
```json
{
  "primary": "#1E3A8A",
  "secondary": "#3B82F6",
  "accent": "#F59E0B",
  "background": "#FFFFFF",
  "text": "#1F2937"
}
```

## 字体选项

### 字体系列
- **无衬线**（默认）：简洁、现代、高度可读
- **衬线**：传统学术外观
- **混合**：正文用衬线，标题用无衬线
- **等宽**：用于代码和技术内容

### 尺寸层次
- **标题**：72-96pt
- **章节标题**：48-60pt
- **小节标题**：36-48pt
- **正文**：24-32pt
- **图注**：18-24pt
- **参考文献**：16-20pt

## 质量保证

### 自动检查
- **文字可读性**：最小字号验证
- **颜色对比度**：无障碍合规性
- **图表质量**：分辨率和清晰度检查
- **布局平衡**：内容分布分析
- **品牌一致性**：标志和颜色验证

### 手动审查清单
1. ☐ 所有图表高分辨率且清晰
2. ☐ 文字在3-6英尺外可读
3. ☐ 配色专业且一致
4. ☐ 无文字重叠或布局问题
5. ☐ 机构标志正确且高质量
6. ☐ 二维码有效且链接正确URL
7. ☐ 作者信息准确
8. ☐ 关键发现突出显示
9. ☐ 参考文献格式正确
10. ☐ 文件尺寸和分辨率适合打印

## 打印准备

### 文件规格
- **格式**：专业打印用 PDF/X-1a 或 PDF/X-4
- **分辨率**：最低300 DPI，精细细节600 DPI
- **颜色模式**：CMYK用于打印（系统自动从RGB转换）
- **出血**：所有边缘0.125"出血（自动添加）
- **字体**：所有字体嵌入PDF

### 打印建议
1. **打印店**：使用专业海报打印服务
2. **纸张类型**：学术海报用哑光或绸缎面
3. **背板**：泡沫板或刚性背板以保证稳定性
4. **保护**：覆膜可选但建议使用
5. **测试打印**：先打印A4/Letter尺寸预览

### 预算选项
- **标准**：专业打印店4'×3'海报$50-100
- **经济**：仅打印$20-40（不含安装）
- **高级**：高端材料和安装$150-300
- **自制**：多页拼贴组装<$10

## 最佳实践

### 内容优化
1. **聚焦关键发现**：海报应一目了然地讲述故事
2. **限制文字**：使用项目符号，避免段落
3. **优先视觉元素**：图表应占据主要空间
4. **清晰的流程**：引导观众按逻辑顺序浏览
5. **突出贡献**：使创新点显而易见

### 设计优化
1. **使用对比**：确保文字易于阅读
2. **保持层次**：尺寸表明重要性
3. **平衡内容**：避免任何版块过于拥挤
4. **一致的样式**：全程使用相同的字体、颜色
5. **留白**：不要填满每一寸空间

### 图表优化
1. **足够大**：主要图表最小宽度6英寸
2. **高分辨率**：最低300 DPI
3. **清晰的标签**：坐标轴标签、图例可读
4. **去除杂乱**：为海报格式简化
5. **使用图注**：简短、信息丰富的描述

## 限制

- 复杂公式可能需要手动调整以提高可读性
- 非常长的论文可能需要内容优先级排序
- 自定义品牌需要手动指定或API访问
- 多语言支持仅限于常见语言
- 3D可视化在2D海报格式中可能会损失质量
