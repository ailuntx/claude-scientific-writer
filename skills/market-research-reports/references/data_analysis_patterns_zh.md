## 市场规模计算框架

### TAM/SAM/SOM 分析

**总潜在市场（TAM）** 代表实现100%市场份额时的总收入机会。

#### 自上而下法
```
TAM = 行业总收入（来自市场研究报告）

示例：
- 全球人工智能软件市场（2024年）：1840亿美元
- 来源：Gartner、IDC或类似机构
```

#### 自下而上法
```
TAM = 潜在客户数量 × 平均客户收入

示例：
- 全球企业数量：4亿
- 目标细分市场（大企业）：50,000家
- 解决方案年均支出：500,000美元
- TAM = 50,000 × 500,000 = 250亿美元
```

**可服务市场（SAM）** 代表基于产品/服务能力可以服务的TAM部分。

```
SAM = TAM × 适用细分市场百分比

示例：
- TAM：250亿美元
- 地域限制（仅限北美）：40%
- 产品适合度（仅限企业）：60%
- SAM = 250亿美元 × 40% × 60% = 60亿美元
```

**可获得市场（SOM）** 代表现实的市场份额获取。

```
SOM = SAM × 可实现市场份额百分比

示例：
- SAM：60亿美元
- 保守市场份额（5%）：3亿美元
- 基准市场份额（10%）：6亿美元
- 乐观市场份额（15%）：9亿美元
```

### 增长率计算

#### 复合年增长率（CAGR）
```
CAGR = (终值 / 起始值)^(1/n) - 1

其中 n = 年数

示例：
- 2020年市场规模：100亿美元
- 2024年市场规模：180亿美元
- n = 4年
- CAGR = (18/10)^(1/4) - 1 = 15.8%
```

#### 同比增长
```
同比增长 = (当年 - 上年) / 上年 × 100

示例：
- 2023年：150亿美元
- 2024年：180亿美元
- 同比增长 = (18-15)/15 × 100 = 20%
```

## 财务分析模式

### 收入预测模型

```
第N年收入 = 第N-1年收入 × (1 + 增长率)

或自下而上：
收入 = 客户数 × 每客户收入 × 留存率
     + 新客户 × 每客户收入 × (1 - 流失率)
```

### 情景分析模板

| 指标 | 保守 | 基准 | 乐观 |
|--------|--------------|-----------|------------|
| 市场增长 | X% | Y% | Z% |
| 市场份额 | X% | Y% | Z% |
| 价格 | 美元X | 美元Y | 美元Z |
| 毛利率 | X% | Y% | Z% |
| **第5年收入** | 美元X | 美元Y | 美元Z |
| **第5年EBITDA** | 美元X | 美元Y | 美元Z |

### 关键财务指标

| 指标 | 公式 | 目标 |
|--------|---------|--------|
| 毛利率 | (收入 - 销货成本) / 收入 | X% |
| EBITDA利润率 | EBITDA / 收入 | X% |
