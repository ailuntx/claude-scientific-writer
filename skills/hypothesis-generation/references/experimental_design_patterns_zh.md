# 实验设计模式

## 科学假设检验的常用方法

本参考提供跨科学领域的实验设计模式和框架。使用这些模式为生成的假设开发严格的检验。

**报告结构说明：** 生成假设报告时，仅在主文本假设框中提及关键实验方法（如"体内敲除研究"或"前瞻性队列设计"）。在**附录B：详细实验设计**中包含完整方法的综合实验方案、样本量、统计方法、可行性评估和资源要求。

## 实验设计选择框架

根据以下因素选择实验方法：

- **假设性质：** 机制性、因果性、相关性、描述性
- **研究系统：** 体外、体内、计算、观察
- **可行性：** 时间、成本、伦理、技术能力
- **所需证据：** 概念验证、因果演示、定量关系

## 实验室实验设计

### 体外实验

**何时使用：** 在控制系统中测试分子、细胞或生化机制。

**常用模式：**

#### 1. 剂量-反应研究
- **目的：** 建立输入与效应之间的定量关系
- **设计：** 测试干预的多个浓度/剂量
- **关键要素：**
  - 阴性对照（无处理）
  - 阳性对照（已知有效处理）
  - 多个剂量水平（通常5-8个点）
  - 技术重复（每个条件≥3次）
  - 适当的统计分析（曲线拟合、IC50/EC50测定）

**示例应用：**
"为测试化合物X是否抑制酶Y，在0、1、10、100、1000 nM化合物X浓度下测量酶活性，每个剂量n=3重复。"

#### 2. 功能获得/丧失研究
- **目的：** 确定特定成分的因果作用
- **设计：** 添加（过表达）或去除（敲除/敲低）成分
- **关键要素：**
  - 野生型对照
  - 功能获得条件（过表达、组成性激活）
  - 功能丧失条件（敲除、敲低、抑制）
  - 拯救实验（恢复功能丧失的功能）
  - 测量下游效应

**示例应用：**
"测试蛋白质X是否导致表型Y：(1)敲除X并观察表型丧失，(2)过表达X并观察表型增强，(3)用X再表达拯救敲除。"

#### 3. 时间过程研究
- **目的：** 了解事件的时间动态和顺序
- **设计：** 在多个时间点测量结果
- **关键要素：**
  - 时间0基线
  - 早期时间点（捕获快速变化）
  - 中间时间点
  - 晚期时间点（稳态）
  - 每个时间点足够的重复

**示例应用：**
"在刺激后0、5、15、30、60、120分钟测量蛋白质磷酸化，以确定峰值激活时间。"

### 体内实验

**何时使用：** 在完整生物体中测试假设，以评估全身性、生理或行为效应。

**常用模式：**

#### 4. 组间设计
- **目的：** 比较接受不同处理的不同组
- **设计：** 将受试者随机分配到处理组
- **关键要素：**
  - 随机分配到各组
  - 适当的样本量（功效分析）
  - 对照组（载体、假手术或标准处理）
  - 盲法（单盲或双盲）
  - 各组标准化条件

**示例应用：**
"将20只小鼠随机分配到载体对照组或药物处理组，每周测量肿瘤大小8周，实验者对组别分配设盲。"

#### 5. 自身对照（重复测量）设计
- **目的：** 每个受试者作为自身对照，减少受试者间变异
- **设计：** 同一受试者在多个条件/时间点测量
- **关键要素：**
  - 基线测量
  - 平衡顺序（如有可能顺序效应）
  - 洗脱期（用于连续处理）
  - 适当的重复测量统计分析

**示例应用：**
"在同一参与者的基线、训练干预后和3个月随访时测量认知表现。"

#### 6. 析因设计
- **目的：** 同时测试多个因素及其交互作用
- **设计：** 交叉多个自变量的所有水平
- **关键要素：**
  - 明确的主效应和交互作用
  - 交互作用检验的足够功效
  - 完整析因或部分析因（视情况而定）

**示例应用：**
"2×2设计：基因型（野生型vs突变型）×处理（载体vs药物），以测试药物效应是否依赖于基因型。"

### 计算/建模实验

**何时使用：** 测试关于复杂系统的假设、进行预测，或当物理实验不可行时。

#### 7. 计算机模拟
- **目的：** 建模复杂系统，检验理论预测
- **设计：** 实现计算模型并改变参数
- **关键要素：**
  - 具有明确假设的明确定义模型
  - 参数敏感性分析
  - 与已知数据验证
  - 生成实验检验的预测

**示例应用：**
"建立疾病传播的智能体模型，改变传播率和干预时间，将预测与经验性流行病数据进行比较。"

#### 8. 生物信息学/荟萃分析
- **目的：** 使用现有数据集检验假设
- **设计：** 分析大规模数据或汇总多项研究
- **关键要素：**
  - 适当的统计校正（多重检验）
  - 在独立数据集中验证
  - 控制混杂因素和批次效应
  - 明确的纳入/排除标准

**示例应用：**
"测试基因X表达是否与15个癌症数据集（总计n>5000患者）的生存相关，使用Cox回归与临床协变量。"

## 观察性研究设计

### 当物理操作不可能或不道德时

#### 9. 横断面研究
- **目的：** 在单个时间点检查关联
- **设计：** 在一个时间点测量人群中感兴趣的变量
- **优势：** 快速、廉价，可确定患病率
- **局限性：** 不能建立时间顺序或因果关系
- **关键要素：**
  - 代表性抽样
  - 标准化测量
  - 控制混杂变量
  - 适当的统计分析

**示例应用：**
"调查1000名成年人以测试饮食模式与生物标志物X的关联，控制年龄、性别BMI和体力活动。"

#### 10. 队列研究（前瞻性/纵向）
- **目的：** 建立时间关系和潜在因果关联
- **设计：** 随时间随访一组，测量暴露和结局
- **优势：** 可建立时间顺序，计算发病率
- **关键要素：**
  - 基线暴露评估
  - 在定义的时间间隔随访
  - 尽量减少失访
  - 考虑时变混杂因素

**示例应用：**
"随访5000名最初健康的人10年，测试基线维生素D水平是否预测心血管疾病发病率。"

#### 11. 病例对照研究
- **目的：** 高效研究罕见结局，比较病例和对照
- **设计：** 识别有结局的病例，选择匹配的对照，比较暴露
- **优势：** 对罕见疾病高效，相对快速
- **局限性：** 回忆偏倚，选择偏倚，不能计算发病率
- **关键要素：**
  - 明确的病例定义
  - 适当的对照选择（匹配或统计校正）
  - 回顾性暴露评估
  - 控制混杂因素

**示例应用：**
"将200名罕见疾病X患者与400名无X的匹配对照进行比较，测试早年暴露Y是否在两组间存在差异。"

## 临床试验设计

#### 12. 随机对照试验（RCT）
- **目的：** 检验人类干预的金标准
- **设计：** 将参与者随机分配到处理或对照
- **关键要素：**
  - 随机化（简单、区组或分层）
  - 分配隐藏
  - 盲法（参与者、提供者、评估者）
  - 意向治疗分析
  - 预先注册的方案和分析计划

**示例应用：**
"双盲RCT：随机分配300名患者接受药物X或安慰剂12周，测量症状改善的主要结局。"

#### 13. 交叉试验
- **目的：** 每个参与者按顺序接受所有处理
- **设计：** 参与者在处理之间交叉，有洗脱期
- **优势：** 减少受试者间变异，需要较少参与者
- **局限性：** 顺序效应，需要可逆条件，持续时间更长
- **关键要素：**
  - 足够的洗脱期
  - 随机处理顺序
  - 延滞效应评估

**示例应用：**
"交叉试验：参与者接受处理A 4周，洗脱2周，然后接受处理B 4周（随机顺序）。"

## 高级设计注意事项

### 样本量和统计功效

**关键问题：**

- 什么效应量是有意义可检测的？
- 将使用什么统计检验？
- 什么alpha（显著性水平）和beta（功效）是适当的？
- 测量的预期变异是什么？

**一般指南：**

- 实验前进行正式功效分析
- 对于初步研究，每组n≥10最低
- 对于确定性研究，目标是≥80%功效
- 在纵向研究中考虑潜在脱落

### 对照

**对照类型：**

- **阴性对照：** 无干预（基线）
- **阳性对照：** 已知有效干预（验证系统）
- **载体对照：** 递送方法无活性成分
- **假手术对照：** 模拟干预无活性成分（手术等）
- **历史对照：** 既往数据（最弱，尽量避免）

### 盲法

**水平：**

- **开放标签：** 不设盲（适用于客观测量）
- **单盲：** 参与者设盲（减少安慰剂效应）
- **双盲：** 参与者和实验者设盲（减少评估偏倚）
- **三盲：** 参与者、实验者和分析师设盲（最强）

### 重复

**技术重复：** 同一样本的重复测量

- 减少测量误差
- 通常2-3次重复足够

**生物学重复：** 独立样本/受试者

- 解决生物学变异
- 对泛化至关重要
- 最低：n≥3，最好n≥5-10每组

**实验重复：** 重复整个实验

- 验证发现跨时间、设备、操作者
- 确认结果的金标准

### 混杂控制

**策略：**

- **随机化：** 将混杂因素均匀分布在各组
- **匹配：** 在条件间配对相似受试者
- **区组：** 按混杂因素分组，然后在区组内随机化
- **统计校正：** 测量混杂因素并在分析中校正
- **标准化：** 保持各组条件恒定

## 选择适当的设计

**决策树：**

1. **变量可以操作吗？**
   - 是 → 实验设计（RCT、实验室实验）
   - 否 → 观察性设计（队列、病例对照、横断面）

2. **系统是什么？**
   - 细胞/分子 → 体外实验
   - 完整生物体 → 体内实验
   - 人类 → 临床试验或观察性研究
   - 复杂系统 → 计算建模

3. **主要目标是什么？**
   - 机制 → 功能获得/丧失、剂量-反应
   - 因果 → RCT、具有良好对照的队列研究
   - 关联 → 横断面、病例对照
   - 预测 → 建模、机器学习
   - 时间动态 → 时间过程、纵向

4. **限制条件是什么？**
   - 时间有限 → 横断面、体外
   - 预算有限 → 计算、观察
   - 伦理问题 → 观察、体外
   - 罕见结局 → 病例对照、荟萃分析

## 整合多种方法

强有力的假设检验通常结合多种设计：

**示例：测试微生物组是否影响认知功能**

1. **观察性：** 队列研究显示微生物组组成与认知的关联
2. **动物模型：** 接受微生物组移植的无菌小鼠显示认知变化
3. **机制：** 体外研究显示微生物代谢物影响神经元功能
4. **临床试验：** 益生菌干预改善认知分数的RCT
5. **计算：** 预测哪些微生物组概况应该影响认知的模型

**三角测量方法：**

- 每种设计解决不同方面/局限性
- 多种方法的趋同证据加强因果主张
- 从观察/体外开始，然后进行确定性因果检验

## 常见陷阱

- 样本量不足（功效不足）
- 缺乏适当的对照
- 混杂变量未考虑
- 统计检验不当
- P黑客或多重检验未校正
- 涉及主观评估时未设盲
- 未能重复发现
- 未预先注册分析计划（临床试验）

## 假设检验的实际应用

设计检验假设的实验时：

1. **将设计与假设细节匹配：** 因果主张需要实验操作；关联可使用观察性设计
2. **先简单后复杂：** 用简单设计进行初步研究，然后增加复杂性
3. **仔细规划对照：** 对照验证系统并分离特定效应
4. **考虑可行性：** 在理想设计与实际限制之间平衡
5. **计划多次实验：** 很少有单一实验明确检验一个假设
6. **预先指定分析：** 在数据收集前决定统计检验
7. **内置验证：** 独立重复、正交方法、趋同证据
